# 🧠 Crop Classification Methods / Métodos de Clasificación de Cultivos

> Cerebro documental — el Handbook es la columna; la investigación es aditiva y citada. / *Documentary brain — Handbook is the backbone; research is additive and cited.*

## English

Crop classification methods have evolved from traditional machine learning (ML) to deep learning (DL), driven by the need for timely and accurate agricultural statistics. Traditional approaches, such as Random Forest (RF) and Support Vector Machines (SVM), rely on hand-crafted features (e.g., vegetation indices, texture) from satellite image time series (SITS). While interpretable, they are sensitive to changes in crop mixtures and temporal patterns. Active learning strategies reduce labeling effort, achieving ~77–80% accuracy several weeks before harvest with RF/SVM [R1]. In-season classification benefits from growth stage normalization, which aligns phenological stages across fields to improve generalization [R2]. Deep learning now dominates, with architectures like ConvNets and Transformers learning representations end-to-end from raw time series, outperforming traditional methods in yield estimation and crop mapping [R3]. Supervised classification remains the standard for crop-type mapping, but unsupervised approaches (e.g., clustering) are used when reference data are scarce. Pixel-based methods assign classes to individual pixels, while object-based image analysis (OBIA) segments fields to reduce noise. Temporal features are critical for distinguishing crops with similar spectral signatures (e.g., maize vs. soybean) [R3]. This theme grounds the Handbook concepts:
- **land-classification**: Provides specific methods for agricultural land use categorization.
- **satellite-image-classification**: Leverages multi-temporal imagery for dynamic mapping.
- **crop-classification**: Employs ML/DL to identify crop types from SITS, with accuracy dependent on timing and feature engineering.
- **classification**: Relates to the broader process of assigning thematic labels using algorithms.
A key gap addressed is the transition from manual feature extraction to automatic learning, enabling scalable and robust crop mapping. Active learning and growth stage normalization further reduce the need for extensive ground data, improving operational feasibility [R1][R2].

## Español

Los métodos de clasificación de cultivos han evolucionado del aprendizaje automático (ML) tradicional al aprendizaje profundo (DL), impulsados por la necesidad de estadísticas agrícolas oportunas y precisas. Los enfoques tradicionales, como Random Forest (RF) y Support Vector Machines (SVM), utilizan características diseñadas manualmente (índices de vegetación, textura) a partir de series temporales de imágenes satelitales (SITS). Aunque interpretables, son sensibles a cambios en las mezclas de cultivos y patrones temporales. Estrategias de aprendizaje activo reducen el esfuerzo de etiquetado, logrando una precisión de ~77–80% varias semanas antes de la cosecha con RF/SVM [R1]. La clasificación dentro de la temporada se beneficia de la normalización del estado fenológico, que alinea las fases entre campos para mejorar la generalización [R2]. El aprendizaje profundo ahora domina, con arquitecturas como ConvNets y Transformers aprendiendo representaciones de extremo a extremo a partir de series temporales crudas, superando a los métodos tradicionales en estimación de rendimiento y mapeo de cultivos [R3]. La clasificación supervisada sigue siendo el estándar para el mapeo de tipos de cultivos, mientras que los enfoques no supervisados (ej., clustering) se usan cuando los datos de referencia son escasos. Los métodos basados en píxeles asignan clases a píxeles individuales, mientras que el análisis de imágenes basado en objetos (OBIA) segmenta campos para reducir ruido. Las características temporales son críticas para distinguir cultivos con firmas espectrales similares [R3]. Este tema fundamenta los conceptos del Manual:
- **land-classification**: Proporciona métodos específicos para la categorización del uso agrícola del suelo.
- **satellite-image-classification**: Aprovecha imágenes multitemporales para mapeo dinámico.
- **crop-classification**: Emplea ML/DL para identificar tipos de cultivos a partir de SITS, con precisión dependiente del momento y la ingeniería de características.
- **classification**: Se relaciona con el proceso general de asignar etiquetas temáticas usando algoritmos.
Una brecha clave abordada es la transición de la extracción manual de características al aprendizaje automático, permitiendo un mapeo escalable y robusto. El aprendizaje activo y la normalización fenológica reducen la necesidad de datos de campo extensivos, mejorando la viabilidad operativa [R1][R2].
