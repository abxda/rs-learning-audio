# ARD / ARD

> **level 3** · *data* · sources: th_data_sources

**What it is / Qué es.**

- **EN.** [[analysis-ready-data|Analysis Ready Data]]: [[satellite-data|satellite data]] processed to meet standards defined by CEOS, including geometric, radiometric, and atmospheric corrections, so that [[images|images]] are consistent across [[sensors|sensors]], times, and locations, with [[cloud|cloud]] and shadow [[masking|masking]] included; prepared for direct use in [[analysis|analysis]] without further preprocessing.
- **ES.** [[analysis-ready-data|Datos Listos para Análisis]]: [[satellite-data|datos satelitales]] procesados para cumplir con los estándares definidos por CEOS, que incluyen correcciones geométricas, radiométricas y atmosféricas, de modo que las [[images|imágenes]] sean consistentes entre [[sensors|sensores]], tiempos y ubicaciones, con [[cloud-masking|enmascaramiento de nubes]] y sombras incluido; preparados para uso directo en [[analysis|análisis]] sin preprocesamiento adicional.

## Understand these first / Entiende esto primero
- [[cloud-collections|Cloud collections]] — *data*
    - **EN.** Cloud collections often consist of Analysis Ready Data (ARD) to simplify downstream processing.
    - **ES.** Las colecciones en la nube a menudo consisten en Datos Listos para Análisis (ARD) para simplificar el procesamiento posterior.
- [[radiometric-corrections|Radiometric corrections]] — *method*
    - **EN.** ARD processing includes radiometric corrections to produce consistent surface reflectance products.
    - **ES.** El procesamiento ARD incluye correcciones radiométricas para producir productos consistentes de reflectancia superficial.
- [[analysis-ready-data|Analysis Ready Data]] — *data*
    - **EN.** ARD stands for Analysis Ready Data, which has been geometrically and radiometrically corrected.
    - **ES.** ARD significa Datos Listos para Análisis, que han sido corregidos geométrica y radiométricamente.
