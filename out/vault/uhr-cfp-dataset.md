# UHR CFP dataset / conjunto de datos UHR CFP

> **level 4** · *data* · sources: uav_field_parcels#experimental-area-and-data

**What it is / Qué es.**

- **EN.** A [[dataset|dataset]] of ultra-high [[resolution|resolution]] (0.2m) UAV [[images|images]] paired with [[label|labels]] for cultivation [[field|field]] [[parcels|parcels]], used to train and evaluate models for automated parcel vectorization.
- **ES.** Un [[dataset|conjunto de datos]] de [[uav-imagery|imágenes de UAV]] de [[resolution|resolución]] ultraalta (0,2 m) emparejadas con etiquetas para [[parcels|parcelas]] de [[field|campo]] de [[crop|cultivo]], utilizado para entrenar y evaluar modelos para la vectorización automatizada de parcelas.

## Understand these first / Entiende esto primero
- [[dataset|Dataset]] — *data*
    - **EN.** The UHR CFP dataset is a labeled collection of UAV images for parcel detection and segmentation.
    - **ES.** El conjunto de datos UHR CFP es una colección etiquetada de imágenes de UAV para detección y segmentación de parcelas.
- [[parcels|Parcels]] — *data*
    - **EN.** Each image in the UHR CFP dataset is paired with polygon labels of individual agricultural field parcels.
    - **ES.** Cada imagen en el conjunto de datos UHR CFP está emparejada con etiquetas de polígonos de parcelas de campo agrícolas individuales.
