# Remote sensing estimation / Estimación por teledetección

> **level 4** · *method* · sources: cy_china

**What it is / Qué es.**

- **EN.** The [[process|process]] of deriving [[agricultural|agricultural]] [[parameters|parameters]], such as [[crop-yield|crop yield]] or [[planted-area|planted area]], from [[satellite-imagery|satellite imagery]] using models that relate spectral measurements to [[ground|ground]] conditions.
- **ES.** [[process|Proceso]] de derivar [[parameters|parámetros]] agrícolas, como el [[crop-yield|rendimiento de cultivos]] o el [[planted-area|área sembrada]], a partir de [[satellite-imagery|imágenes satelitales]] utilizando modelos que relacionan las mediciones espectrales con las condiciones del [[ground|terreno]].

## Understand these first / Entiende esto primero
- [[estimation|Estimation]] — *method*
    - **EN.** Estimation infers unknown quantities like crop area from observed data such as satellite imagery.
    - **ES.** La estimación infiere cantidades desconocidas como el área de cultivo a partir de datos observados como imágenes satelitales.
- [[remote-sensing|Remote sensing]] — *method*
    - **EN.** Remote sensing provides satellite imagery from which agricultural parameters can be estimated.
    - **ES.** La teledetección proporciona imágenes satelitales a partir de las cuales se pueden estimar parámetros agrícolas.
