# Layers / Capas

> **level 1** · *data* · sources: th_remote_sensing

**What it is / Qué es.**

- **EN.** Layers are the individual [[spectral-bands|spectral bands]] or derived [[data|data]] components in a [[satellite-image-collection|satellite image collection]], each capturing light in a specific [[wavelength|wavelength]] [[range|range]]. Multiple layers form a multi-spectral image used for [[analysis|analysis]].
- **ES.** Las capas son las [[bands|bandas espectrales]] individuales o componentes de [[data|datos]] derivados en una [[satellite-image-collection|colección de imágenes satelitales]], cada una capturando luz en un [[range|rango]] de [[wavelength|longitud de onda]] específico. Múltiples capas forman una imagen multiespectral utilizada para el [[analysis|análisis]].

## Understand these first / Entiende esto primero
- [[images|Images]] — *data*
    - **EN.** Images are composed of layers, each representing a specific spectral band or derived data component.
    - **ES.** Las imágenes se componen de capas, cada una representa una banda espectral específica o un componente de datos derivado.
