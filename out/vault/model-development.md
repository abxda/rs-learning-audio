# Model Development / Desarrollo de modelos

> **level 5** · *method* · sources: cy_indonesia

**What it is / Qué es.**

- **EN.** The [[process|process]] of creating, [[training|training]], and tuning a computational [[model|model]], such as a [[machine-learning|machine learning]] classifier, to perform a specific task like [[crop-classification|crop classification]].
- **ES.** El [[process|proceso]] de crear, entrenar y ajustar un [[model|modelo]] computacional, como un clasificador de [[machine-learning|aprendizaje automático]], para realizar una tarea específica como la [[crop-classification|clasificación de cultivos]].

## Understand these first / Entiende esto primero
- [[model|Model]] — *method*
    - **EN.** Model development is the process of creating, training, and tuning a computational model.
    - **ES.** El desarrollo de modelos es el proceso de crear, entrenar y ajustar un modelo computacional.
- [[training|Training]] — *method*
    - **EN.** Model development involves training a model on labeled data to learn patterns.
    - **ES.** El desarrollo de modelos implica entrenar un modelo con datos etiquetados para aprender patrones.
- [[model-evaluation|Model evaluation]] — *method*
    - **EN.** Model development includes model evaluation to assess performance and guide improvements.
    - **ES.** El desarrollo de modelos incluye la evaluación de modelos para evaluar el rendimiento y guiar mejoras.
