# Machine / Máquina

> **level 1** · *primitive* · sources: th_machine_learning · th_quality_control

**What it is / Qué es.**

- **EN.** A trained [[algorithm|algorithm]], typically from [[machine-learning|machine learning]], that maps [[input|input]] features to output [[predictions|predictions]] or classifications.
- **ES.** Un [[algorithm|algoritmo]] entrenado, típicamente de [[machine-learning|aprendizaje automático]], que mapea atributos de [[input|entrada]] a [[predictions|predicciones]] o clasificaciones de salida.

## Understand these first / Entiende esto primero
- [[model|Model]] — *method*
    - **EN.** A machine is a trained model that maps input features to output predictions.
    - **ES.** Una máquina es un modelo entrenado que asigna atributos de entrada a predicciones de salida.
