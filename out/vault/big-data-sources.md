# Big Data sources / Fuentes de macrodatos

> **level 3** · *other* · sources: introduction

**What it is / Qué es.**

- **EN.** Large and complex datasets, such as [[satellite-imagery|satellite imagery]] archives, that require specialized [[processing|processing]] [[methods|methods]] for [[analysis|analysis]].
- **ES.** Conjuntos de [[data|datos]] grandes y complejos, como archivos de [[satellite-imagery|imágenes satelitales]], que requieren [[methods|métodos]] de [[processing|procesamiento]] especializados para su [[analysis|análisis]].

## Understand these first / Entiende esto primero
- [[source|Source]] — *primitive*
    - **EN.** Big data sources include satellite archives and public datasets that provide petabyte-scale imagery.
    - **ES.** Las fuentes de macrodatos incluyen archivos satelitales y conjuntos de datos públicos que proporcionan imágenes a escala de petabytes.
- [[big-data|Big Data]] — *data*
    - **EN.** Big data sources are origins of large-scale datasets that require specialized processing methods.
    - **ES.** Las fuentes de macrodatos son orígenes de grandes conjuntos de datos que requieren métodos de procesamiento especializados.
