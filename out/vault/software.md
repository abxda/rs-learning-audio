# Software / Software

> **level 0** · *primitive*

**What it is / Qué es.**

- **EN.** Programs and operating [[information|information]] used by a computer.
- **ES.** Programas e [[information|información]] operativa utilizados por una computadora.

## This unlocks / Esto habilita
- [[tools|Tools]] — Tools are software systems that implement methods for processing and analyzing Earth observation data.
- [[component|Component]] — Components are custom software pieces that execute well-defined analytical functions.
- [[package|Package]] — A package bundles reusable software components and data for a specific analysis, like the sits R package.
- [[reproducibility|Reproducibility]] — Reproducibility relies on software containers or virtual environments that replicate the exact computational setup.
- [[reproducible-code|Reproducible code]] — Reproducible code packages analysis scripts and dependencies so that others can rerun the same computations.
- [[jupyterlab|JupyterLab]] — JupyterLab provides a platform for interactive code development, data analysis, and visualization.
