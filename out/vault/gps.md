# GPS / GPS

> **level 2** · *sensor* · sources: th_alignment

**What it is / Qué es.**

- **EN.** Global Positioning [[system|System]], a [[satellite|satellite]]-based navigation system that provides precise geographic [[coordinates|coordinates]] for [[ground|ground]] control [[points|points]], [[field|field]] [[samples|sample]] locations, and [[survey|survey]] points.
- **ES.** Sistema de Posicionamiento Global, un [[system|sistema]] de navegación basado en satélites que proporciona [[coordinates|coordenadas]] geográficas precisas para [[points|puntos]] de control terrestre, ubicaciones de [[field-samples|muestras de campo]] y puntos de [[survey|encuesta]].

## Understand these first / Entiende esto primero
- [[satellite|Satellite]] — *platform*
    - **EN.** GPS uses a constellation of satellites to provide precise geographic coordinates for ground locations.
    - **ES.** El GPS utiliza una constelación de satélites para proporcionar coordenadas geográficas precisas para ubicaciones en tierra.
