# DataVolumes / Volúmenes de datos

> **level 1** · *data* · sources: howto

**What it is / Qué es.**

- **EN.** A [[kubernetes|Kubernetes]] resource that manages importing and [[persistent-storage|persistent storage]] of [[data|data]] from external sources into a [[cluster|cluster]] for use by [[applications|applications]].
- **ES.** Un recurso de [[kubernetes|Kubernetes]] que gestiona la importación y el [[persistent-storage|almacenamiento persistente]] de [[data|datos]] desde fuentes externas a un [[cluster|clúster]] para que las [[applications|aplicaciones]] los utilicen.

## Understand these first / Entiende esto primero
- [[kubernetes|Kubernetes]] — *primitive*
    - **EN.** DataVolumes are a Kubernetes resource that manages importing and persistent storage of data.
    - **ES.** DataVolumes es un recurso de Kubernetes que gestiona la importación y el almacenamiento persistente de datos.
- [[persistent-storage|Persistent storage]] — *primitive*
    - **EN.** They provide persistent storage for containerized applications in a cluster.
    - **ES.** Proporcionan almacenamiento persistente para aplicaciones contenerizadas en un clúster.
