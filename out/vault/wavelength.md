# Wavelength / Longitud de onda

> **level 0** · *primitive*

**What it is / Qué es.**

- **EN.** Distance between successive peaks of a wave, determining the [[type|type]] of [[radiation|radiation]].
- **ES.** Distancia entre picos sucesivos de una onda, que determina el [[type|tipo]] de [[radiation|radiación]].

## This unlocks / Esto habilita
- [[spectral-bands|Spectral bands]] — Each spectral band corresponds to a narrow range of wavelengths in the electromagnetic spectrum.
- [[electromagnetic-radiation|Electromagnetic radiation]] — Wavelength is the fundamental property that distinguishes different types of electromagnetic radiation used in remote sensing.
- [[spectral-signature|Spectral signature]] — A spectral signature plots reflectance values as a function of wavelength, creating a pattern for identification.
- [[electromagnetic-spectrum|Electromagnetic spectrum]] — The electromagnetic spectrum orders radiation by wavelength, from short to long waves.
