# Satellite data cube / Cubo de datos satelitales

> **level 6** · *data* · sources: th_data_cubes

**What it is / Qué es.**

- **EN.** A multi-dimensional array of [[satellite|satellite]] [[images|images]] organized by spatial [[coordinates|coordinates]] and [[time|time]], providing a consistent [[grid|grid]] for [[time-series-analysis|time series analysis]] and [[machine-learning|machine learning]].
- **ES.** Arreglo multidimensional de [[satellite-imagery|imágenes satelitales]] organizado por [[coordinates|coordenadas]] espaciales y [[time|tiempo]], que proporciona una [[grid|cuadrícula]] consistente para el [[time-series-analysis|análisis de series temporales]] y el [[machine-learning|aprendizaje automático]].

## Understand these first / Entiende esto primero
- [[satellite-imagery|Satellite imagery]] — *data*
    - **EN.** Satellite imagery is stacked along spatial and temporal dimensions to form a data cube.
    - **ES.** Las imágenes satelitales se apilan a lo largo de dimensiones espaciales y temporales para formar un cubo de datos.
- [[data-cube|Data cube]] — *data*
    - **EN.** A data cube organizes satellite imagery into a regular spatiotemporal array.
    - **ES.** Un cubo de datos organiza las imágenes satelitales en un arreglo espacio-temporal regular.
