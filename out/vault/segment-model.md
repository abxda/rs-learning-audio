# Segment Model / Modelo de segmentación

> **level 0** · *method* · sources: dis_china_floods · dis_china_floods

**What it is / Qué es.**

- **EN.** A [[deep-learning|deep learning]] [[model|model]] that performs [[image-segmentation|image segmentation]] by generating object masks from [[input|input]] [[images|images]] and optional prompts. The [[segment|Segment]] Anything Model (SAM) is used for flood extent [[detection|detection]] in this [[handbook|handbook]].
- **ES.** Un [[model|modelo]] de [[deep-learning|aprendizaje profundo]] que realiza [[image-segmentation|segmentación de imágenes]] generando máscaras de objetos a partir de [[images|imágenes]] de [[input|entrada]] y avisos opcionales. El Modelo de Segmentación de Cualquier Cosa (SAM) se utiliza para la [[detection|detección]] de extensión de inundaciones en este [[handbook|manual]].

## This unlocks / Esto habilita
- [[deep-learning|Deep learning]] — A Segment Model is a deep learning model that performs image segmentation by generating object masks from input images.
- [[image-segmentation|Image segmentation]] — It is designed specifically for the task of image segmentation, often using architectures like SAM.
