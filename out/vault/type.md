# Type / Tipo

> **level 0** · *primitive* · sources: th_lucc · th_design_frames

**What it is / Qué es.**

- **EN.** A category or [[classes|class]] assigned to an entity, such as a [[land-cover-classes|land cover class]] or [[crop|crop]] species, within a [[classification-scheme|classification schema]].
- **ES.** Una categoría o clase asignada a una entidad, como una clase de [[land-cover|cobertura del suelo]] o especie de [[crop|cultivo]], dentro de un [[classification-scheme|esquema de clasificación]].

## This unlocks / Esto habilita
- [[classification-scheme|Classification scheme]] — A type is a category in a classification scheme, such as a land cover class or crop species.
- [[crop-types|Crop types]] — A type is a category; crop types divide agricultural production into distinct classes for classification.
