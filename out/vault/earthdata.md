# EarthData / EarthData

> **level 2** · *data* · sources: th_data_sources

**What it is / Qué es.**

- **EN.** [[nasa|NASA]]'s platform for discovering and accessing [[earth-observation|Earth observation data]]. [[users|Users]] must register and log in via a [[netrc-file|.netrc file]] to access certain collections such as [[harmonized-landsat-sentinel|Harmonized Landsat Sentinel]] (HLS).
- **ES.** La plataforma de la [[nasa|NASA]] para descubrir y acceder a [[data|datos]] de [[earth-observation|observación de la Tierra]]. Los [[users|usuarios]] deben registrarse e iniciar sesión mediante un [[netrc-file|archivo .netrc]] para acceder a ciertas colecciones como [[harmonized-landsat-sentinel|Landsat Sentinel Armonizado]] (HLS).

## Understand these first / Entiende esto primero
- [[earthdatalogin|Earthdatalogin]] — *software*
    - **EN.** earthdatalogin is an R package that helps users set up credentials to access NASA Earthdata.
    - **ES.** earthdatalogin es un paquete de R que ayuda a los usuarios a configurar credenciales para acceder a EarthData de la NASA.
