# Values / Valores

> **level 2** · *primitive* · sources: th_remote_sensing#introduction-to-remote-sensing-imagery

**What it is / Qué es.**

- **EN.** Numerical measurements recorded at each [[pixels|pixel]], such as [[spectral-reflectance|spectral reflectance]] or [[vegetation-indices|vegetation index]] values.
- **ES.** Mediciones numéricas registradas en cada píxel, como [[spectral-reflectance|reflectancia espectral]] o valores de [[vegetation-indices|índices de vegetación]].

## Understand these first / Entiende esto primero
- [[pixels|Pixels]] — *data*
    - **EN.** Values are numerical measurements recorded at each pixel, such as spectral reflectance.
    - **ES.** Los valores son mediciones numéricas registradas en cada píxel, como la reflectancia espectral.

## Lineage to fundamentals / Linaje hasta los fundamentos
[[images|Images]] → [[pixels|Pixels]] → [[values|Values]]

## This unlocks / Esto habilita
- [[reflectance|Reflectance]] — Pixels in satellite images store digital numbers that are converted to reflectance values.
- [[variability|Variability]] — Variability refers to the degree of spread or variation among data values, affecting classification complexity.
- [[variance|Variance]] — It quantifies how much individual values differ from the average of the dataset.
- [[probability|Probability]] — Probability assigns a numerical value between 0 and 1 to the likelihood of an event.
- [[regression|Regression]] — It uses numeric values of predictors to predict a continuous outcome, such as crop yield.
- [[thresholding|Thresholding]] — Thresholding assigns pixels to classes by comparing their values against a chosen cut‑off point.
