# Satellite-derived models / Modelos derivados de satélite

> **level 5** · *method* · sources: crop_statistics_ppi#example-yield-estimation · cy_finland#machine-learning

**What it is / Qué es.**

- **EN.** Computational models that use [[satellite-imagery|satellite imagery]] as [[input|input]] to predict [[biophysical|biophysical]] variables (e.g., [[crop-yield|crop yield]], plant [[area|area]] index) or classify [[land|land]] surface types. They are calibrated with [[ground-observations|ground observations]] and can be [[statistical|statistical]] or [[machine|machine]]-[[machine-learning|learning]] based.
- **ES.** Modelos computacionales que utilizan [[satellite-imagery|imágenes satelitales]] como [[input|entrada]] para predecir variables biofísicas (por [[example|ejemplo]], [[crop-yield|rendimiento de cultivos]], [[leaf-area-index|índice de área foliar]]) o clasificar tipos de [[earth|superficie terrestre]]. Se basan en [[ground-observations|observaciones de campo]] y pueden ser [[statisticians|estadísticos]] o de [[machine-learning|aprendizaje automático]].

## Understand these first / Entiende esto primero
- [[model|Model]] — *method*
    - **EN.** Satellite-derived models use satellite imagery as input to predict biophysical variables like crop yield or biomass.
    - **ES.** Los modelos derivados de satélite utilizan imágenes satelitales como entrada para predecir variables biofísicas como el rendimiento de cultivos o la biomasa.
- [[satellite-imagery|Satellite imagery]] — *data*
    - **EN.** Satellite imagery provides the input data for training and applying satellite-derived models to estimate agricultural parameters.
    - **ES.** Las imágenes satelitales proporcionan los datos de entrada para entrenar y aplicar modelos derivados de satélite para estimar parámetros agrícolas.
