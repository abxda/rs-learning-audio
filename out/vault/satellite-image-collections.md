# Satellite image collections / Colecciones de imágenes satelitales

> **level 3** · *data* · sources: th_data_sources#major-global-and-large-regional-cloud-providers

**What it is / Qué es.**

- **EN.** Organized archives of [[satellite-imagery|satellite imagery]], typically provided as [[analysis-ready-data|Analysis Ready Data]] (ARD) or regular [[data|data]] cubes, accessible via [[cloud|cloud]] [[platforms|platforms]] such as AWS, MPC, CDSE, or DEAFRICA.
- **ES.** Archivos organizados de [[satellite-imagery|imágenes satelitales]], típicamente proporcionados como [[analysis-ready-data|Datos Listos para Análisis]] (ARD) o cubos de [[data|datos]] regulares, accesibles a través de [[platforms|plataformas]] en la [[cloud|nube]] como AWS, MPC, CDSE o DEAFRICA.

## Understand these first / Entiende esto primero
- [[satellite|Satellite]] — *platform*
    - **EN.** Satellite image collections are organized archives of imagery from satellite missions.
    - **ES.** Las colecciones de imágenes satelitales son archivos organizados de imágenes de misiones satelitales.
- [[collection|Collection]] — *method*
    - **EN.** A satellite image collection is a set of images sharing a common source, format, and theme.
    - **ES.** Una colección de imágenes satelitales es un conjunto de imágenes que comparten una fuente, formato y tema común.
- [[cloud-collections|Cloud collections]] — *data*
    - **EN.** Cloud collections are satellite image collections stored on cloud platforms for scalable access.
    - **ES.** Las colecciones en la nube son colecciones de imágenes satelitales almacenadas en plataformas en la nube para un acceso escalable.
- [[ard-image-collections|ARD image collections]] — *data*
    - **EN.** Satellite image collections can be stored as ARD image collections, processed to analysis-ready standards.
    - **ES.** Las colecciones de imágenes satelitales pueden almacenarse como colecciones de imágenes ARD, procesadas según estándares listos para análisis.
