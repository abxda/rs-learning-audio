# Probability / Probabilidad

> **level 3** · *statistic* · sources: th_uncertainty#measuring-uncertainty

**What it is / Qué es.**

- **EN.** A numerical value between 0 and 1 representing the likelihood that a [[pixels|pixel]] belongs to a particular [[land-cover-classes|land cover class]], as output by a classifier.
- **ES.** Un valor numérico entre 0 y 1 que representa la probabilidad de que un píxel pertenezca a una clase de [[land-cover|cobertura terrestre]] particular, según la salida de un clasificador.

## Understand these first / Entiende esto primero
- [[values|Values]] — *primitive*
    - **EN.** Probability assigns a numerical value between 0 and 1 to the likelihood of an event.
    - **ES.** La probabilidad asigna un valor numérico entre 0 y 1 a la probabilidad de un evento.
- [[set|Set]] — *primitive*
    - **EN.** Probability measures the chance that an element from a set belongs to a particular subset.
    - **ES.** La probabilidad mide la posibilidad de que un elemento de un conjunto pertenezca a un subconjunto particular.
