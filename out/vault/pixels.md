# Pixels / Píxeles

> **level 1** · *data* · sources: th_data_cubes#working-with-satellite-image-time-series

**What it is / Qué es.**

- **EN.** The smallest discrete units in [[satellite-imagery|satellite imagery]], each representing a specific [[land|land]] [[area|area]] and containing [[spectral-reflectance|spectral reflectance]] [[values|values]].
- **ES.** Las unidades discretas más pequeñas en las [[satellite-imagery|imágenes satelitales]], cada una representa un [[area|área]] de [[land|tierra]] específica y contiene [[values|valores]] de [[spectral-reflectance|reflectancia espectral]].

## Understand these first / Entiende esto primero
- [[images|Images]] — *data*
    - **EN.** Pixels are the smallest discrete units in satellite images, each representing a specific land area.
    - **ES.** Los píxeles son las unidades discretas más pequeñas en imágenes satelitales, cada una representa un área de tierra específica.

## This unlocks / Esto habilita
- [[values|Values]] — Values are numerical measurements recorded at each pixel, such as spectral reflectance.
- [[resolution|Resolution]] — Resolution is the level of detail in an image, often described by the ground area each pixel covers.
- [[segmentation|Segmentation]] — Segmentation partitions an image into meaningful regions by grouping adjacent pixels with similar properties.
- [[raster|Raster]] — A raster is a grid of pixels where each pixel stores a value representing a measurement.
- [[pixel-classification|Pixel classification]] — Pixel classification assigns a class label to every pixel based on its spectral values.
- [[quality-flags|Quality Flags]] — Quality flags are per-pixel indicators of data reliability, such as cloud or shadow presence.
- [[digital-elevation-model|Digital Elevation Model]] — Digital elevation models store elevation values in each pixel, allowing slope and aspect calculations.
- [[image-segmentation|Image segmentation]] — Image segmentation partitions an image into meaningful groups of adjacent pixels (segments).
- [[segment|Segment]] — Segments group meaningful regions from pixels, used in object-based classification.
