# Classification map / Mapa de clasificación

> **level 3** · *product* · sources: th_validation

**What it is / Qué es.**

- **EN.** A classification map is a [[raster|thematic raster]] [[product|product]] that assigns each [[pixels|pixel]] a [[land-cover|land cover]] or [[land-use|land use]] [[label|label]] based on supervised or unsupervised [[analysis|analysis]] of [[satellite-imagery|satellite imagery]].
- **ES.** Un mapa de clasificación es un [[product|producto]] [[raster|ráster]] temático que asigna a cada píxel una [[label|etiqueta]] de [[land-cover|cobertura del suelo]] o [[land-use|uso del suelo]] basada en [[analysis|análisis]] supervisado o no supervisado de [[satellite-imagery|imágenes satelitales]].

## Understand these first / Entiende esto primero
- [[classification|Classification]] — *method*
    - **EN.** A classification map is a thematic map produced by applying a classification algorithm to satellite imagery.
    - **ES.** Un mapa de clasificación es un mapa temático producido al aplicar un algoritmo de clasificación a imágenes satelitales.
- [[crop-type-map|Crop type map]] — *product*
    - **EN.** A crop-type map is a classification map that focuses on differentiating crop types using spectral-temporal signatures.
    - **ES.** Un mapa de tipos de cultivo es un mapa de clasificación que se enfoca en diferenciar tipos de cultivos usando firmas espectro-temporales.
