# Persistent storage / Almacenamiento persistente

> **level 0** · *primitive*

**What it is / Qué es.**

- **EN.** [[data|Data]] storage that retains [[information|information]] after the [[process|process]] that created it ends.
- **ES.** Almacenamiento de [[data|datos]] que retiene la [[information|información]] después de que finaliza el [[process|proceso]] que la creó.

## This unlocks / Esto habilita
- [[data-volumes|DataVolumes]] — They provide persistent storage for containerized applications in a cluster.
