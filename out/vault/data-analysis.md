# Data Analysis / Análisis de datos

> **level 3** · *method* · sources: th_machine_learning

**What it is / Qué es.**

- **EN.** Data analysis involves examining, cleaning, transforming, and [[modeling|modeling]] [[data|data]] to discover useful [[information|information]] and support decision-making. In this [[handbook|handbook]], it refers to [[statistical|statistical]] and [[machine-learning|machine learning]] techniques applied to [[satellite-image-time-series|satellite time series]].
- **ES.** El análisis de datos implica examinar, limpiar, transformar y modelar [[data|datos]] para descubrir [[information|información]] útil y apoyar la toma de decisiones. En este [[handbook|manual]], se refiere a técnicas [[statistics|estadísticas]] y de [[machine-learning|aprendizaje automático]] aplicadas a series temporales satelitales.

## Understand these first / Entiende esto primero
- [[data|Data]] — *primitive*
    - **EN.** Data analysis systematically examines data to extract information, starting from raw factual information.
    - **ES.** El análisis de datos examina sistemáticamente los datos para extraer información, partiendo de datos factuales en bruto.
- [[analysis|Analysis]] — *method*
    - **EN.** Data analysis is the systematic examination of data, extending the general concept of analysis to data.
    - **ES.** El análisis de datos es el examen sistemático de los datos, extendiendo el concepto general de análisis a los datos.
