# Function / Función

> **level 0** · *primitive* · sources: howto#dagger-sdk-implementation

**What it is / Qué es.**

- **EN.** In the [[reproducible-analysis-system|reproducible analysis system]], a Dagger SDK function that performs a build-[[time|time]] task, such as building compute [[images|images]], packaging [[data|data]] artifacts, or generating [[metadata|metadata]].
- **ES.** En el [[reproducible-analysis-system|sistema de análisis reproducible]], una función del SDK de Dagger que realiza una tarea en [[time|tiempo]] de compilación, como construir [[images|imágenes]] de cómputo, empaquetar artefactos de [[data|datos]] o generar [[metadata|metadatos]].

## This unlocks / Esto habilita
- [[reproducible-analysis-system|Reproducible analysis system]] — A function in the system is a Dagger SDK unit that encapsulates a single build-time operation.
- [[component|Component]] — Functions are atomic executables that components rely on to perform specific tasks.
