# Data code availability / Disponibilidad de datos y código

> **level 1** · *primitive* · sources: howto

**What it is / Qué es.**

- **EN.** The practice of sharing datasets and [[source|source]] code used in analyses to ensure [[transparency|transparency]] and [[reproducibility|reproducibility]].
- **ES.** La práctica de compartir conjuntos de [[data|datos]] y código [[source|fuente]] utilizados en [[analysis|análisis]] para garantizar [[transparency|transparencia]] y [[reproducibility|reproducibilidad]].

## Understand these first / Entiende esto primero
- [[transparency|Transparency]] — *primitive*
    - **EN.** Data code availability ensures transparency in research by making inputs and scripts accessible.
    - **ES.** La disponibilidad de datos y código garantiza la transparencia en la investigación al hacer accesibles las entradas y los scripts.

## This unlocks / Esto habilita
