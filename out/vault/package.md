# Package / Paquete

> **level 1** · *software* · sources: th_data_sources

**What it is / Qué es.**

- **EN.** A [[collection|collection]] of reusable [[software|software]] functions and datasets bundled for a specific purpose, e.g., the [[satellite-image-time-series-2|sits]] [[r-package|R package]] for [[satellite-image-time-series|satellite image time series]] [[analysis|analysis]] and [[land-classification|land classification]].
- **ES.** [[collection|Colección]] de funciones de software reutilizables y conjuntos de [[data|datos]] empaquetados para un propósito específico, por [[example|ejemplo]], el paquete sits de R para [[time-series-analysis|análisis de series temporales]] de [[satellite-imagery|imágenes satelitales]] y [[land-classification|clasificación de tierras]].

## Understand these first / Entiende esto primero
- [[software|Software]] — *primitive*
    - **EN.** A package bundles reusable software components and data for a specific analysis, like the sits R package.
    - **ES.** Un paquete agrupa componentes de software reutilizables y datos para un análisis específico, como el paquete sits de R.
