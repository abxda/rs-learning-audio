# Grid / Cuadrícula

> **level 2** · *data* · sources: th_data_sources

**What it is / Qué es.**

- **EN.** A spatial partition [[system|system]], such as [[military-grid-reference-system|MGRS]] or WRS-2, that divides the [[earth|Earth]]'s surface into regular cells for organizing [[satellite-imagery|satellite imagery]] into [[analysis|analysis]]-ready [[tiles|tiles]].
- **ES.** [[system|Sistema]] de partición espacial, como [[military-grid-reference-system|MGRS]] o WRS-2, que divide la [[earth|superficie terrestre]] en celdas regulares para organizar [[satellite-imagery|imágenes satelitales]] en [[tiles|teselas]] listas para [[analysis|análisis]].

## Understand these first / Entiende esto primero
- [[system|System]] — *other*
    - **EN.** A grid is a spatial system of regular cells used to partition and index Earth observation data.
    - **ES.** Una cuadrícula es un sistema espacial de celdas regulares utilizado para dividir e indexar datos de observación de la Tierra.
- [[tiles|Tiles]] — *data*
    - **EN.** Tiles are the individual cells within a grid that partition satellite imagery into manageable spatial units.
    - **ES.** Las teselas son las celdas individuales dentro de una cuadrícula que dividen las imágenes satelitales en unidades espaciales manejables.

## This unlocks / Esto habilita
- [[spatial-datasets|Spatial datasets]] — A grid organizes spatial datasets into regular cells for systematic analysis and efficient retrieval.
- [[raster|Raster]] — A raster organizes spatial data into a regular grid of cells (pixels) for analysis.
- [[military-grid-reference-system|Military Grid Reference System]] — MGRS is a global tiling system that divides the Earth into 60 UTM zones, each further divided into grid cells.
- [[tiling-system|Tiling system]] — A tiling system divides the Earth into regular grid cells for managing large satellite image collections.
- [[area-frame|Area frame]] — Area frames use a grid to partition the land into contiguous cells for random sampling.
