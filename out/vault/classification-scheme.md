# Classification scheme / Esquema de clasificación

> **level 1** · *method* · sources: th_lucc

**What it is / Qué es.**

- **EN.** A set of rules, definitions, and hierarchies that organize landscape features into discrete, quantifiable categories, enabling consistent labeling of [[satellite-imagery|satellite imagery]].
- **ES.** [[set|Conjunto]] de reglas, definiciones y jerarquías que organizan las características del paisaje en categorías discretas y cuantificables, permitiendo un etiquetado consistente de las [[satellite-imagery|imágenes satelitales]].

## Understand these first / Entiende esto primero
- [[classes|Classes]] — *data*
    - **EN.** A classification scheme defines the set of classes used to label land cover features.
    - **ES.** Un esquema de clasificación define el conjunto de clases utilizado para etiquetar las características de cobertura terrestre.
- [[level|Level]] — *primitive*
    - **EN.** A level in a classification scheme indicates increasing detail or specificity within a hierarchy.
    - **ES.** Un nivel en un esquema de clasificación indica detalle o especificidad creciente dentro de una jerarquía.
- [[type|Type]] — *primitive*
    - **EN.** A type is a category in a classification scheme, such as a land cover class or crop species.
    - **ES.** Un tipo es una categoría en un esquema de clasificación, como una clase de cobertura del suelo o una especie de cultivo.
