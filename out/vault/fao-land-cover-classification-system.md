# FAO Land Cover Classification System / Sistema de Clasificación de la Cobertura del Suelo de la FAO

> **level 4** · *standard* · sources: th_lucc

**What it is / Qué es.**

- **EN.** A standardized [[framework|framework]] by FAO for classifying [[land-cover|land cover]], using a hierarchical set of diagnostic criteria to ensure interoperability.
- **ES.** Un [[framework|marco]] estandarizado de la FAO para clasificar la [[land-cover|cobertura del suelo]], que utiliza un [[set|conjunto]] jerárquico de criterios de diagnóstico para garantizar la interoperabilidad.

## Understand these first / Entiende esto primero
- [[classes|Classes]] — *data*
    - **EN.** The FAO Land Cover Classification System defines hierarchical classes and diagnostic criteria for land cover.
    - **ES.** El Sistema de Clasificación de la Cobertura del Suelo de la FAO define clases jerárquicas y criterios diagnósticos para la cobertura del suelo.
- [[land|Land]] — *phenomenon*
    - **EN.** The FAO Land Cover Classification System provides a standardized framework to classify land cover types.
    - **ES.** El Sistema de Clasificación de la Cobertura del Suelo de la FAO proporciona un marco estandarizado para clasificar tipos de cobertura del suelo.
- [[classification-systems|Classification Systems]] — *method*
    - **EN.** The FAO Land Cover Classification System extends classification systems with hierarchical diagnostic criteria.
    - **ES.** El Sistema de Clasificación de la Cobertura del Suelo de la FAO extiende los sistemas de clasificación con criterios diagnósticos jerárquicos.
