# Collection / Colección

> **level 2** · *method* · sources: th_data_sources

**What it is / Qué es.**

- **EN.** A set of [[satellite|satellite]] [[images|images]] or other [[data|data]] items sharing a common [[source|source]], format, and theme, often organized for access.
- **ES.** Un [[set|conjunto]] de [[satellite-imagery|imágenes satelitales]] u otros [[data|datos]] que comparten una [[source|fuente]], formato y tema común, a menudo organizados para su acceso.

## Understand these first / Entiende esto primero
- [[dataset|Dataset]] — *data*
    - **EN.** A collection is a set of datasets or satellite images sharing a common source, format, and theme.
    - **ES.** Una colección es un conjunto de conjuntos de datos o imágenes satelitales que comparten una fuente, formato y tema común.
- [[satellite|Satellite]] — *platform*
    - **EN.** A collection organizes satellite images from specific missions into usable groups.
    - **ES.** Una colección organiza imágenes satelitales en grupos utilizables.
