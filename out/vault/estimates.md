# Estimates / Estimaciones

> **level 2** · *statistic* · sources: th_validation

**What it is / Qué es.**

- **EN.** Numerical approximations of [[population|population]] quantities, such as [[crop-area|crop area]] or [[yield|yield]], derived from [[samples|sample]] [[data|data]] and often corrected using [[remote-sensing|remote sensing]] maps.
- **ES.** Aproximaciones numéricas de cantidades poblacionales, como el [[crop-area|área de cultivo]] o el [[yield|rendimiento]], derivadas de [[data|datos]] de muestra y a menudo corregidas mediante mapas de [[remote-sensing|teledetección]].

## Understand these first / Entiende esto primero
- [[statistics|Statistics]] — *statistic*
    - **EN.** Statistics provides the framework for producing reliable estimates from sample data.
    - **ES.** La estadística proporciona el marco para producir estimaciones fiables a partir de datos de muestra.

## This unlocks / Esto habilita
- [[estimation|Estimation]] — Estimation is the process of deriving numerical approximations for unknown population quantities.
- [[confidence-intervals|Confidence Intervals]] — Confidence intervals provide a range around an estimate that likely contains the true population value.
