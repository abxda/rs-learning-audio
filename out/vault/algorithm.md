# Algorithm / Algoritmo

> **level 4** · *method* · sources: th_machine_learning · th_quality_control

**What it is / Qué es.**

- **EN.** A defined set of [[step|step]]-by-step instructions or rules designed to perform a specific task, such as [[classification|classification]] or [[regression|regression]].
- **ES.** Un [[set|conjunto]] definido de instrucciones [[step|paso]] a paso o reglas diseñadas para realizar una tarea específica, como [[classification|clasificación]] o [[regression|regresión]].

## Understand these first / Entiende esto primero
- [[methods|Methods]] — *method*
    - **EN.** Methods are systematic techniques, often implemented through algorithms, for processing data.
    - **ES.** Los métodos son técnicas sistemáticas, a menudo implementadas mediante algoritmos, para procesar datos.
- [[machine-learning|Machine learning]] — *method*
    - **EN.** Machine learning involves algorithms that improve through experience with data.
    - **ES.** El aprendizaje automático involucra algoritmos que mejoran a través de la experiencia con datos.
- [[step|Step]] — *primitive*
    - **EN.** An algorithm is a defined set of step-by-step instructions or rules designed to perform a task.
    - **ES.** Un algoritmo es un conjunto definido de instrucciones o reglas paso a paso diseñadas para realizar una tarea.
- [[parameters|Parameters]] — *primitive*
    - **EN.** Parameters adjust how an algorithm runs, like the number of neighbors in a classifier.
    - **ES.** Los parámetros ajustan cómo funciona un algoritmo, como el número de vecinos en un clasificador.
