# Remote sensing / Teledetección

> **level 2** · *method* · sources: th_remote_sensing · introduction

**What it is / Qué es.**

- **EN.** The science of obtaining [[information|information]] about objects or areas from a distance, typically from satellites or aircraft, using [[electromagnetic-radiation|electromagnetic radiation]].
- **ES.** La ciencia de obtener [[information|información]] sobre objetos o áreas a distancia, típicamente desde satélites o aeronaves, utilizando [[electromagnetic-radiation|radiación electromagnética]].

## Understand these first / Entiende esto primero
- [[satellite|Satellite]] — *platform*
    - **EN.** Remote sensing obtains information about Earth's surface from a distance, often using satellites.
    - **ES.** La teledetección obtiene información sobre la superficie de la Tierra a distancia, a menudo utilizando satélites.

## This unlocks / Esto habilita
- [[sensors|Sensors]] — Remote sensing uses sensors on satellites or aircraft to detect electromagnetic radiation.
- [[geoinformatics|Geoinformatics]] — It integrates remote sensing data with spatial analysis methods for Earth observation.
- [[remote-sensing-estimation|Remote sensing estimation]] — Remote sensing provides satellite imagery from which agricultural parameters can be estimated.
- [[radar|Radar]] — Radar is a remote sensing technique that uses actively emitted microwave signals instead of passive solar radiation.
- [[crop-assessment|Crop Assessment]] — Remote sensing provides spectral and temporal data essential for crop assessment over large areas.
- [[remote-sensing-science|Remote Sensing Science]] — Remote sensing science is the academic discipline underlying the practice of remote sensing.
- [[earth-observation|Earth Observation]] — Earth Observation is the application of remote sensing to monitor and study the planet's surface.
- [[satellite-sensors|Satellite sensors]] — In remote sensing, satellite sensors are the key hardware that record data from a distance.
