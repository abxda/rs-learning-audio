# Data processing / Procesamiento de datos

> **level 2** · *method* · sources: th_data_cubes · ct_poland

**What it is / Qué es.**

- **EN.** The conversion of raw [[satellite-data|satellite data]] into usable [[information|information]] through steps like [[geometric-correction|geometric correction]], [[cloud-masking|cloud masking]], and [[temporal-compositing|temporal compositing]].
- **ES.** La conversión de [[satellite-data|datos satelitales]] brutos en [[information|información]] utilizable mediante [[stages|etapas]] como [[geometric-correction|corrección geométrica]], [[cloud-masking|enmascaramiento de nubes]] y [[temporal-compositing|composición temporal]].

## Understand these first / Entiende esto primero
- [[processing|Processing]] — *method*
    - **EN.** Data processing is the generic term for the steps applied to any data, including raw satellite data.
    - **ES.** El procesamiento de datos es el término genérico para los pasos aplicados a cualquier dato, incluidos los datos satelitales en bruto.
