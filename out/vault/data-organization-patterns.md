# Data Organization Patterns / Patrones de organización de datos

> **level 3** · *method* · sources: howto

**What it is / Qué es.**

- **EN.** Standard ways to store [[chapter|chapter]]-specific and shared [[data|data]] within a [[project|project]]. [[chapter-specific-data|Chapter-specific data]] is placed in `data/<chapter>/` directories, while [[shared-teaching-artifacts|shared teaching artifacts]] are in an `etc/` directory.
- **ES.** Formas estándar de almacenar [[chapter-specific-data|datos específicos del capítulo]] y compartidos dentro de un [[project|proyecto]]. Los datos específicos del capítulo se colocan en directorios `data/<chapter>/`, mientras que los [[shared-teaching-artifacts|recursos didácticos compartidos]] se encuentran en un directorio `etc/`.

## Understand these first / Entiende esto primero
- [[project|Project]] — *primitive*
    - **EN.** In a project, these patterns separate chapter-specific and shared data to ensure reproducibility.
    - **ES.** En un proyecto, estos patrones separan los datos específicos del capítulo y los datos compartidos para garantizar la reproducibilidad.
- [[database|Database]] — *software*
    - **EN.** Data organization patterns define standard ways to structure data within a database for a project.
    - **ES.** Los patrones de organización de datos definen formas estándar de estructurar datos dentro de una base de datos para un proyecto.
