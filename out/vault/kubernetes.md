# Kubernetes / Kubernetes

> **level 0** · *primitive*

**What it is / Qué es.**

- **EN.** Open-[[source|source]] platform for automating deployment and scaling of containerized apps.
- **ES.** Plataforma de código abierto para automatizar el despliegue y escalado de [[applications|aplicaciones]] en contenedores.

## This unlocks / Esto habilita
- [[data-volumes|DataVolumes]] — DataVolumes are a Kubernetes resource that manages importing and persistent storage of data.
- [[csi-image-driver|CSI Image Driver]] — In a Kubernetes cluster, the CSI Image Driver exposes container images as storage volumes to pods.
