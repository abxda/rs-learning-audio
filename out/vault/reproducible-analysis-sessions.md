# Reproducible analysis sessions / Sesiones de análisis reproducibles

> **level 0** · *method* · sources: howto

**What it is / Qué es.**

- **EN.** One-click [[jupyterlab|JupyterLab]] environments that replicate a [[chapter|chapter]]'s [[analysis|analysis]]. They are pre-configured with all code, [[data|data]], dependencies, and [[cloud|cloud]] credentials, ensuring that readers can verify and build upon the [[results|results]].
- **ES.** Entornos [[jupyterlab|JupyterLab]] de un solo clic que replican el [[analysis|análisis]] de un [[chapter|capítulo]]. Están preconfigurados con todo el código, [[data|datos]], dependencias y credenciales en la [[cloud|nube]], lo que garantiza que los lectores puedan verificar y desarrollar los [[results|resultados]].

## This unlocks / Esto habilita
- [[oci-data-artifacts|OCI Data Artifacts]] — OCI data artifacts enable reproducible analysis sessions by providing versioned, self-contained data that can be mounted in JupyterLab environments.
- [[jupyterlab|JupyterLab]] — Reproducible analysis sessions are one-click JupyterLab environments pre-configured with code, data, and dependencies.
- [[reproducibility|Reproducibility]] — Such sessions ensure reproducibility by encapsulating all software and data needed to re-run an analysis.
