# Clustering / Agrupamiento

> **level 6** · *method* · sources: th_quality_control

**What it is / Qué es.**

- **EN.** A [[machine-learning|machine learning]] technique that groups a set of objects (e.g., [[satellite-imagery|satellite image]] [[pixels|pixels]] or [[time-series|time series]]) into clusters such that objects within the same [[cluster|cluster]] are more similar to each other than to those in other clusters. It is used for pattern discovery, [[data|data]] summarization, and [[samples|sample]] [[quality-control|quality control]].
- **ES.** Una técnica de [[machine-learning|aprendizaje automático]] que agrupa un [[set|conjunto]] de objetos (por [[example|ejemplo]], [[pixels|píxeles]] de [[satellite-imagery|imágenes satelitales]] o series temporales) en clústeres de modo que los objetos dentro del mismo [[cluster|clúster]] sean más similares entre sí que con los de otros clústeres. Se utiliza para descubrimiento de [[patterns|patrones]], resumen de [[data|datos]] y [[quality-control|control de calidad]] de [[samples|muestras]].

## Understand these first / Entiende esto primero
- [[cluster|Cluster]] — *method*
    - **EN.** Clustering is the process of partitioning data into clusters where points are similar to each other.
    - **ES.** El agrupamiento es el proceso de dividir datos en clústeres donde los puntos son similares entre sí.
- [[self-organizing-maps|Self-Organizing Maps]] — *method*
    - **EN.** Self-organizing maps are an unsupervised method that clusters high-dimensional data into a low-dimensional grid.
    - **ES.** Los mapas autoorganizados son un método no supervisado que agrupa datos de alta dimensión en una cuadrícula de baja dimensión.
