# Classification framework / Marco de clasificación

> **level 4** · *method* · sources: th_lucc · th_lucc

**What it is / Qué es.**

- **EN.** A structured [[approach|approach]] that defines the rules, hierarchies, and [[classes|classes]] used to assign [[label|labels]] to [[remote-sensing|remote sensing]] [[data|data]] based on [[land-cover|land cover]] or [[land-use|land use]].
- **ES.** Un [[approach|enfoque]] estructurado que define las reglas, jerarquías y [[classes|clases]] utilizadas para asignar etiquetas a [[data|datos]] de [[remote-sensing|teledetección]] según la [[cover|cobertura]] o el [[land-use|uso del suelo]].

## Understand these first / Entiende esto primero
- [[classes|Classes]] — *data*
    - **EN.** It defines how classes are structured and related in a hierarchical system.
    - **ES.** Define cómo se estructuran y relacionan las clases en un sistema jerárquico.
- [[classification-systems|Classification Systems]] — *method*
    - **EN.** Classification systems are structured frameworks such as FAO LCCS with rules and hierarchies for assigning semantic labels to land cover.
    - **ES.** Los sistemas de clasificación son marcos estructurados como el LCCS de la FAO con reglas y jerarquías para asignar etiquetas semánticas a la cobertura terrestre.
- [[classification-scheme|Classification scheme]] — *method*
    - **EN.** A classification framework organizes a classification scheme into rules and hierarchies.
    - **ES.** Un marco de clasificación organiza un esquema de clasificación en reglas y jerarquías.
