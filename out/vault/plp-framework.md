# PLP Framework / marco PLP

> **level 2** · *method* · sources: ad_learning

**What it is / Qué es.**

- **EN.** A digital [[infrastructure|infrastructure]] linking user profiles, [[knowledge-areas|knowledge areas]], [[machine-learning|learning]] modules, and a course catalogue to create personalized [[training|training]] programs for [[earth-observation|Earth observation]] and [[agricultural-statistics|agricultural statistics]].
- **ES.** [[infrastructure|Infraestructura]] digital que vincula perfiles de usuario, [[knowledge-areas|áreas de conocimiento]], módulos de aprendizaje y un catálogo de cursos para crear programas de capacitación personalizados en [[earth-observation|observación de la Tierra]] y [[agricultural-statistics|estadísticas agrícolas]].

## Understand these first / Entiende esto primero
- [[framework|Framework]] — *primitive*
    - **EN.** The PLP Framework extends a structured framework by linking user profiles and learning modules to customize education.
    - **ES.** El marco PLP extiende un marco estructurado vinculando perfiles de usuario y módulos de aprendizaje para personalizar la educación.
