# Sampling frame / Marco de muestreo

> **level 7** · *method* · sources: th_design_frames#outline

**What it is / Qué es.**

- **EN.** An ordered list or spatial structure from which a [[statistical|statistical]] [[samples|sample]] is selected; for [[agricultural-surveys|agricultural surveys]], an [[area-frame|area frame]] or [[list-frame|list frame]] is used, and can be improved with [[earth-observation|Earth Observation data]].
- **ES.** Una lista ordenada o estructura espacial de la cual se selecciona una muestra estadística; para [[agricultural-surveys|encuestas agrícolas]], se utiliza un [[area-frame|marco de área]] o un [[list-frame|marco de lista]], y puede mejorarse con [[data|datos]] de [[earth-observation|observación de la Tierra]].

## Understand these first / Entiende esto primero
- [[sampling|Sampling]] — *method*
    - **EN.** A sampling frame is the list or spatial structure from which a sample is selected.
    - **ES.** Un marco de muestreo es la lista o estructura espacial de la cual se selecciona una muestra.
- [[sample-design|Sample design]] — *method*
    - **EN.** Sample design requires a sampling frame from which units are selected following a defined procedure.
    - **ES.** El diseño muestral requiere un marco de muestreo del cual se seleccionan unidades siguiendo un procedimiento definido.
- [[robust-sampling-frame|Robust sampling frame]] — *method*
    - **EN.** A robust sampling frame is a sampling frame that is complete, accurate, and up-to-date.
    - **ES.** Un marco muestral robusto es aquel que es completo, preciso y actualizado.
- [[population|Population]] — *statistic*
    - **EN.** The sampling frame must represent the entire target population for valid inference.
    - **ES.** El marco de muestreo debe representar a toda la población objetivo para una inferencia válida.
