# Estimation / Estimación

> **level 3** · *method* · sources: th_validation · crop_statistics_area

**What it is / Qué es.**

- **EN.** The [[process|process]] of inferring the value of an unknown quantity, such as [[crop-yield|crop yield]] or [[area|area]], using [[statistical|statistical]] or [[machine-learning|machine learning]] [[methods|methods]].
- **ES.** El [[process|proceso]] de inferir el valor de una cantidad desconocida, como el [[crop-yield|rendimiento de cultivos]] o el [[area|área]], utilizando [[statistical-methods|métodos estadísticos]] o de [[machine-learning|aprendizaje automático]].

## Understand these first / Entiende esto primero
- [[statistics|Statistics]] — *statistic*
    - **EN.** Estimation uses statistical methods to infer unknown quantities like crop area from sample data.
    - **ES.** La estimación utiliza métodos estadísticos para inferir cantidades desconocidas como el área de cultivo a partir de datos de muestra.
- [[estimates|Estimates]] — *statistic*
    - **EN.** Estimation is the process of deriving numerical approximations for unknown population quantities.
    - **ES.** La estimación es el proceso de derivar aproximaciones numéricas para cantidades poblacionales desconocidas.

[[data|Data]] → [[statistics|Statistics]] → [[estimates|Estimates]] → [[estimation|Estimation]]

## This unlocks / Esto habilita
- [[estimator|Estimator]] — An estimator is a statistical rule for estimating a population parameter from sample data.
- [[area-estimation|Area estimation]] — Area estimation applies statistical methods to estimate the spatial extent of land cover or crop types from sample data.
- [[remote-sensing-estimation|Remote sensing estimation]] — Estimation infers unknown quantities like crop area from observed data such as satellite imagery.
- [[reliable-statistical-analyses|Reliable statistical analyses]] — Reliable statistical analyses combine map predictions with design-based estimators to produce unbiased estimates.
- [[statistics-production|Statistics production]] — Statistics production generates official estimates of crop area and yield using rigorous estimation methods.
- [[survey-approach|Survey approach]] — The survey approach is a pure estimation method that does not incorporate satellite map predictions.
- [[crop-yield-estimation|Crop yield estimation]] — Crop yield estimation is the process of inferring yield values from satellite and ground data.
- [[sample-based-estimators|Sample-based estimators]] — Sample-based estimators use data from a probability sample to estimate population parameters like total area.
- [[crop-area|Crop area]] — Crop area estimation uses statistical methods to infer total planted area from samples.
