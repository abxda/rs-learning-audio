# Machine learning models / Modelos de aprendizaje automático

> **level 4** · *method* · sources: th_machine_learning

**What it is / Qué es.**

- **EN.** [[algorithm|Algorithms]] trained on [[data|data]] to make [[predictions|predictions]] or decisions. For [[satellite-image-classification|satellite image classification]], they learn to assign [[land-cover|land cover]] [[label|labels]] based on spectral and temporal [[patterns|patterns]]. Examples include [[random-forest|Random Forest]] and TempCNN.
- **ES.** Algoritmos entrenados con [[data|datos]] para realizar [[predictions|predicciones]] o tomar decisiones. Para la [[satellite-image-classification|clasificación de imágenes satelitales]], aprenden a asignar etiquetas de [[land-cover|cobertura del suelo]] basándose en [[patterns|patrones]] espectrales y temporales. Los ejemplos incluyen [[random-forest|Random Forest]] y TempCNN.

## Understand these first / Entiende esto primero
- [[model|Model]] — *method*
    - **EN.** A model is a representation that learns from features and labels to perform classification or regression.
    - **ES.** Un modelo es una representación que aprende de atributos y etiquetas para realizar clasificación o regresión.
- [[machine-learning|Machine learning]] — *method*
    - **EN.** Machine learning models are algorithms trained on data to recognize patterns and make predictions.
    - **ES.** Los modelos de aprendizaje automático son algoritmos entrenados con datos para reconocer patrones y hacer predicciones.
