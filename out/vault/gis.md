# GIS / GIS

> **level 5** · *software* · sources: th_alignment

**What it is / Qué es.**

- **EN.** Geographic [[information|Information]] [[system|System]], a [[software|software]] [[framework|framework]] for capturing, storing, analyzing, and displaying [[spatial-datasets|spatial data]]; used for [[tasks|tasks]] such as [[parcels|parcel]] digitization, map creation, and [[spatial-analysis|spatial analysis]].
- **ES.** Sistema de Información Geográfica, un [[framework|marco]] de [[package|software]] para capturar, almacenar, analizar y mostrar [[geospatial-data|datos geoespaciales]]; utilizado para [[tasks|tareas]] como digitalización de [[parcels|parcelas]], creación de mapas y [[spatial-analysis|análisis espacial]].

## Understand these first / Entiende esto primero
- [[geospatial-data|Geospatial Data]] — *data*
    - **EN.** A GIS is a software system for capturing, storing, analyzing, and displaying geospatial data layers.
