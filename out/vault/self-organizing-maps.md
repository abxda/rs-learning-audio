# Self-Organizing Maps / Mapas Autoorganizados

> **level 4** · *method* · sources: th_quality_control#quality-control-using-self-organizing-maps

**What it is / Qué es.**

- **EN.** Self-Organizing Maps (SOM) are an unsupervised [[neural-networks|neural network]] technique that projects high-dimensional [[data|data]] onto a 2D [[grid|grid]] while preserving topological relationships, used for [[clustering|clustering]] and [[quality-control|quality control]] of [[training-samples|training samples]].
- **ES.** Los Mapas Autoorganizados (SOM) son una técnica no supervisada de [[neural-networks|redes neuronales]] que proyecta [[data|datos]] de alta dimensión en una [[grid|cuadrícula]] 2D preservando relaciones topológicas, utilizada para [[clustering|agrupamiento]] y [[quality-control|Control de Calidad]] de [[training-samples|Muestras de Entrenamiento]].

## Understand these first / Entiende esto primero
- [[machine-learning|Machine learning]] — *method*
    - **EN.** Self-organizing maps use a neural network to learn a topological representation of input data without labels.
    - **ES.** Los mapas autoorganizados utilizan una red neuronal para aprender una representación topológica de los datos de entrada sin etiquetas.
- [[som-analysis|SOM analysis]] — *method*
    - **EN.** SOM analysis uses Self-Organizing Maps to cluster time series and detect confusions in training data.
    - **ES.** El análisis SOM utiliza mapas autoorganizados para agrupar series temporales y detectar confusiones en los datos de entrenamiento.
