# Validation datasets / Conjuntos de datos de validación

> **level 5** · *data* · sources: th_validation · ct_zimbabwe

**What it is / Qué es.**

- **EN.** Labeled [[data|data]] independent from the [[training|training]] set used to assess the [[accuracy|accuracy]] of a [[crop-classification-models|classification model]]. They enable unbiased evaluation [[metrics|metrics]] such as [[overall-accuracy|overall accuracy]] and [[f1-score|F1-score]].
- **ES.** [[data|Datos]] etiquetados independientes del [[set|conjunto]] de [[training|entrenamiento]] utilizados para evaluar la [[accuracy|exactitud]] de un [[model|modelo]] de [[classification|clasificación]]. Permiten [[metrics|métricas]] de [[assessment|evaluación]] no sesgadas como la [[overall-accuracy|precisión general]] y la [[f1-score|puntuación F1]].

## Understand these first / Entiende esto primero
- [[reference-datasets|Reference datasets]] — *data*
    - **EN.** Validation datasets are independent reference datasets used to assess the accuracy of a classification model.
    - **ES.** Los conjuntos de datos de validación son conjuntos de datos de referencia independientes utilizados para evaluar la precisión de un modelo de clasificación.
- [[accuracy-assessment|Accuracy assessment]] — *method*
    - **EN.** They enable accuracy assessment by providing ground truth for comparison with model predictions.
    - **ES.** Permiten la evaluación de precisión al proporcionar verdad terreno para comparar con las predicciones del modelo.
