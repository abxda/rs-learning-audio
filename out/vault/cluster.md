# Cluster / Clúster

> **level 5** · *method* · sources: th_quality_control

**What it is / Qué es.**

- **EN.** A set of [[data|data]] [[points|points]] (e.g., [[satellite-image-time-series|satellite image time series]]) grouped together by a [[clustering-algorithm|clustering algorithm]] because they share similar properties, used to identify [[patterns|patterns]] or separate [[land-cover-classes|land cover classes]].
- **ES.** [[set|Conjunto]] de [[points|puntos]] de [[data|datos]] (por [[example|ejemplo]], [[satellite-image-time-series|series temporales de imágenes satelitales]]) agrupados por un [[clustering-algorithm|algoritmo de agrupamiento]] porque comparten propiedades similares, utilizado para identificar [[patterns|patrones]] o separar [[land-cover-classes|clases de cobertura del suelo]].

## Understand these first / Entiende esto primero
- [[data|Data]] — *primitive*
    - **EN.** Data points are grouped by a clustering algorithm into clusters based on similarity metrics.
    - **ES.** Los puntos de datos se agrupan mediante un algoritmo de agrupamiento en clústeres basados en métricas de similitud.
- [[algorithm|Algorithm]] — *method*
    - **EN.** A clustering algorithm partitions data into clusters of similar points.
    - **ES.** Un algoritmo de agrupamiento divide los datos en clústeres de puntos similares.
