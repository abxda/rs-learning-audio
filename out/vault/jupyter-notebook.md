# Jupyter Notebook / Jupyter Notebook

> **level 6** · *software* · sources: howto · howto

**What it is / Qué es.**

- **EN.** An interactive web-based computing [[environment|environment]] that supports R, Python, and other languages, used for reproducible [[data-analysis|data analysis]] and visualization.
- **ES.** [[environment|Entorno]] informático interactivo basado en web que admite R, Python y otros lenguajes, utilizado para el [[data-analysis|análisis de datos]] reproducible y la visualización.

## Understand these first / Entiende esto primero
- [[reproducible-code|Reproducible code]] — *software*
    - **EN.** Jupyter Notebooks enable reproducible code by combining narrative text with executable code cells.
    - **ES.** Los Jupyter Notebook permiten código reproducible al combinar texto narrativo con celdas de código ejecutable.
- [[jupyterlab|JupyterLab]] — *software*
    - **EN.** A Jupyter Notebook runs within JupyterLab, providing an interactive computing environment for analysis.
    - **ES.** Un Jupyter Notebook se ejecuta dentro de JupyterLab, proporcionando un entorno informático interactivo para el análisis.
