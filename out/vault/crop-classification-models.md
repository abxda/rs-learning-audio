# Crop classification models / Modelos de clasificación de cultivos

> **level 4** · *model* · sources: th_machine_learning

**What it is / Qué es.**

- **EN.** Crop classification models are [[machine-learning-algorithms|machine learning algorithms]] trained to identify [[crop-types|crop types]] from [[satellite-image-time-series|satellite image time series]]. Examples include [[random-forest|Random Forest]], XGBoost, and [[deep-learning|deep learning]] architectures like TempCNN and ResNet.
- **ES.** Los modelos de clasificación de cultivos son [[machine-learning-algorithms|algoritmos de aprendizaje automático]] entrenados para identificar [[crop-types|tipos de cultivos]] a partir de [[satellite-image-time-series|series temporales de imágenes satelitales]]. Ejemplos incluyen [[random-forest|Random Forest]], XGBoost y arquitecturas de [[deep-learning|aprendizaje profundo]] como TempCNN y ResNet.

## Understand these first / Entiende esto primero
- [[crop|Crop]] — *phenomenon*
    - **EN.** These models are trained to identify specific crop types from satellite imagery.
    - **ES.** Estos modelos se entrenan para identificar tipos específicos de cultivos a partir de imágenes satelitales.
- [[classification|Classification]] — *method*
    - **EN.** Crop classification models are machine learning algorithms that perform classification on image data.
    - **ES.** Los modelos de clasificación de cultivos son algoritmos de aprendizaje automático que realizan clasificación sobre datos de imágenes.
- [[machine-learning|Machine learning]] — *method*
    - **EN.** Machine learning provides the algorithms that learn patterns from labeled samples.
    - **ES.** El aprendizaje automático proporciona los algoritmos que aprenden patrones a partir de muestras etiquetadas.
