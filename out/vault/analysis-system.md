# Analysis System / Sistema de análisis

> **level 1** · *software* · sources: howto · ct_zimbabwe

**What it is / Qué es.**

- **EN.** A [[software|software]] [[framework|framework]] that provides an integrated [[workflow|workflow]] for [[processing|processing]] [[earth-observation|Earth observation data]], from [[data|data]] access to [[classification|classification]] and [[mapping|mapping]], often with [[reproducibility|reproducibility]] features.
- **ES.** Un [[framework|marco]] de [[package|software]] que proporciona un [[workflow|flujo de trabajo]] integrado para procesar [[data|datos]] de [[earth-observation|observación de la Tierra]], desde el acceso a datos hasta la [[classification|clasificación]] y el [[mapping|mapeo]], a menudo con características de [[reproducibility|reproducibilidad]].

## Understand these first / Entiende esto primero
- [[system|System]] — *other*
    - **EN.** An analysis system is a system designed to provide an integrated workflow for processing Earth observation data.
