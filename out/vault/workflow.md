# Workflow / Flujo de trabajo

> **level 4** · *method* · sources: ct_mexico#methods

**What it is / Qué es.**

- **EN.** A sequence of [[processing|processing]] steps for analyzing [[remote-sensing|remote sensing]] [[data|data]], such as loading [[imagery|imagery]], classifying [[pixels|pixels]], and validating [[results|results]].
- **ES.** Una secuencia de pasos de [[processing|procesamiento]] para analizar [[data|datos]] de [[remote-sensing|teledetección]], como cargar [[images|imágenes]], clasificar [[pixels|píxeles]] y validar [[results|resultados]].

## Understand these first / Entiende esto primero
- [[process|Process]] — *method*
    - **EN.** A workflow is a specific sequence of steps (a process) that is formally defined to analyze data.
    - **ES.** Un flujo de trabajo es una secuencia específica de etapas (un proceso) que se define formalmente para analizar datos.

[[system|System]] → [[methods|Methods]] → [[step|Step]] → [[process|Process]] → [[workflow|Workflow]]

## This unlocks / Esto habilita
- [[classification-workflows|Classification workflows]] — Classification workflows are a specific type of workflow that sequences steps for assigning labels.
- [[reproducible-code|Reproducible code]] — Reproducible code encapsulates the entire workflow from data loading to figure generation in a shareable script.
