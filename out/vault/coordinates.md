# Coordinates / Coordenadas

> **level 0** · *primitive*

**What it is / Qué es.**

- **EN.** Numerical [[values|values]] defining a position in a [[coordinate-system|coordinate system]].
- **ES.** [[values|Valores]] numéricos que definen una posición en un [[coordinate-system|sistema de coordenadas]].

## This unlocks / Esto habilita
- [[coordinate-reference-system|Coordinate Reference System]] — A coordinate reference system (CRS) defines how geographic coordinates map to positions on the Earth.
- [[geographic-data|Geographic data]] — Geographic data often uses coordinates to define positions of features.
- [[coordinate-system|Coordinate system]] — A coordinate system defines how coordinates are assigned to points on Earth, enabling georeferencing.
