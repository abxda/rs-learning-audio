# Data Division / División de datos

> **level 4** · *method* · sources: th_quality_control

**What it is / Qué es.**

- **EN.** The [[process|process]] of splitting a [[dataset|dataset]] into separate parts, such as [[training|training]], [[validation|validation]], and testing sets, to develop and evaluate [[machine-learning-models|machine learning models]].
- **ES.** El [[process|proceso]] de dividir un [[dataset|conjunto de datos]] en partes separadas, como conjuntos de [[training|entrenamiento]], [[validation|validación]] y prueba, para desarrollar y evaluar [[machine-learning-models|modelos de aprendizaje automático]].

## Understand these first / Entiende esto primero
- [[dataset|Dataset]] — *data*
    - **EN.** Data division splits a dataset into training, validation, and test subsets for model development.
    - **ES.** La división de datos separa un conjunto de datos en subconjuntos de entrenamiento, validación y prueba para el desarrollo de modelos.
- [[validation|Validation]] — *method*
    - **EN.** Data division reserves a validation set to tune model parameters and prevent overfitting.
    - **ES.** La división de datos reserva un conjunto de validación para ajustar los parámetros del modelo y evitar el sobreajuste.
