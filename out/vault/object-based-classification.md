# Object-based classification / Clasificación basada en objetos

> **level 3** · *method* · sources: ct_mexico

**What it is / Qué es.**

- **EN.** A [[classification|classification]] [[approach|approach]] that analyzes image objects (segments) rather than individual [[pixels|pixels]]. Objects are created by aggregating pixels into homogeneous [[regions|regions]], often corresponding to real-world features like [[agricultural|agricultural]] [[parcels|parcels]].
- **ES.** [[approach|Enfoque]] de [[classification|clasificación]] que analiza objetos de imagen (segmentos) en lugar de [[pixels|píxeles]] individuales. Los objetos se crean agregando píxeles en [[regions|regiones]] homogéneas, a menudo correspondientes a características del mundo real como [[agricultural-plots|parcelas agrícolas]].

## Understand these first / Entiende esto primero
- [[classification|Classification]] — *method*
    - **EN.** Object-based classification improves upon pixel-based classification by using image objects as the unit of analysis.
    - **ES.** La clasificación basada en objetos mejora la clasificación basada en píxeles al usar objetos de imagen como unidad de análisis.
- [[segmentation|Segmentation]] — *method*
    - **EN.** Object-based classification first segments an image into objects (groups of pixels) and then classifies each object.
    - **ES.** La clasificación basada en objetos primero segmenta una imagen en objetos (grupos de píxeles) y luego clasifica cada objeto.
