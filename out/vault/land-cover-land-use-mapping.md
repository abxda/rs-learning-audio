# LULC mapping / mapeo LULC

> **level 0** · *method* · sources: ct_chile#introduction

**What it is / Qué es.**

- **EN.** The [[process|process]] of using [[satellite-imagery|satellite imagery]] to create maps that classify [[land-cover|land cover]] (e.g., [[forest|forest]]) and [[land-use|land use]] (e.g., [[agriculture|agriculture]]), essential for [[agricultural-statistics|agricultural statistics]] and [[environmental-monitoring|environmental monitoring]].
- **ES.** El [[process|proceso]] de utilizar [[satellite-imagery|imágenes satelitales]] para crear mapas que clasifican la [[land-cover|cobertura del suelo]] (por [[example|ejemplo]], [[forest|bosque]]) y el [[land-use|uso del suelo]] (por ejemplo, [[agriculture|agricultura]]), esencial para las [[agricultural-statistics|estadísticas agrícolas]] y el [[environmental-monitoring|monitoreo ambiental]].

## This unlocks / Esto habilita
- [[land-cover-mapping|Land cover mapping]] — LULC mapping extends land cover mapping by also classifying the socioeconomic function of the land.
- [[land-use-mapping|Land use mapping]] — LULC mapping integrates both biophysical cover and human-use categories into a single thematic map.
