# Framework / Marco

> **level 1** · *primitive* · sources: th_alignment · ad_world_cereal

**What it is / Qué es.**

- **EN.** A structured set of guidelines, standards, or [[tools|tools]] that support consistent and [[reproducible-analysis-system|reproducible analysis]], such as the FAO in-situ [[data|data]] [[quality-framework|quality framework]] or the WorldCereal [[crop-mapping|crop mapping]] [[system|system]].
- **ES.** [[set|Conjunto]] estructurado de directrices, estándares o [[tools|herramientas]] que apoyan un [[analysis|análisis]] consistente y reproducible, como el [[quality-framework|marco de calidad]] de [[data|datos]] in situ de la FAO o el [[system|sistema]] de [[crop-mapping|mapeo de cultivos]] de WorldCereal.

## Understand these first / Entiende esto primero

    - **EN.** A framework provides a structured set of guidelines and standards, similar to a system but more conceptual.
    - **ES.** Un marco proporciona un conjunto estructurado de directrices y estándares, similar a un sistema pero más conceptual.
