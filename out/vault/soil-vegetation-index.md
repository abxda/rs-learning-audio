# Soil Vegetation Index / Índice de vegetación de suelo

> **level 5** · *index* · sources: uav_crop_growth#methods

**What it is / Qué es.**

- **EN.** A [[spectral-index|spectral index]], such as OSAVI, that minimizes [[soil|soil]] background effects to better estimate [[vegetation|vegetation]] [[cover|cover]] and vigor from [[remote-sensing|remote sensing]] [[data|data]].
- **ES.** [[spectral-index|Índice espectral]], como el OSAVI, que minimiza los efectos del fondo del [[ground|suelo]] para estimar mejor la [[cover|cobertura]] y el vigor de la [[vegetation|vegetación]] a partir de [[data|datos]] de [[remote-sensing|teledetección]].

## Understand these first / Entiende esto primero
- [[soil|Soil]] — *phenomenon*
    - **EN.** This index adjusts for soil brightness to improve vegetation signal in areas with partial cover.
    - **ES.** Este índice ajusta el brillo del suelo para mejorar la señal de vegetación en áreas con cobertura parcial.
- [[vegetation-indices|Vegetation indices]] — *index*
    - **EN.** A soil-vegetation index (e.g., OSAVI) minimizes soil background influence to better estimate vegetation cover.
    - **ES.** Un índice de vegetación de suelo (ej., OSAVI) minimiza la influencia del fondo del suelo para estimar mejor la cobertura vegetal.
