# Classification method / Método de clasificación

> **level 5** · *method* · sources: th_machine_learning

**What it is / Qué es.**

- **EN.** A [[machine-learning|machine learning]] or [[deep-learning|deep learning]] [[algorithm|algorithm]] used to assign [[land-cover|land cover]] [[label|labels]] to [[pixels|pixels]] based on their spectral-temporal features. Examples include [[random-forest|Random Forest]], XGBoost, and TempCNN.
- **ES.** Un [[algorithm|algoritmo]] de [[machine-learning|aprendizaje automático]] o profundo utilizado para asignar etiquetas de [[land-cover|cobertura del suelo]] a los [[pixels|píxeles]] basándose en sus características espectro-temporales. Los ejemplos incluyen [[random-forest|Random Forest]], XGBoost y TempCNN.

## Understand these first / Entiende esto primero
- [[machine-learning|Machine learning]] — *method*
    - **EN.** A classification method is a machine learning algorithm that learns from samples to assign classes.
    - **ES.** Un método de clasificación es un algoritmo de aprendizaje automático que aprende de muestras para asignar clases.
- [[algorithm|Algorithm]] — *method*
    - **EN.** It is a specific algorithm—e.g., decision tree or random forest—that processes features to label pixels.
    - **ES.** Es un algoritmo específico—por ejemplo, árbol de decisión o random forest—que procesa atributos para etiquetar píxeles.
