# Geographic distribution / Distribución geográfica

> **level 4** · *phenomenon*

**What it is / Qué es.**

- **EN.** The spatial arrangement or spread of features, such as [[crop-types|crop types]] or [[land-cover-classes|land cover classes]], across a geographic [[area|area]], analyzed to identify [[patterns|patterns]] and [[variability|variability]].
- **ES.** La disposición espacial o extensión de características, como [[crop-types|tipos de cultivos]] o [[land-cover-classes|clases de cobertura del suelo]], a través de un [[area|área]] geográfica, analizada para identificar [[patterns|patrones]] y [[variability|variabilidad]].

## Understand these first / Entiende esto primero
- [[area|Area]] — *metric*
    - **EN.** Geographic distribution describes how a variable's values are spread across spatial coordinates within a region.
    - **ES.** La distribución geográfica describe cómo los valores de una variable se distribuyen entre las coordenadas espaciales dentro de una región.
- [[map|Map]] — *product*
    - **EN.** A map visually displays the geographic distribution by showing where each class or measurement occurs.
    - **ES.** Un mapa muestra visualmente la distribución geográfica indicando dónde ocurre cada clase o medición.
