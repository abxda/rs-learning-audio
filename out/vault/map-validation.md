# Map Validation / Validación de mapas

> **level 4** · *method* · sources: th_validation · th_validation · th_validation

**What it is / Qué es.**

- **EN.** The [[assessment|assessment]] of a [[classified-map|classified map]]'s [[accuracy|accuracy]] by comparing it against independent, probabilistically selected [[reference-samples|reference samples]], using [[area-weighted-error-matrices|area-weighted error matrices]].
- **ES.** La [[assessment|evaluación]] de la [[accuracy|exactitud]] de un [[classified-map|mapa clasificado]] mediante su comparación con [[reference-samples|muestras de referencia]] independientes seleccionadas probabilísticamente, utilizando [[area-weighted-error-matrices|matrices de error ponderadas por área]].

## Understand these first / Entiende esto primero
- [[validation|Validation]] — *method*
    - **EN.** Map validation is the process of assessing a map's accuracy by comparing it to reference data.
    - **ES.** La validación de mapas es el proceso de evaluar la exactitud de un mapa comparándolo con datos de referencia.
