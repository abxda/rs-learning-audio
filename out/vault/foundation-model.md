# Foundation model / Modelo fundacional

> **level 6** · *software* · sources: ad_world_cereal

**What it is / Qué es.**

- **EN.** A large pre-trained [[model|model]] that learns general representations from unlabeled [[data|data]], such as [[satellite-imagery|satellite imagery]], and can be fine-tuned for specific [[tasks|tasks]] like [[crop-mapping|crop mapping]] with limited [[labeled-samples|labeled samples]].
- **ES.** Un [[model|modelo]] preentrenado grande que aprende representaciones generales a partir de [[data|datos]] no etiquetados, como [[satellite-imagery|imágenes satelitales]], y puede ajustarse para [[tasks|tareas]] específicas como el [[crop-mapping|mapeo de cultivos]] con [[labeled-samples|muestras etiquetadas]] limitadas.

## Understand these first / Entiende esto primero
- [[training|Training]] — *method*
    - **EN.** Foundation models are trained on massive unlabeled datasets and can be fine-tuned for specific tasks.
    - **ES.** Los modelos fundacionales se entrenan con grandes conjuntos de datos no etiquetados y pueden ajustarse para tareas específicas.
- [[deep-learning|Deep learning]] — *method*
    - **EN.** A foundation model is a large pre-trained deep learning model that learns general representations from data.
    - **ES.** Un modelo fundacional es un modelo grande de aprendizaje profundo preentrenado que aprende representaciones generales a partir de datos.
