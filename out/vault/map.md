# Map / Mapa

> **level 3** · *product* · sources: th_validation

**What it is / Qué es.**

- **EN.** A thematic spatial representation, such as a [[crop-types|crop type]] or [[land-cover-map|land cover map]], produced by classifying [[satellite-imagery|satellite imagery]] to show [[geographic-distribution|geographic distribution]].
- **ES.** Representación espacial temática, como un [[crop-type-map|mapa de tipos de cultivo]] o [[land-cover|cobertura del suelo]], producida clasificando [[satellite-imagery|imágenes satelitales]] para mostrar [[geographic-distribution|distribución geográfica]].

## Understand these first / Entiende esto primero
- [[area|Area]] — *metric*
    - **EN.** Area can be estimated by counting pixels in a classified map or summing polygon extents.
    - **ES.** El área puede estimarse contando píxeles en un mapa clasificado o sumando las extensiones de los polígonos.
- [[classification|Classification]] — *method*
    - **EN.** A map is the thematic output of classification, showing spatial distribution of classes.
    - **ES.** Un mapa es el producto temático de la clasificación, que muestra la distribución espacial de las clases.
- [[images|Images]] — *data*
    - **EN.** A map is produced by classifying satellite images into thematic categories.
    - **ES.** Un mapa se produce clasificando imágenes satelitales en categorías temáticas.

[[data|Data]] → [[samples|Samples]] → [[classification|Classification]] → [[map|Map]]

## This unlocks / Esto habilita
- [[mapping|Mapping]] — Mapping is the process of producing thematic maps from remote sensing data.
- [[land-cover-map|Land cover map]] — Thematic maps like land cover maps represent classified satellite imagery spatially.
- [[map-product|Map product]] — A map product is the final output, typically a thematic map derived from satellite imagery.
- [[basemap|Basemap]] — A basemap is a georeferenced map used as a visual reference layer beneath other spatial data.
- [[segmentation-map|Segmentation map]] — A segmentation map is a thematic map showing homogeneous regions like fields or forest patches.
- [[map-approach|Map approach]] — The Map approach uses only satellite-derived map predictions, without survey data, to estimate quantities.
- [[anomaly-score-map|Anomaly score map]] — An anomaly score map is a thematic map where each pixel shows deviation from expected patterns.
- [[geographic-distribution|Geographic distribution]] — A map visually displays the geographic distribution by showing where each class or measurement occurs.
- [[thematic-map|Thematic map]] — A map that shows the distribution of classes such as land cover or crop types is called a thematic map.
- [[spatial-patterns|Spatial patterns]] — By analyzing a classified map, one can identify spatial patterns of crop distribution or land cover.
