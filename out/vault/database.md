# Database / Base de datos

> **level 2** · *software* · sources: ct_poland

**What it is / Qué es.**

- **EN.** A structured [[collection|collection]] of [[data|data]], such as [[satellite|satellite]] [[images|images]] or [[ground-observations|ground observations]], organized for efficient storage, retrieval, and [[analysis|analysis]].
- **ES.** [[collection|Colección]] estructurada de [[data|datos]], como [[satellite-imagery|imágenes satelitales]] u [[ground-observations|observaciones de campo]], organizada para el almacenamiento, recuperación y [[analysis|análisis]] eficientes.

## Understand these first / Entiende esto primero
- [[dataset|Dataset]] — *data*
    - **EN.** A database is a structured collection of datasets designed for efficient storage and retrieval.
    - **ES.** Una base de datos es una colección estructurada de conjuntos de datos diseñada para un almacenamiento y recuperación eficientes.
