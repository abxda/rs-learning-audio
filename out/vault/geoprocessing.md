# Geoprocessing / Geoprocesamiento

> **level 6** · *method* · sources: th_parcel_extraction

**What it is / Qué es.**

- **EN.** The manipulation and [[analysis|analysis]] of [[spatial-datasets|spatial data]] using specialized [[software|software]] to derive [[information|information]], including operations like [[buffering|buffering]], overlay, and spatial joins.
- **ES.** La manipulación y [[data-analysis|análisis de datos]] espaciales utilizando [[package|software]] especializado para derivar [[information|información]], incluyendo operaciones como [[buffering|búfer]], superposición y uniones espaciales.

## Understand these first / Entiende esto primero
- [[geographic-information-systems|Geographic information systems]] — *software*
    - **EN.** Geoprocessing implements the analytical functions of a geographic information system.
    - **ES.** El geoprocesamiento implementa las funciones analíticas de un sistema de información geográfica.
- [[spatial-analysis|Spatial Analysis]] — *method*
    - **EN.** Geoprocessing is a set of spatial analysis operations that manipulate geographic data.
    - **ES.** El geoprocesamiento es un conjunto de operaciones de análisis espacial que manipulan datos geográficos.
