# System / Sistema

> **level 0** · *other* · sources: cy_poland#introduction

**What it is / Qué es.**

- **EN.** A structured set of [[methods|methods]], [[tools|tools]], and workflows designed to perform a specific task, such as [[crop-mapping|crop mapping]] or [[yield-forecasting|yield forecasting]].
- **ES.** Un [[set|conjunto]] estructurado de [[methods|métodos]], [[tools|herramientas]] y flujos de trabajo diseñados para realizar una tarea específica, como el [[crop-mapping|mapeo de cultivos]] o el [[yield-forecasting|pronóstico de rendimiento]].

## This unlocks / Esto habilita
- [[methods|Methods]] — A system is a structured set of methods, tools, and workflows for a specific task.
- [[framework|Framework]] — A framework provides a structured set of guidelines and standards, similar to a system but more conceptual.
- [[statistical-systems|Statistical systems]] — Statistical systems are the structured frameworks for producing official statistics.
- [[data-systems|Data Systems]] — Data systems are integrated platforms that combine data storage, processing, and analysis tools.
- [[analysis-system|Analysis System]] — An analysis system is a system designed to provide an integrated workflow for processing Earth observation data.
- [[component|Component]] — The system defines the framework into which components are integrated as functional modules.
- [[grid|Grid]] — A grid is a spatial system of regular cells used to partition and index Earth observation data.
- [[platforms|Platforms]] — Platforms are systems that integrate data, compute, and tools for Earth observation workflows.
