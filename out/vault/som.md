# SOM / SOM

> **level 7** · *method* · sources: th_quality_control

**What it is / Qué es.**

- **EN.** Self-Organizing Map: an unsupervised [[neural-networks|neural network]] that reduces high-dimensional [[data|data]] (e.g., [[satellite-image-time-series|satellite image time series]]) to a two-dimensional [[grid|grid]] while preserving topological relationships. It is used for [[clustering|clustering]], visualizing data [[patterns|patterns]], and detecting noisy or mislabeled [[training-samples|training samples]].
- **ES.** Mapa autoorganizado: una red neuronal no supervisada que reduce [[data|datos]] de alta dimensión (por [[example|ejemplo]], [[satellite-image-time-series|series temporales de imágenes satelitales]]) a una [[grid|cuadrícula]] bidimensional preservando relaciones topológicas. Se utiliza para [[clustering|agrupamiento]], visualización de [[patterns|patrones]] de datos y [[detection|detección]] de [[training-samples|muestras de entrenamiento]] ruidosas o mal etiquetadas.

## Understand these first / Entiende esto primero
- [[time-series|Time series]] — *data*
    - **EN.** SOM organizes input time series into clusters, preserving topological relationships for pattern discovery.
    - **ES.** SOM organiza series temporales de entrada en clústeres, preservando relaciones topológicas para el descubrimiento de patrones.
- [[clustering-algorithm|Clustering Algorithm]] — *method*
    - **EN.** SOM is an unsupervised clustering algorithm that maps high-dimensional time series to a 2D grid.
    - **ES.** SOM es un algoritmo de agrupamiento no supervisado que mapea series temporales de alta dimensión a una cuadrícula 2D.
