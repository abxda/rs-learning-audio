# WorldCereal Reference Data Module / Módulo de Datos de Referencia de WorldCereal

> **level 4** · *software* · sources: ad_world_cereal · ad_world_cereal

**What it is / Qué es.**

- **EN.** A community-driven [[framework|framework]] for collecting, harmonizing, documenting, and sharing [[reference-datasets|reference data]] for [[crop-mapping|crop mapping]], integrating [[field-surveys|field surveys]], [[parcels|parcel]] systems, and citizen science. It reduces the cost of [[training-dataset|training data]] for [[earth-observation|Earth observation]] [[applications|applications]].
- **ES.** Un [[framework|marco]] impulsado por la comunidad para recopilar, armonizar, documentar y compartir [[ground-truth-data|datos de referencia]] para el [[crop-mapping|mapeo de cultivos]], integrando [[ground-surveys|encuestas de campo]], sistemas de [[parcels|parcelas]] y ciencia ciudadana. Reduce el costo de los [[training-dataset|datos de entrenamiento]] para [[applications|aplicaciones]] de [[earth-observation|observación de la Tierra]].

## Understand these first / Entiende esto primero
- [[crop-type-mapping|Crop type mapping]] — *method*
    - **EN.** It supports crop type mapping by standardizing reference data collection and documentation across projects.
    - **ES.** Apoya el mapeo de tipos de cultivo al estandarizar la recopilación y documentación de datos de referencia entre proyectos.
- [[reference-datasets|Reference datasets]] — *data*
    - **EN.** The WorldCereal Reference Data Module provides a framework for harmonizing and sharing reference datasets for crop mapping.
    - **ES.** El Módulo de Datos de Referencia de WorldCereal proporciona un marco para armonizar y compartir conjuntos de datos de referencia para mapeo de cultivos.
