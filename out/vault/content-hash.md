# Content hash / Hash de contenido

> **level 0** · *primitive*

**What it is / Qué es.**

- **EN.** A fixed-size string derived from [[data|data]] content, used to verify integrity or uniqueness.
- **ES.** Cadena de tamaño fijo derivada del contenido de los [[data|datos]], utilizada para verificar la integridad o unicidad.

## This unlocks / Esto habilita
- [[oci-artifact|OCI artifact]] — OCI artifacts are identified by a content hash, enabling versioned, data integrity verification.
