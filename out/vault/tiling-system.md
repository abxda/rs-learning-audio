# Tiling system / Sistema de teselas

> **level 4** · *data* · sources: th_data_sources

**What it is / Qué es.**

- **EN.** A spatial partition that divides the [[earth|Earth]]'s surface into regular, uniquely identified [[grid|grid]] cells ([[tiles|tiles]]) used to organize [[satellite-image-collections|satellite image collections]], such as the [[military-grid-reference-system|MGRS]] grid for Sentinel-2 or the WRS-2 for [[landsat|Landsat]], enabling consistent [[data|data]] management and [[processing|processing]].
- **ES.** Una partición espacial que divide la superficie de la [[land|Tierra]] en celdas de [[grid|cuadrícula]] regulares e identificadas de manera única ([[tiles|teselas]]) utilizadas para organizar [[satellite-image-collections|colecciones de imágenes satelitales]], como la cuadrícula [[military-grid-reference-system|MGRS]] para Sentinel-2 o la WRS-2 para [[landsat|Landsat]], permitiendo una gestión y [[data-processing|procesamiento de datos]] consistentes.

## Understand these first / Entiende esto primero
- [[grid|Grid]] — *data*
    - **EN.** A tiling system divides the Earth into regular grid cells for managing large satellite image collections.
    - **ES.** Un sistema de teselas divide la Tierra en celdas de cuadrícula regulares para gestionar grandes colecciones de imágenes satelitales.
- [[military-grid-reference-system|Military Grid Reference System]] — *data*
    - **EN.** The Military Grid Reference System is a standard tiling system used for global satellite data organization.
    - **ES.** El Sistema de Referencia de Cuadrícula Militar es un sistema de teselas estándar utilizado para la organización global de datos satelitales.
