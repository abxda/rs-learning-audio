# Map approach / Enfoque cartográfico

> **level 4** · *method* · sources: crop_statistics_ppi

**What it is / Qué es.**

- **EN.** An [[estimation|estimation]] [[methods|method]] that uses only [[satellite|satellite]]-derived map [[predictions|predictions]] for a quantity of interest, without incorporating [[ground-truth|ground-truth]] [[survey|survey]] [[data|data]], and may therefore be biased.
- **ES.** Método de [[estimation|estimación]] que utiliza únicamente [[predictions|predicciones]] de [[map|mapa]] derivadas de [[satellite|satélite]] para una cantidad de interés, sin incorporar [[data|datos]] de [[survey|encuesta]] de [[ground-truth|verdad terreno]], y que por lo tanto puede estar sesgado.

## Understand these first / Entiende esto primero
- [[map|Map]] — *product*
    - **EN.** The Map approach uses only satellite-derived map predictions, without survey data, to estimate quantities.
    - **ES.** El enfoque cartográfico utiliza solo predicciones derivadas de satélite, sin datos de encuesta, para estimar cantidades.
- [[approach|Approach]] — *primitive*
    - **EN.** An approach is a methodology; the Map approach selects the classification map as the sole estimator.
    - **ES.** Un enfoque es una metodología; el enfoque cartográfico selecciona el mapa de clasificación como el único estimador.
