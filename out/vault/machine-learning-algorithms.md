# Machine learning algorithms / Algoritmos de aprendizaje automático

> **level 5** · *method* · sources: th_machine_learning

**What it is / Qué es.**

- **EN.** Computational [[methods|methods]] that learn [[patterns|patterns]] from [[data|data]] to make [[predictions|predictions]] or decisions without explicit programming for each task.
- **ES.** [[methods|Métodos]] computacionales que aprenden [[patterns|patrones]] a partir de [[data|datos]] para hacer [[predictions|predicciones]] o tomar decisiones sin programación explícita para cada tarea.
