# Map SF / Mapa SF

> **level 3** · *product* · sources: th_validation

**What it is / Qué es.**

- **EN.** A [[satellite-data-cube|classified data cube]] that has been labeled by assigning each [[pixels|pixel]] the [[land-cover-classes|land cover class]] with the highest [[probability|probability]]. It represents the final [[thematic-map|thematic map]] output.
- **ES.** Un [[cube|cubo de datos]] clasificado que ha sido etiquetado asignando a cada píxel la clase de [[land-cover|cobertura del suelo]] con mayor [[probability|probabilidad]]. Representa el [[map-product|producto cartográfico]] temático final.

## Understand these first / Entiende esto primero
- [[classification|Classification]] — *method*
    - **EN.** After classification, each pixel of the cube receives a class label.
    - **ES.** Tras la clasificación, cada píxel del cubo recibe una etiqueta de clase.
- [[cube|Cube]] — *data*
    - **EN.** A classified data cube is labeled by selecting for each pixel the class with highest probability.
    - **ES.** Un cubo de datos clasificado se etiqueta seleccionando para cada píxel la clase con mayor probabilidad.
