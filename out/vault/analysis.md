# Analysis / Análisis

> **level 2** · *method* · sources: th_machine_learning

**What it is / Qué es.**

- **EN.** The systematic examination of [[data|data]] to extract [[information|information]], such as classifying [[images|images]] or estimating [[crop-area|crop area]] and [[yield|yield]] using [[statistical|statistical]] or [[machine|machine]]-[[machine-learning|learning]] [[methods|methods]].
- **ES.** Examen sistemático de [[data|datos]] para extraer [[information|información]], como clasificar [[images|imágenes]] o estimar [[crop-area|área de cultivo]] y [[yield|rendimiento]] usando [[methods|métodos]] de [[machine-learning|aprendizaje automático]].

## Understand these first / Entiende esto primero
- [[data|Data]] — *primitive*
    - **EN.** Analysis involves systematic examination of data to extract meaningful information.
    - **ES.** El análisis implica el examen sistemático de datos para extraer información significativa.
- [[results|Results]] — *primitive*
    - **EN.** Results are the outputs of analysis, such as classified maps or accuracy metrics.
    - **ES.** Los resultados son las salidas del análisis, como mapas clasificados o métricas de exactitud.
- [[methods|Methods]] — *method*
    - **EN.** Analysis applies defined methods to transform data into results.
    - **ES.** El análisis aplica métodos definidos para producir resultados.
- [[statistics|Statistics]] — *statistic*
    - **EN.** Statistics are quantitative measures derived from data analysis, such as crop area and yield.
    - **ES.** Las estadísticas son medidas cuantitativas derivadas del análisis de datos, como el área de cultivo y el rendimiento.
