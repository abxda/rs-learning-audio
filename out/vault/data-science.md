# Data Science / Ciencia de Datos

> **level 4** · *primitive* · sources: th_machine_learning

**What it is / Qué es.**

- **EN.** Data science is an interdisciplinary [[field|field]] that uses [[statistical|statistical]] and computational [[methods|methods]] to extract insights from [[data|data]]. In this [[handbook|handbook]], it underpins [[machine-learning|machine learning]] and [[deep-learning|deep learning]] approaches for analyzing [[satellite-image-time-series|satellite image time series]].
- **ES.** La Ciencia de Datos es un [[field|campo]] interdisciplinario que utiliza [[statistical-methods|métodos estadísticos]] y computacionales para extraer [[information|información]] de los [[data|datos]]. En este [[handbook|manual]], sustenta los enfoques de [[machine-learning|aprendizaje automático]] y [[deep-learning|aprendizaje profundo]] para analizar [[satellite-image-time-series|series temporales de imágenes satelitales]].

## Understand these first / Entiende esto primero
- [[data|Data]] — *primitive*
    - **EN.** Data science applies statistical and computational methods to extract insights from data.
    - **ES.** La ciencia de datos aplica métodos estadísticos y computacionales para extraer información de los datos.
- [[statistics|Statistics]] — *statistic*
    - **EN.** Data science relies on statistical principles to analyze and interpret data.
    - **ES.** La ciencia de datos se basa en principios estadísticos para analizar e interpretar datos.
- [[machine-learning|Machine learning]] — *method*
    - **EN.** Using machine learning algorithms, data science builds models from data for predictions.
    - **ES.** Usando algoritmos de aprendizaje automático, la Ciencia de Datos construye modelos a partir de datos para predicciones.
