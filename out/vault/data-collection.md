# Data collection / Recopilación de datos

> **level 1** · *method* · sources: th_alignment

**What it is / Qué es.**

- **EN.** The [[process|process]] of gathering [[ground-truth-2|ground truth]] observations, such as [[crop-types|crop type]] or [[yield|yield]] measurements, used to train and validate [[remote-sensing|remote sensing]] models.
- **ES.** [[process|Proceso]] de recopilación de observaciones de [[ground-truth|verdad terreno]], como el [[type|tipo]] de [[crop|cultivo]] o mediciones de [[yield|rendimiento]], utilizadas para entrenar y validar modelos de [[remote-sensing|teledetección]].

## Understand these first / Entiende esto primero
- [[data|Data]] — *primitive*
    - **EN.** Data collection gathers observations such as crop type and yield from the field.
    - **ES.** La recopilación de datos reúne observaciones como el tipo de cultivo y el rendimiento del campo.
