# Methodology / Metodología

> **level 2** · *method* · sources: introduction · th_validation

**What it is / Qué es.**

- **EN.** A systematic set of [[principles|principles]], procedures, and techniques used to conduct a [[study|study]] or produce [[statistics|statistics]], ensuring [[reproducibility|reproducibility]] and rigor, such as the [[workflow|workflow]] for [[crop-classification|crop classification]] from [[satellite-imagery|satellite imagery]].
- **ES.** [[set|Conjunto]] sistemático de [[principles|principios]], procedimientos y técnicas utilizados para realizar un [[study|estudio]] o producir [[statistics|estadísticas]], garantizando [[reproducibility|reproducibilidad]] y rigor, como el [[workflow|flujo de trabajo]] para la [[crop-classification|clasificación de cultivos]] a partir de [[satellite-imagery|imágenes satelitales]].

## Understand these first / Entiende esto primero
- [[methods|Methods]] — *method*
    - **EN.** Methodology is a systematic set of procedures, extending individual methods to ensure consistent and reproducible analysis.
    - **ES.** La metodología es un conjunto sistemático de procedimientos, que extiende métodos individuales para garantizar un análisis consistente y reproducible.
