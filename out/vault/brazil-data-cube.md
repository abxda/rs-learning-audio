# Brazil Data Cube / Brazil Data Cube

> **level 6** · *data* · sources: th_data_sources · th_data_cubes

**What it is / Qué es.**

- **EN.** A [[cloud|cloud]] service providing open, regularized [[earth-observation|Earth observation data]] cubes from Sentinel-2, [[landsat|Landsat]]-8, CBERS-4/4A, and MODIS over Brazil. It provides ready-to-use [[data|data]] for [[machine-learning|machine learning]] [[classification|classification]].
- **ES.** Un servicio en la [[cloud|nube]] que proporciona cubos de [[data|datos]] de [[earth-observation|observación de la Tierra]] abiertos y regularizados a partir de Sentinel-2, [[landsat|Landsat]]-8, CBERS-4/4A y MODIS sobre Brasil. Proporciona datos listos para usar en [[classification|clasificación]] por [[machine-learning|aprendizaje automático]].

## Understand these first / Entiende esto primero
- [[data-cube|Data cube]] — *data*
    - **EN.** The Brazil Data Cube is a cloud service providing open Earth observation data cubes from multiple satellite missions.
    - **ES.** El Brazil Data Cube es un servicio en la nube que proporciona cubos de datos abiertos de observación de la Tierra de múltiples misiones satelitales.
