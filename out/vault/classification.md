# Classification / Clasificación

> **level 2** · *method* · sources: th_machine_learning

**What it is / Qué es.**

- **EN.** The [[process|process]] of assigning a [[classes|class]] [[label|label]] to each [[pixels|pixel]] or object in [[satellite-imagery|satellite imagery]] based on spectral, temporal, or spatial characteristics using [[algorithm|algorithms]].
- **ES.** [[process|Proceso]] de asignar una [[label|etiqueta]] de clase a cada píxel u objeto en [[satellite-imagery|imágenes satelitales]] basado en características espectrales, temporales o espaciales mediante algoritmos.

## Understand these first / Entiende esto primero
- [[model|Model]] — *method*
    - **EN.** A trained classification model maps input data to class labels.
    - **ES.** Un modelo de clasificación entrenado asigna datos de entrada a etiquetas de clase.
- [[classes|Classes]] — *data*
    - **EN.** Classification assigns one of the predefined classes to each pixel or object in imagery.
    - **ES.** La clasificación asigna una de las clases predefinidas a cada píxel u objeto en las imágenes.
- [[samples|Samples]] — *data*
    - **EN.** Labeled samples train the classification model to recognize class patterns.
    - **ES.** Las muestras etiquetadas entrenan el modelo de clasificación para reconocer patrones de clase.

- [[map|Map]] — A map is the thematic output of classification, showing spatial distribution of classes.
- [[mapping|Mapping]] — Mapping produces thematic maps by applying classification to satellite imagery.
- [[large-scale-agricultural-mapping|Large scale agricultural mapping]] — Large-scale agricultural mapping applies classification algorithms to satellite data for regional crop maps.
- [[crop-type-mapping|Crop type mapping]] — Crop type mapping is a specific classification task that identifies crop types from satellite imagery.
- [[crop-classification|Crop classification]] — Crop classification is a type of classification that specifically assigns crop type labels using satellite imagery.
- [[classification-map|Classification map]] — A classification map is a thematic map produced by applying a classification algorithm to satellite imagery.
- [[classified-map|Classified map]] — A classified map is produced by assigning each pixel or object to a land cover class via classification.
- [[land-cover-mapping|Land cover mapping]] — It applies classification algorithms to satellite imagery to label land cover classes.
- [[classification-task|Classification task]] — A classification task is a specific problem where a classifier assigns labels to data.
- [[classification-process|Classification process]] — The classification process is the entire workflow from data preparation to label assignment.
- [[crop-identification|Crop identification]] — It uses classification methods to label pixels or fields with crop types.
- [[higher-classification-accuracy|Higher classification accuracy]] — Improving classification methods can yield higher classification accuracy compared to simpler models.
- [[remote-image-classification|Remote image classification]] — Remote image classification applies the general classification process to satellite imagery instead of other data.
- [[errors|Errors]] — Errors in classification are the mismatches between assigned labels and true ground labels.
- [[map-sf|Map SF]] — After classification, each pixel of the cube receives a class label.
- [[earth-observation-classifications|EO classifications]] — EO classifications are the labels produced by classifying satellite imagery into land cover or use categories.
- [[identification|Identification]] — Identification is a classification task that recognizes specific entities like crop types from satellite data.
- [[land-use|Land use]] — Land use mapping classifies land according to its human activities, requiring classification of land cover as a basis.
- [[accuracy-assessment|Accuracy assessment]] — Accuracy assessment evaluates how correctly a classification assigns labels to pixels or objects.
