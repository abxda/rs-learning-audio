# Cropland / Tierras de cultivo

> **level 2** · *phenomenon* · sources: th_lucc

**What it is / Qué es.**

- **EN.** Cropland is [[land|land]] used for cultivating [[crop|crops]], including both actively farmed fields and temporarily [[fallow|fallow]] areas. It is a fundamental [[land-cover-classes|land cover class]] in [[agricultural-monitoring|agricultural monitoring]] and [[crop-type-mapping|crop type mapping]].
- **ES.** Las tierras de cultivo son tierras utilizadas para el [[crop|cultivo]] de cultivos, incluyendo tanto campos cultivados activamente como áreas temporalmente en [[fallow|barbecho]]. Es una clase fundamental de [[land-cover|cobertura del suelo]] en el [[agricultural-monitoring|monitoreo agrícola]] y el [[crop-type-mapping|mapeo de tipos de cultivo]].

## Understand these first / Entiende esto primero
- [[land|Land]] — *phenomenon*
    - **EN.** Cropland is land that is used for growing crops, distinguishing it from other land cover types.
    - **ES.** Las tierras de cultivo son tierras utilizadas para cultivar, distinguiéndose de otros tipos de cobertura del suelo.

## This unlocks / Esto habilita
- [[fallow|Fallow]] — Fallow is a state of cropland where no crop is grown, usually to allow soil recovery before the next planting season.
