# Set / Conjunto

> **level 1** · *primitive* · sources: th_quality_control

**What it is / Qué es.**

- **EN.** A [[collection|collection]] of distinct elements, such as [[pixels|pixels]], [[polygons|polygons]], or [[label|labels]], grouped together for [[analysis|analysis]] or [[processing|processing]].
- **ES.** Una [[collection|colección]] de elementos distintos, como [[pixels|píxeles]], [[polygons|polígonos]] o etiquetas, agrupados para su [[analysis|análisis]] o [[processing|procesamiento]].
