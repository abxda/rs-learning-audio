# Large scale agricultural mapping / Cartografía agrícola a gran escala

> **level 4** · *method* · sources: ct_zimbabwe

**What it is / Qué es.**

- **EN.** The [[process|process]] of creating [[land-cover|land cover]] or [[crop-type-map|crop type maps]] over extensive areas, often using [[satellite-imagery|satellite imagery]] and automated [[classification|classification]].
- **ES.** El [[process|proceso]] de crear mapas de [[land-cover|cobertura del suelo]] o tipos de [[crop|cultivo]] en áreas extensas, a menudo utilizando [[satellite-imagery|imágenes satelitales]] y [[classification|clasificación]] automatizada.

## Understand these first / Entiende esto primero
- [[classification|Classification]] — *method*
    - **EN.** Large-scale agricultural mapping applies classification algorithms to satellite data for regional crop maps.
    - **ES.** La cartografía agrícola a gran escala aplica algoritmos de clasificación a datos satelitales para mapas regionales de cultivos.
- [[satellite|Satellite]] — *platform*
    - **EN.** Large-scale agricultural mapping uses satellite imagery to cover wide regions efficiently.
    - **ES.** La cartografía agrícola a gran escala utiliza imágenes satelitales para cubrir regiones extensas de manera eficiente.
- [[crop-type-mapping|Crop type mapping]] — *method*
    - **EN.** Large-scale agricultural mapping extends crop type mapping to cover extensive areas using automated methods.
    - **ES.** La cartografía agrícola a gran escala extiende el mapeo de tipos de cultivo para cubrir áreas extensas mediante métodos automatizados.
