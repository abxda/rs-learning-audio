# Model training / Entrenamiento del modelo

> **level 3** · *method*

**What it is / Qué es.**

- **EN.** The [[process|process]] of fitting a [[machine-learning|machine learning]] [[algorithm|algorithm]] to labeled [[data|data]] by adjusting its [[parameters|parameters]] to minimize [[predictions|prediction]] error. It requires a representative [[dataset|dataset]] and produces a [[model|model]] for [[classification|classification]] or [[regression|regression]].
- **ES.** El [[process|proceso]] de ajustar un [[algorithm|algoritmo]] de [[machine-learning|aprendizaje automático]] a [[data|datos]] etiquetados modificando sus [[parameters|parámetros]] para minimizar el error de predicción. Requiere un [[dataset|conjunto de datos]] representativo y produce un [[model|modelo]] para [[classification|clasificación]] o [[regression|regresión]].

## Understand these first / Entiende esto primero
- [[samples|Samples]] — *data*
    - **EN.** During model training, the algorithm learns to map input features to output classes using labeled samples.
    - **ES.** Durante el entrenamiento del modelo, el algoritmo aprende a mapear atributos de entrada a clases de salida usando muestras etiquetadas.
- [[training|Training]] — *method*
    - **EN.** Model training is the process of fitting a machine learning algorithm to labeled data by adjusting its parameters.
    - **ES.** El entrenamiento del modelo es el proceso de ajustar un algoritmo de aprendizaje automático a datos etiquetados mediante el ajuste de sus parámetros.
