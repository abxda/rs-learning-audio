# Time series analysis / Análisis de series temporales

> **level 3** · *method* · sources: th_data_cubes#working-with-satellite-image-time-series

**What it is / Qué es.**

- **EN.** A [[methods|method]] that examines sequential observations of the same location over [[time|time]] to detect temporal [[patterns|patterns]], trends, and changes in [[land|land]] surface properties from [[satellite-data|satellite data]].
- **ES.** Un método que examina observaciones secuenciales de la misma ubicación a lo largo del [[time|tiempo]] para detectar [[patterns|patrones]] temporales, tendencias y cambios en las propiedades de la [[earth|superficie terrestre]] a partir de [[satellite-data|datos satelitales]].

## Understand these first / Entiende esto primero
- [[time|Time]] — *primitive*
    - **EN.** Time series analysis examines sequential data over time to detect patterns and trends.
    - **ES.** El análisis de series temporales examina datos secuenciales a lo largo del tiempo para detectar patrones y tendencias.
- [[analysis|Analysis]] — *method*
    - **EN.** Time series analysis is a systematic examination of temporal data to extract information.
    - **ES.** El análisis de series temporales es un examen sistemático de datos temporales para extraer información.
- [[time-series|Time series]] — *data*
    - **EN.** Time series analysis focuses on sequences of data points collected at successive times.
    - **ES.** El análisis de series temporales se centra en secuencias de puntos de datos recolectados en tiempos sucesivos.
