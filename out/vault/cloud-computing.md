# Cloud computing / Computación en la nube

> **level 3** · *method* · sources: th_data_sources#cloud-platforms-providing-ard-data

**What it is / Qué es.**

- **EN.** A paradigm for delivering on-demand computing [[resources|resources]]—[[processing|processing]], storage, and [[software|software]]—over the internet, enabling scalable [[analysis|analysis]] of large [[earth-observation|Earth observation]] datasets without local [[infrastructure|infrastructure]].
- **ES.** Un paradigma para proporcionar [[resources|recursos]] informáticos bajo demanda—[[processing|procesamiento]], almacenamiento y [[package|software]]—a través de internet, permitiendo el [[analysis|análisis]] escalable de [[large-datasets|grandes conjuntos de datos]] de [[earth-observation|observación de la Tierra]] sin [[infrastructure|infraestructura]] local.

## Understand these first / Entiende esto primero
- [[infrastructure|Infrastructure]] — *primitive*
    - **EN.** Cloud computing is a type of infrastructure that delivers on-demand computing resources over the internet.
    - **ES.** La computación en la nube es un tipo de infraestructura que entrega recursos informáticos bajo demanda a través de internet.
- [[big-data|Big Data]] — *data*
    - **EN.** Cloud computing provides the scalable infrastructure needed to process and store big Earth observation data.
    - **ES.** La computación en la nube proporciona la infraestructura escalable necesaria para procesar y almacenar grandes datos de observación de la Tierra.
