# Input / Entrada

> **level 1** · *primitive* · sources: th_data_cubes

**What it is / Qué es.**

- **EN.** [[data|Data]] fed into a [[system|system]] or [[model|model]], such as [[satellite-imagery|satellite imagery]] or [[field-samples|field samples]], which is processed to produce an output like a [[classification-map|classification map]].
- **ES.** [[data|Datos]] introducidos en un [[system|sistema]] o [[model|modelo]], como [[satellite-imagery|imágenes satelitales]] o [[field-samples|muestras de campo]], que se procesan para producir un resultado como un [[classification-map|mapa de clasificación]].

## Understand these first / Entiende esto primero
- [[data|Data]] — *primitive*
    - **EN.** Input is the data fed into a system or model for processing into output.
    - **ES.** La entrada son los datos alimentados a un sistema o modelo para su procesamiento en salida.
