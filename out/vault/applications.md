# Applications / Aplicaciones

> **level 2** · *primitive* · sources: th_remote_sensing

**What it is / Qué es.**

- **EN.** Practical uses of [[remote-sensing|remote sensing]] [[data|data]], such as [[operational-crop-monitoring|crop monitoring]], [[yield-forecasting|yield forecasting]], and disaster [[assessment|assessment]].
- **ES.** Usos prácticos de los [[data|datos]] de [[remote-sensing|teledetección]], como el [[monitoring|monitoreo]] de cultivos, el [[yield-forecasting|pronóstico de rendimiento]] y la [[assessment|evaluación]] de desastres.

## Understand these first / Entiende esto primero
- [[methods|Methods]] — *method*
    - **EN.** Applications implement methods to solve real-world problems in agriculture and land management.
    - **ES.** Las aplicaciones implementan métodos para resolver problemas reales en agricultura y gestión de tierras.
- [[use|Use]] — *primitive*
    - **EN.** Applications are the practical uses of remote sensing data for specific purposes like crop monitoring.
    - **ES.** Las aplicaciones son los usos prácticos de los datos de teledetección para fines específicos como el monitoreo de cultivos.
