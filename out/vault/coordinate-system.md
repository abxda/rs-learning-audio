# Coordinate system / Sistema de coordenadas

> **level 5** · *other* · sources: th_data_sources

**What it is / Qué es.**

- **EN.** A [[reference|reference]] [[system|system]] used to uniquely identify the location of [[points|points]] on the [[earth|Earth]]'s surface. It includes geographic [[coordinates|coordinates]] (latitude/longitude) as well as projected coordinate systems (e.g., UTM) that convert the Earth's curved surface to a flat map, enabling [[area|area]] and distance calculations.
- **ES.** Un [[system|sistema]] de [[reference|referencia]] utilizado para identificar de manera única la ubicación de [[points|puntos]] en la superficie de la [[land|Tierra]]. Incluye [[coordinates|coordenadas]] geográficas (latitud/longitud), así como sistemas de coordenadas proyectadas (por [[example|ejemplo]], UTM) que convierten la superficie curva de la Tierra a un [[map|mapa]] plano, permitiendo cálculos de [[area|área]] y distancia.

## Understand these first / Entiende esto primero
- [[georeferenced-information|Georeferenced information]] — *data*
    - **EN.** A coordinate system provides the reference for georeferenced information, linking data to locations.
    - **ES.** Un sistema de coordenadas proporciona la referencia para la información georreferenciada, vinculando los datos con ubicaciones.
- [[basemap|Basemap]] — *product*
    - **EN.** A basemap is projected in a coordinate system to align with other geospatial layers.
    - **ES.** Un mapa base se proyecta en un sistema de coordenadas para alinearse con otras capas geoespaciales.
- [[coordinates|Coordinates]] — *primitive*
    - **EN.** A coordinate system defines how coordinates are assigned to points on Earth, enabling georeferencing.
    - **ES.** Un sistema de coordenadas define cómo se asignan coordenadas a puntos en la Tierra, permitiendo la georreferenciación.
