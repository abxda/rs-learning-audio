# Satellite Image Analysis / Análisis de imágenes satelitales

> **level 5** · *method* · sources: th_remote_sensing

**What it is / Qué es.**

- **EN.** The use of computer [[algorithm|algorithms]] to interpret [[data|data]] collected by [[satellite|satellite]]-mounted [[sensors|sensors]], extracting meaningful [[information|information]] about [[earth|Earth]]'s surface for [[applications|applications]] like [[agriculture|agriculture]].
- **ES.** Uso de algoritmos computacionales para interpretar [[data|datos]] recolectados por [[sensors|sensores]] montados en satélites, extrayendo [[information|información]] significativa sobre la superficie de la [[land|Tierra]] para [[applications|aplicaciones]] como la [[agriculture|agricultura]].

## Understand these first / Entiende esto primero
- [[algorithm|Algorithm]] — *method*
    - **EN.** Algorithms interpret satellite imagery to extract meaningful patterns and information.
    - **ES.** Los algoritmos interpretan imágenes satelitales para extraer patrones e información significativos.
- [[satellite-imagery|Satellite imagery]] — *data*
    - **EN.** Satellite imagery is the primary input for analysis using computer algorithms.
    - **ES.** Las imágenes satelitales son la entrada principal para el análisis mediante algoritmos computacionales.
