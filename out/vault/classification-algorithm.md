# Classification algorithm / Algoritmo de clasificación

> **level 5** · *method* · sources: th_machine_learning · th_machine_learning

**What it is / Qué es.**

- **EN.** A computational [[methods|method]] that uses [[labeled-samples|labeled samples]] to learn [[patterns|patterns]] and then assigns [[classes|class]] [[label|labels]] to new observations, such as [[random-forest|Random Forest]] or [[neural-networks|neural networks]].
- **ES.** Un método computacional que utiliza [[labeled-samples|muestras etiquetadas]] para aprender [[patterns|patrones]] y luego asigna etiquetas de clase a nuevas observaciones, como [[random-forest|Random Forest]] o [[neural-networks|redes neuronales]].

## Understand these first / Entiende esto primero
- [[algorithm|Algorithm]] — *method*
    - **EN.** A classification algorithm is an algorithm specifically designed to assign class labels to data.
    - **ES.** Un algoritmo de clasificación es un algoritmo diseñado específicamente para asignar etiquetas de clase a los datos.
