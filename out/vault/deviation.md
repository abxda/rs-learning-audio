# Deviation / Desviación

> **level 0** · *primitive*

**What it is / Qué es.**

- **EN.** Difference between an observed value and a [[reference|reference]] or expected value.
- **ES.** Diferencia entre un valor observado y un valor de [[reference|referencia]] o esperado.
