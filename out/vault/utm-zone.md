# UTM zone / zona UTM

> **level 6** · *data* · sources: th_data_sources · uav_agriculture_cook_islands

**What it is / Qué es.**

- **EN.** A 6-degree-wide longitudinal division of the Universal Transverse Mercator [[coordinate-system|coordinate system]] used for accurate spatial measurements.
- **ES.** Una división longitudinal de 6 grados de ancho del [[coordinate-system|sistema de coordenadas]] Universal Transversal de Mercator, utilizada para mediciones espaciales precisas.

## Understand these first / Entiende esto primero
- [[military-grid-reference-system|Military Grid Reference System]] — *data*
    - **EN.** The Military Grid Reference System extends UTM zones by adding a grid of 100 km squares for precise location.
    - **ES.** El Sistema de Referencia de Cuadrícula Militar extiende las zonas UTM añadiendo una cuadrícula de cuadrados de 100 km para una ubicación precisa.
- [[tiling-system|Tiling system]] — *data*
    - **EN.** UTM zones partition the Earth into zones for consistent mapping and tile organization.
    - **ES.** Las zonas UTM dividen la Tierra en zonas para un mapeo consistente y organización de teselas.
- [[coordinate-system|Coordinate system]] — *other*
    - **EN.** A UTM zone is a 6-degree longitudinal division of the Universal Transverse Mercator coordinate system.
    - **ES.** Una zona UTM es una división longitudinal de 6 grados del sistema de coordenadas Universal Transversal de Mercator.
