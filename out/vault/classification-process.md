# Classification process / Proceso de clasificación

> **level 5** · *method* · sources: th_lucc

**What it is / Qué es.**

- **EN.** The set of operations, from [[training-dataset|training data]] preparation to [[model|model]] [[applications|application]], that assigns semantic [[label|labels]] to [[pixels|pixels]] or objects in [[satellite-imagery|satellite imagery]].
- **ES.** El [[set|conjunto]] de operaciones, desde la [[data-preparation|preparación de datos]] de [[training|entrenamiento]] hasta la [[software|aplicación]] del [[model|modelo]], que asigna etiquetas semánticas a [[pixels|píxeles]] u objetos en [[satellite-imagery|imágenes satelitales]].

## Understand these first / Entiende esto primero
- [[classification|Classification]] — *method*
    - **EN.** The classification process is the entire workflow from data preparation to label assignment.
    - **ES.** El proceso de clasificación es el flujo de trabajo completo desde la preparación de datos hasta la asignación de etiquetas.
- [[stages|Stages]] — *primitive*
    - **EN.** It includes stages like training, classification, and validation in a sequential order.
    - **ES.** Incluye etapas como entrenamiento, clasificación y validación en un orden secuencial.
