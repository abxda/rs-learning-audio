# Crop type map / Mapa de tipos de cultivo

> **level 0** · *product* · sources: ct_poland

**What it is / Qué es.**

- **EN.** A [[crop|crop]]-[[type|type]] map is a [[thematic-map|thematic map]] assigning each [[pixels|pixel]] or [[field|field]] [[parcels|parcel]] to a specific crop category, produced by classifying [[satellite-image-time-series|satellite image time series]] with [[machine-learning-algorithms|machine learning algorithms]] trained on [[reference-datasets|reference data]].
- **ES.** Un mapa de tipos de cultivo es un [[thematic-map|mapa temático]] que asigna cada píxel o [[plot|parcela]] de [[field|campo]] a una categoría de [[crop|cultivo]] específica, producido mediante la [[satellite-image-time-series-classification|clasificación de series temporales de imágenes satelitales]] con [[machine-learning-algorithms|algoritmos de aprendizaje automático]] entrenados en [[ground-truth-data|datos de referencia]].

## This unlocks / Esto habilita
- [[classification-map|Classification map]] — A crop-type map is a classification map that focuses on differentiating crop types using spectral-temporal signatures.
