# Predictors / Predictores

> **level 2** · *data* · sources: cy_poland

**What it is / Qué es.**

- **EN.** Variables used in a [[model|model]] to predict an outcome. In [[yield-forecasting|yield forecasting]], predictors include [[satellite|satellite]]-derived [[vegetation-indices|vegetation indices]], temperature, precipitation, and [[soil-moisture|soil moisture]] [[data|data]].
- **ES.** Variables utilizadas en un [[model|modelo]] para predecir un resultado. En la predicción del [[yield|rendimiento]], los predictores incluyen [[vegetation-indices|índices de vegetación]] derivados de satélites, [[data|datos]] de temperatura, precipitación y [[soil-moisture|humedad del suelo]].

## Understand these first / Entiende esto primero
- [[model|Model]] — *method*
    - **EN.** In a model, predictors are the input variables that help forecast the outcome.
    - **ES.** En un modelo, los predictores son las variables de entrada que ayudan a pronosticar el resultado.
- [[feature|Feature]] — *data*
    - **EN.** Predictors are features such as spectral bands, vegetation indices, or time series used in a model.
    - **ES.** Los predictores son atributos como bandas espectrales, índices de vegetación o series temporales utilizados en un modelo.
- [[input|Input]] — *primitive*
    - **EN.** Input variables serve as predictors for a model to estimate an outcome.
    - **ES.** Las variables de entrada sirven como predictores para que un modelo estime un resultado.
