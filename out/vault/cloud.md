# Cloud / Nube

> **level 1** · *phenomenon* · sources: th_data_sources

**What it is / Qué es.**

- **EN.** Atmospheric water vapor or ice that obscures [[satellite-imagery|satellite imagery]], requiring [[masking|masking]] or compositing to obtain clear observations.
- **ES.** Vapor de agua atmosférico o hielo que oscurece las [[satellite-imagery|imágenes satelitales]], requiriendo [[masking|enmascaramiento]] o composición para obtener observaciones despejadas.

## Understand these first / Entiende esto primero
- [[images|Images]] — *data*
    - **EN.** Cloud appears in satellite images as opaque areas that need to be identified and removed.
    - **ES.** La nube aparece en imágenes satelitales como áreas opacas que deben identificarse y eliminarse.

## This unlocks / Esto habilita
- [[atmospheric-correction|Atmospheric correction]] — Cloud is atmospheric water vapor that obscures satellite imagery and must be masked or composited.
- [[infrastructure|Infrastructure]] — Infrastructure comprises cloud computing and storage systems needed to process and analyze large Earth observation data.
- [[copernicus-data-space-ecosystem|Copernicus Data Space Ecosystem]] — It is a cloud-based platform for storing and analyzing Earth observation data.
- [[cloud-collections|Cloud collections]] — Cloud computing enables storage and retrieval of large satellite image collections as cloud collections.
- [[platforms|Platforms]] — Platforms like cloud services provide the infrastructure to access and process satellite imagery.
- [[masking|Masking]] — Cloud masking identifies and removes cloud-covered pixels to obtain clear observations.
- [[cloud-based-services|Cloud-based services]] — Cloud‑based services deliver on‑demand access to satellite data archives and computational resources via the internet.
- [[cloud-masking|Cloud Masking]] — Cloud masking uses cloud detection algorithms to remove cloud-affected pixels from satellite imagery.
- [[cloud-shadow|Cloud shadow]] — Cloud shadow is the dark area on the ground caused by a cloud blocking sunlight, requiring masking.
