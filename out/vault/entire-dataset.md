# Entire dataset / Conjunto de datos completo

> **level 2** · *data* · sources: th_machine_learning

**What it is / Qué es.**

- **EN.** The complete [[collection|collection]] of available [[data|data]] [[points|points]] used for [[analysis|analysis]], as opposed to a subset or [[samples|sample]], often used for [[training|training]] [[machine-learning-models|machine learning models]].
- **ES.** La [[collection|colección]] completa de [[points|puntos]] de [[data|datos]] disponibles para el [[analysis|análisis]], en lugar de un subconjunto o muestra, utilizada a menudo para entrenar [[machine-learning-models|modelos de aprendizaje automático]].

## Understand these first / Entiende esto primero
- [[dataset|Dataset]] — *data*
    - **EN.** An entire dataset refers to the complete collection of all available data points.
    - **ES.** Un conjunto de datos completo se refiere a la colección total de todos los puntos de datos disponibles.
