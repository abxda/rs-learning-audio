# Implementation / Implementación

> **level 3** · *primitive* · sources: howto#implementation-roadmap · cy_finland#solution-architecture

**What it is / Qué es.**

- **EN.** The [[process|process]] of deploying a [[methods|method]], [[system|system]], or [[algorithm|algorithm]] into [[operational-use|operational use]]. Implementation covers [[setup|setup]], execution, and integration of [[software|software]] and hardware components.
- **ES.** [[process|Proceso]] de poner en uso operativo un método, [[system|sistema]] o [[algorithm|algoritmo]]. La implementación abarca la [[setup|configuración]], ejecución e integración de componentes de [[package|software]] y hardware.

## Understand these first / Entiende esto primero
- [[methods|Methods]] — *method*
    - **EN.** Implementation puts a method into practice within a specific operational context.
    - **ES.** La implementación pone en práctica un método dentro de un contexto operativo específico.
- [[development|Development]] — *primitive*
    - **EN.** Implementation follows the development phase to deploy a system for routine use.
    - **ES.** La implementación sigue a la fase de desarrollo para desplegar un sistema para uso rutinario.

## Lineage to fundamentals / Linaje hasta los fundamentos
[[system|System]] → [[methods|Methods]] → [[development|Development]] → [[implementation|Implementation]]

## This unlocks / Esto habilita
- [[operational-application|Operational Application]] — Moving from research to operational application requires robust implementation and automation of the workflow.
