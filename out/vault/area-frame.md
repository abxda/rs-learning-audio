# Area frame / Marco de área

> **level 8** · *statistic* · sources: crop_statistics_regression

**What it is / Qué es.**

- **EN.** A [[sampling|sampling]] [[methods|method]] based on dividing the [[land|land]] surface into contiguous, exhaustive [[area|area]] units (such as [[grid|grid]] cells or segments), from which a [[probability|probability]] [[samples|sample]] is selected. It is used to estimate [[land-cover|land cover]] proportions and [[crop|crop]] areas without relying on pre-existing farmer lists.
- **ES.** Método de [[sampling|muestreo]] que divide la [[earth|superficie terrestre]] en unidades de [[area|área]] exhaustivas y contiguas (como celdas de [[grid|cuadrícula]] o segmentos), de las cuales se selecciona una muestra probabilística. Se utiliza para estimar proporciones de [[land-cover|cobertura del suelo]] y áreas de [[crop|cultivo]] sin depender de listas preexistentes de agricultores.

## Understand these first / Entiende esto primero
- [[grid|Grid]] — *data*
    - **EN.** Area frames use a grid to partition the land into contiguous cells for random sampling.
    - **ES.** Los marcos de área utilizan una cuadrícula para dividir la tierra en celdas para el muestreo aleatorio.
- [[sampling-frame|Sampling frame]] — *method*
    - **EN.** An area frame is a sampling frame based on exhaustive area units like grid cells.
    - **ES.** Un marco de área es un marco de muestreo basado en unidades de área exhaustivas como celdas de cuadrícula.
