# Deep learning / Aprendizaje profundo

> **level 5** · *method* · sources: th_machine_learning

**What it is / Qué es.**

- **EN.** A [[classes|class]] of [[machine-learning-algorithms|machine learning algorithms]] using multi-layered [[neural-networks|neural networks]] to automatically learn hierarchical features from [[large-datasets|large datasets]], widely used for image [[classification-task|classification tasks]].
- **ES.** Una clase de [[machine-learning-algorithms|algoritmos de aprendizaje automático]] que utilizan [[neural-networks|redes neuronales]] multicapa para aprender automáticamente atributos jerárquicos de [[large-datasets|grandes conjuntos de datos]], ampliamente utilizado para [[tasks|tareas]] de [[classification|clasificación]] de [[images|imágenes]].

## Understand these first / Entiende esto primero
- [[machine-learning|Machine learning]] — *method*
    - **EN.** Deep learning is a subset of machine learning using multi-layered neural networks to learn hierarchical features.
    - **ES.** El aprendizaje profundo es un subconjunto del aprendizaje automático que utiliza redes neuronales multicapa para aprender atributos jerárquicos.
- [[neural-networks|Neural networks]] — *method*
    - **EN.** It extends neural networks by adding many hidden layers that automatically extract complex patterns from data.
    - **ES.** Extiende las redes neuronales añadiendo muchas capas ocultas que extraen automáticamente patrones complejos de los datos.
- [[segment-model|Segment Model]] — *method*
    - **EN.** A Segment Model is a deep learning model that performs image segmentation by generating object masks from input images.
    - **ES.** Un modelo de segmentación es un modelo de aprendizaje profundo que realiza segmentación de imágenes generando máscaras de objetos a partir de imágenes de entrada.
