# Field data collection / Recolección de datos de campo

> **level 4** · *method* · sources: th_alignment · th_design_frames

**What it is / Qué es.**

- **EN.** The [[process|process]] of gathering [[ground-observations|ground observations]] such as [[crop-types|crop type]], GPS [[coordinates|coordinates]], and [[phenological-stage|phenological stage]], used for [[training|training]] and validating [[remote-sensing|remote sensing]] models. [[quality|Quality]] protocols ensure fitness for [[earth-observation|Earth observation]] use.
- **ES.** El [[process|proceso]] de recopilar [[ground-observations|observaciones de campo]] como [[type|tipo]] de [[crop|cultivo]], [[coordinates|coordenadas]] GPS y [[phenological-stage|etapa fenológica]], utilizado para entrenar y validar modelos de [[remote-sensing|teledetección]]. Los protocolos de [[quality|calidad]] garantizan la idoneidad para el uso en [[earth-observation|observación de la Tierra]].

## Understand these first / Entiende esto primero
- [[field|Field]] — *phenomenon*
    - **EN.** Field data collection involves gathering ground observations such as crop type and GPS coordinates from agricultural fields.
    - **ES.** La recolección de datos de campo implica recopilar observaciones de campo como tipo de cultivo y coordenadas GPS desde campos agrícolas.
- [[ground-surveys|Ground surveys]] — *method*
    - **EN.** It is a systematic process of recording ground surveys to provide reference data for remote sensing analysis.
    - **ES.** Es un proceso sistemático de registro de encuestas de campo para proporcionar datos de referencia para el análisis de teledetección.
