# Population / Población

> **level 5** · *statistic* · sources: crop_statistics_calibration

**What it is / Qué es.**

- **EN.** The entire [[collection|collection]] of [[statistical|statistical]] units (e.g., all [[pixels|pixels]] in a [[study-area|study area]]) from which a [[samples|sample]] is drawn; in [[survey|survey]] [[sampling|sampling]], the target set for [[inference|inference]].
- **ES.** La [[collection|colección]] completa de unidades [[statistics|estadísticas]] (ej. todos los [[pixels|Píxeles]] en un [[study-area|Área de Estudio]]) de la cual se extrae una Muestra; en encuestas por [[sampling|muestreo]], el [[set|conjunto]] objetivo para la [[inference|Inferencia]].

## Understand these first / Entiende esto primero
- [[sampling|Sampling]] — *method*
    - **EN.** Sampling selects a subset of units from a population to estimate characteristics while reducing cost.
    - **ES.** El muestreo selecciona un subconjunto de unidades de una población para estimar características mientras reduce el costo.
- [[statistical-unit|Statistical unit]] — *primitive*
    - **EN.** The population is the complete set of statistical units (e.g., all fields) from which a sample is drawn.
    - **ES.** La población es el conjunto completo de unidades estadísticas (p. ej., todos los campos) del cual se extrae una muestra.
