# Estimator / Estimador

> **level 4** · *statistic* · sources: crop_statistics_area

**What it is / Qué es.**

- **EN.** A [[statistical|statistical]] rule or formula for calculating an estimate of a [[population|population]] parameter from [[samples|sample]] [[data|data]], such as [[crop-area|crop area]].
- **ES.** Una regla o fórmula estadística para calcular una [[estimation|estimación]] de un parámetro poblacional a partir de [[data|datos]] de muestra, como el [[crop-area|área de cultivo]].

## Understand these first / Entiende esto primero
- [[statistics|Statistics]] — *statistic*
    - **EN.** An estimator uses statistical methods to infer unknown quantities like crop area or yield.
    - **ES.** Un estimador utiliza métodos estadísticos para inferir cantidades desconocidas como el área de cultivo o el rendimiento.
- [[estimation|Estimation]] — *method*
    - **EN.** An estimator is a statistical rule for estimating a population parameter from sample data.
    - **ES.** Un estimador es una regla estadística para estimar un parámetro poblacional a partir de datos muestrales.
