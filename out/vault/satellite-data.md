# Satellite data / Datos satelitales

> **level 7** · *data*

**What it is / Qué es.**

- **EN.** [[information|Information]] collected by [[sensors|sensors]] aboard satellites orbiting [[earth|Earth]], including [[imagery|imagery]] and measurements of [[electromagnetic-radiation|electromagnetic radiation]]. It is used for [[monitoring|monitoring]] [[land|land]], atmosphere, and oceans.
- **ES.** [[information|Información]] recolectada por [[sensors|sensores]] a bordo de satélites que orbitan la [[land|Tierra]], incluyendo [[images|imágenes]] y mediciones de [[electromagnetic-radiation|radiación electromagnética]]. Se utiliza para monitorear tierra, atmósfera y océanos.

## Understand these first / Entiende esto primero
- [[data|Data]] — *primitive*
    - **EN.** Satellite data include imagery and other measurements used as input for Earth observation analysis.
    - **ES.** Los datos satelitales incluyen imágenes y otras mediciones utilizadas como entrada para el análisis de observación de la Tierra.
- [[satellite|Satellite]] — *platform*
    - **EN.** Satellite data are the measurements collected by sensors aboard satellites orbiting Earth.
    - **ES.** Los datos satelitales son las mediciones recopiladas por sensores a bordo de satélites en órbita terrestre.
- [[earthdata|EarthData]] — *data*
    - **EN.** It provides access to satellite data, including Landsat and MODIS, for research and applications.
    - **ES.** Proporciona acceso a datos satelitales, incluyendo Landsat y MODIS, para investigación y aplicaciones.
- [[brazil-data-cube|Brazil Data Cube]] — *data*
    - **EN.** It organizes satellite data into a regular spatiotemporal array for easier analysis and time series processing.
    - **ES.** Organiza los datos satelitales en un arreglo espacio-temporal regular para facilitar el análisis y el procesamiento de series temporales.
