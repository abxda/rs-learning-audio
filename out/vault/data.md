# Data / Datos

> **level 0** · *primitive* · sources: index

**What it is / Qué es.**

- **EN.** Factual [[information|information]] from [[earth-observation|Earth observation]], surveys, or [[sensors|sensors]] used as [[input|input]] for [[processing|processing]] and [[analysis|analysis]], such as [[satellite-imagery|satellite imagery]] or [[ground-measurements|ground measurements]].
- **ES.** [[information|Información]] factual de la [[earth-observation|observación de la Tierra]], encuestas o [[sensors|sensores]] utilizada como [[input|entrada]] para [[processing|procesamiento]] y [[analysis|análisis]], como [[satellite-imagery|imágenes satelitales]] o [[ground-measurements|mediciones de campo]].

## This unlocks / Esto habilita
- [[samples|Samples]] — Samples are individual data points with known labels used for training or validation.
- [[dataset|Dataset]] — A dataset is a structured collection of data, such as satellite images or survey records.
- [[analysis|Analysis]] — Analysis involves systematic examination of data to extract meaningful information.
- [[statistics|Statistics]] — Statistics summarize and describe data, including remote sensing and survey data.
- [[information|Information]] — Information is meaningful data derived from processing and interpreting remote sensing data.
- [[survey|Survey]] — A survey systematically collects ground data, such as field visits, for validation or analysis.
- [[observation|Observation]] — An observation is a single data point from a sensor, e.g., a satellite image acquisition.
- [[machine-learning|Machine learning]] — Machine learning uses algorithms to learn patterns from data and make predictions or decisions.
- [[feature|Feature]] — A feature is a measurable property or characteristic extracted from data, like spectral reflectance.
- [[source|Source]] — A source is the origin of data, such as a satellite mission or cloud platform providing imagery.
- [[large-datasets|Large datasets]] — Large datasets refer to data volumes too big for traditional processing, requiring scalable methods.
- [[chart|Chart]] — It displays data in a graphical format, such as bar charts or line plots.
- [[oci-data-artifacts|OCI Data Artifacts]] — OCI data artifacts are immutable, content-hashed snapshots of chapter-specific data packaged as OCI container images.
- [[big-data|Big Data]] — Big data are extremely large datasets that exceed traditional processing capabilities, requiring specialized methods.
- [[geographic-information-systems|Geographic information systems]] — Geographic information systems are computer systems that capture, store, analyze, and display geospatial data.
- [[cluster|Cluster]] — Data points are grouped by a clustering algorithm into clusters based on similarity metrics.
- [[geospatial-data|Geospatial Data]] — Geospatial data is any data that includes geographic location information.
- [[data-collection|Data collection]] — Data collection gathers observations such as crop type and yield from the field.
- [[metadata|Metadata]] — Metadata is descriptive information about a dataset, such as its source and quality.
- [[input|Input]] — Input is the data fed into a system or model for processing into output.
