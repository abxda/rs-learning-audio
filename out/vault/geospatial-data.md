# Geospatial Data / Datos geoespaciales

> **level 4** · *data* · sources: th_data_sources

**What it is / Qué es.**

- **EN.** [[data|Data]] that contains geographic location [[information|information]], such as [[satellite|satellite]] [[images|images]] or [[vector|vector]] [[boundaries|boundaries]], referenced to a [[coordinate-system|coordinate system]] and used for [[spatial-analysis|spatial analysis]].
- **ES.** [[data|Datos]] que contienen [[information|información]] de ubicación geográfica, como [[satellite-imagery|imágenes satelitales]] o [[boundaries|límites]] vectoriales, referenciados a un [[coordinate-system|sistema de coordenadas]] y utilizados para [[spatial-analysis|análisis espacial]].

## Understand these first / Entiende esto primero
- [[data|Data]] — *primitive*
    - **EN.** Geospatial data is any data that includes geographic location information.
    - **ES.** Los datos geoespaciales son cualquier dato que incluye información de ubicación geográfica.
- [[spatial-information|Spatial information]] — *data*
    - **EN.** Spatial information provides the coordinate reference for geospatial data.
    - **ES.** La información espacial proporciona la referencia de coordenadas para los datos geoespaciales.

## Lineage to fundamentals / Linaje hasta los fundamentos
[[data|Data]] → [[information|Information]] → [[georeferenced-information|Georeferenced information]] → [[spatial-information|Spatial information]] → [[geospatial-data|Geospatial Data]]

## This unlocks / Esto habilita
- [[spatial-analysis|Spatial Analysis]] — It operates on geospatial data, including satellite imagery and vector boundaries, to derive spatial statistics.
- [[gis|GIS]] — A GIS is a software system for capturing, storing, analyzing, and displaying geospatial data layers.
