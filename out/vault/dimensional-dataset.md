# Dimensional dataset / Conjunto de datos dimensional

> **level 2** · *data* · sources: th_data_cubes

**What it is / Qué es.**

- **EN.** A dimensional dataset is an [[earth-observation|Earth observation data]] [[cube|cube]] organized along spatial, temporal, and spectral dimensions, providing regular grids for consistent [[time-series-analysis|time series analysis]] and [[machine-learning|machine learning]] [[classification|classification]].
- **ES.** Un conjunto de datos dimensional es un [[cube|cubo de datos]] de [[earth-observation|observación de la Tierra]] organizado a lo largo de dimensiones espacial, temporal y espectral, que proporciona cuadrículas regulares para [[analysis|análisis]] consistente de series temporales y [[classification|clasificación]] con [[machine-learning|aprendizaje automático]].

## Understand these first / Entiende esto primero
- [[cube|Cube]] — *data*
    - **EN.** A dimensional dataset organizes Earth observation data into a multi-dimensional array called a data cube.
    - **ES.** Un conjunto de datos dimensional organiza los datos de observación de la Tierra en un arreglo multidimensional llamado cubo de datos.
