# SMOTE / SMOTE

> **level 1** · *method* · sources: th_quality_control

**What it is / Qué es.**

- **EN.** Synthetic Minority [[oversampling|Oversampling]] Technique: a [[data|data]] augmentation [[methods|method]] that generates synthetic [[samples|samples]] for minority [[classes|classes]] by interpolating between nearest neighbors in the [[feature|feature]] space. It is used to rebalance imbalanced [[training|training]] datasets, reducing [[bias|bias]] towards majority classes and improving classifier [[performance|performance]].
- **ES.** Técnica de sobremuestreo sintético de minorías: un método de aumento de [[data|datos]] que genera [[samples|muestras]] sintéticas para [[classes|clases]] minoritarias mediante interpolación entre vecinos más cercanos en el espacio de atributos. Se utiliza para reequilibrar conjuntos de [[training-dataset|datos de entrenamiento]] desbalanceados, reduciendo el [[bias|sesgo]] hacia las clases mayoritarias y mejorando el [[yield|rendimiento]] del clasificador.

## Understand these first / Entiende esto primero
- [[oversampling|Oversampling]] — *method*
    - **EN.** SMOTE oversamples minority classes by generating synthetic examples, balancing imbalanced training datasets.
    - **ES.** SMOTE sobremuestrea las clases minoritarias generando ejemplos sintéticos, equilibrando conjuntos de datos de entrenamiento desbalanceados.
