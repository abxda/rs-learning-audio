# High-performance computing / Computación de alto rendimiento

> **level 3** · *software* · sources: cy_finland

**What it is / Qué es.**

- **EN.** Powerful computing [[resources|resources]] (e.g., supercomputers, GPU clusters) required to [[process|process]] large [[earth-observation|Earth observation data]] volumes and run complex [[machine-learning-models|machine learning models]] efficiently.
- **ES.** [[resources|Recursos]] informáticos potentes (por [[example|ejemplo]], supercomputadoras, clústeres GPU) necesarios para procesar [[large-datasets|grandes volúmenes de datos]] de [[earth-observation|observación de la Tierra]] y ejecutar [[machine-learning-models|modelos de aprendizaje automático]] complejos de manera eficiente.

## Understand these first / Entiende esto primero
- [[processing|Processing]] — *method*
    - **EN.** HPC uses parallel processing and specialized hardware (e.g., GPUs) to accelerate Earth observation workflows.
    - **ES.** La computación de alto rendimiento utiliza procesamiento paralelo y hardware especializado (por ejemplo, GPU) para acelerar los flujos de trabajo de observación de la Tierra.
- [[big-data|Big Data]] — *data*
    - **EN.** High‑performance computing (HPC) provides the processing power needed to handle big data and complex models.
    - **ES.** La computación de alto rendimiento proporciona la potencia de procesamiento necesaria para manejar macrodatos y modelos complejos.
