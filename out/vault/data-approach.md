# Data approach / Enfoque de datos

> **level 3** · *method* · sources: th_data_cubes · crop_statistics_calibration

**What it is / Qué es.**

- **EN.** A [[strategy|strategy]] for handling [[data|data]] in [[analysis|analysis]], such as using [[satellite-image-time-series|satellite image time series]] ([[time|time]]-first) versus single-date [[imagery|imagery]], or combining [[ground-surveys|ground surveys]] with [[remote-sensing|remote sensing]] for improved [[estimates|estimates]].
- **ES.** [[strategy|Estrategia]] para el manejo de [[data|datos]] en el [[analysis|análisis]], como el uso de [[satellite-image-time-series|series temporales de imágenes satelitales]] (primero [[time|tiempo]]) frente a [[images|imágenes]] de una sola fecha, o la combinación de [[ground-surveys|encuestas de campo]] con [[remote-sensing|teledetección]] para mejorar las [[estimates|estimaciones]].

## Understand these first / Entiende esto primero
- [[satellite-image-time-series|Satellite image time series]] — *data*
    - **EN.** A data approach like 'time-first, space-later' leverages satellite image time series for classification.
    - **ES.** Un enfoque de datos como 'tiempo primero, espacio después' aprovecha las series temporales de imágenes satelitales para la clasificación.
