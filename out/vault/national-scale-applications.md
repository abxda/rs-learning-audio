# National scale applications / Aplicaciones a escala nacional

> **level 1** · *other* · sources: ct_zimbabwe

**What it is / Qué es.**

- **EN.** The [[operational-use|operational use]] of [[remote-sensing|remote sensing]] and [[statistical-methods|statistical methods]] to generate [[official-agricultural-statistics|official agricultural statistics]] for an entire country, supporting policy and [[food-security|food security]] [[monitoring|monitoring]].
- **ES.** El uso operativo de [[methods|métodos]] de [[remote-sensing|teledetección]] y [[statisticians|estadísticos]] para generar [[official-agricultural-statistics|estadísticas agrícolas oficiales]] para un país entero, apoyando el [[monitoring|monitoreo]] de políticas y [[food-security|seguridad alimentaria]].

## Understand these first / Entiende esto primero
- [[scale|Scale]] — *metric*
    - **EN.** National-scale applications apply remote sensing methods across an entire country for official statistics.
    - **ES.** Las aplicaciones a escala nacional aplican métodos de teledetección en todo un país para producir estadísticas oficiales.
