# Better reference data series / Mejor serie de datos de referencia

> **level 4** · *data* · sources: ct_poland

**What it is / Qué es.**

- **EN.** A set of [[reference-datasets|reference data]], such as [[ground-observations|field observations]], that is more comprehensive, accurate, or timely than previously available, improving [[model-training|model training]] and [[validation|validation]].
- **ES.** Un [[dataset|conjunto de datos]] de [[reference|referencia]], como [[ground-observations|observaciones de campo]], que es más completo, preciso u oportuno que los disponibles anteriormente, mejorando el [[training|entrenamiento]] y la [[validation|validación]] del [[model|modelo]].

## Understand these first / Entiende esto primero
- [[time-series|Time series]] — *data*
    - **EN.** A better reference data series may include temporal information to capture phenological patterns.
    - **ES.** Una mejor serie de datos de referencia puede incluir información temporal para capturar patrones fenológicos.
- [[training-dataset|Training dataset]] — *data*
    - **EN.** A better reference data series serves as an improved training dataset for classification models.
    - **ES.** Una mejor serie de datos de referencia sirve como un conjunto de datos de entrenamiento mejorado para modelos de clasificación.
