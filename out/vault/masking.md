# Masking / Enmascaramiento

> **level 2** · *method* · sources: th_data_sources

**What it is / Qué es.**

- **EN.** The [[process|process]] of setting certain [[pixels|pixels]] to no-[[data|data]] [[values|values]] based on criteria such as [[cloud|cloud]] [[cover|cover]], shadow, or non-[[agricultural|agricultural]] [[land|land]], to exclude them from [[analysis|analysis]].
- **ES.** [[process|Proceso]] de asignar [[values|valores]] sin [[data|datos]] a ciertos [[pixels|píxeles]] basándose en criterios como [[cover|cobertura]] de nubes, sombra o [[land|tierra]] no [[agricultural|agrícola]], para excluirlos del [[analysis|análisis]].

## Understand these first / Entiende esto primero
- [[images|Images]] — *data*
    - **EN.** Masking removes unwanted pixels from images, such as those affected by clouds or shadows.
    - **ES.** El enmascaramiento elimina píxeles no deseados de las imágenes, como aquellos afectados por nubes o sombras.
- [[cloud|Cloud]] — *phenomenon*
    - **EN.** Cloud masking identifies and removes cloud-covered pixels to obtain clear observations.
    - **ES.** El enmascaramiento de nubes identifica y elimina píxeles cubiertos por nubes para obtener observaciones claras.

[[images|Images]] → [[cloud|Cloud]] → [[masking|Masking]]

## This unlocks / Esto habilita
- [[cloud-shadow|Cloud shadow]] — Cloud shadow is identified and removed by masking algorithms that detect adjacent dark pixels.
