# Labeled samples / Muestras etiquetadas

> **level 4** · *data* · sources: th_quality_control

**What it is / Qué es.**

- **EN.** [[data|Data]] [[points|points]] (e.g., [[pixels|pixels]] or [[polygons|polygons]]) that have been assigned a known [[classes|class]] [[label|label]] by an expert, used to train and validate supervised [[machine-learning-models|machine learning models]].
- **ES.** [[points|Puntos]] de [[data|datos]] (por [[example|ejemplo]], [[pixels|píxeles]] o [[polygons|polígonos]]) a los que un experto ha asignado una [[label|etiqueta]] de clase conocida, utilizados para entrenar y validar [[machine-learning-models|modelos de aprendizaje automático]] supervisado.

## Understand these first / Entiende esto primero
- [[samples|Samples]] — *data*
    - **EN.** Labeled samples are observations that have been assigned a ground truth class by an expert.
    - **ES.** Las muestras etiquetadas son observaciones a las que un experto ha asignado una clase de verdad terreno.
- [[training-dataset|Training dataset]] — *data*
    - **EN.** A training dataset consists of labeled samples that serve as ground truth for supervised learning.
    - **ES.** Un conjunto de datos de entrenamiento consiste en muestras etiquetadas que sirven como verdad terreno para el aprendizaje supervisado.
- [[reference-datasets|Reference datasets]] — *data*
    - **EN.** Each reference dataset consists of labeled samples that specify the correct class for a location.
    - **ES.** Cada conjunto de datos de referencia consiste en muestras etiquetadas que especifican la clase correcta para una ubicación.
- [[label|Label]] — *data*
    - **EN.** A label is the correct class category attached to each sample for supervised learning.
    - **ES.** Una etiqueta es la categoría de clase correcta asignada a cada muestra para el aprendizaje supervisado.
