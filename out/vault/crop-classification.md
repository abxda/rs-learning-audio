# Crop classification / Clasificación de cultivos

> **level 3** · *method* · sources: ct_chile

**What it is / Qué es.**

- **EN.** Crop classification is the [[process|process]] of assigning [[crop-types|crop type]] [[label|labels]] to [[satellite-imagery|satellite image]] [[pixels|pixels]] or [[field|field]] [[parcels|parcels]] using [[machine-learning-algorithms|machine learning algorithms]] trained on [[reference-datasets|reference data]], enabling [[production|production]] of [[crop|crop]]-[[type|type]] maps for [[agricultural-statistics|agricultural statistics]].
- **ES.** La clasificación de cultivos es el [[process|proceso]] de asignar etiquetas de tipos de [[crop|cultivo]] a [[pixels|píxeles]] de [[satellite-imagery|imágenes satelitales]] o [[parcels|parcelas]] de [[field|campo]] utilizando [[machine-learning-algorithms|algoritmos de aprendizaje automático]] entrenados con [[ground-truth-data|datos de referencia]], lo que permite la [[production|producción]] de mapas de tipos de cultivo para [[agricultural-statistics|estadísticas agrícolas]].

## Understand these first / Entiende esto primero
- [[classification|Classification]] — *method*
    - **EN.** Crop classification is a type of classification that specifically assigns crop type labels using satellite imagery.
    - **ES.** La clasificación de cultivos es un tipo de clasificación que asigna etiquetas de tipos de cultivo mediante imágenes satelitales.
