# Label / Etiqueta

> **level 2** · *data* · sources: th_quality_control · th_machine_learning

**What it is / Qué es.**

- **EN.** A categorical descriptor assigned to a [[samples|sample]] or [[pixels|pixel]] in [[classification-process|supervised classification]], representing the [[land-cover|land cover]] or [[crop-types|crop type]] it belongs to.
- **ES.** Un descriptor categórico asignado a una muestra o píxel en [[classification|clasificación]] supervisada, que representa la [[land-cover|cobertura del suelo]] o el [[type|tipo]] de [[crop|cultivo]] al que pertenece.

## Understand these first / Entiende esto primero
- [[classes|Classes]] — *data*
    - **EN.** A label is the specific class assignment for a pixel or sample, using the set of classes.
    - **ES.** Una etiqueta es la asignación específica de clase para un píxel o muestra, utilizando el conjunto de clases.
- [[samples|Samples]] — *data*
    - **EN.** Labels are assigned to each sample in the training set, providing known categories for classification.
    - **ES.** Las etiquetas se asignan a cada muestra en el conjunto de entrenamiento, proporcionando categorías conocidas para la clasificación.

[[data|Data]] → [[samples|Samples]] → [[label|Label]]

## This unlocks / Esto habilita
- [[identification|Identification]] — Identification assigns a known label to a pixel or object after analyzing its characteristics.
- [[training-samples|Training samples]] — Training samples consist of labeled observations that guide the model during supervised learning.
- [[labeled-samples|Labeled samples]] — A label is the correct class category attached to each sample for supervised learning.
