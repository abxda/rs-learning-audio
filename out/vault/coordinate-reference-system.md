# Coordinate Reference System / Sistema de referencia de coordenadas

> **level 5** · *other* · sources: uav_agriculture_cook_islands

**What it is / Qué es.**

- **EN.** A [[framework|framework]] for defining real-world geographic [[coordinates|coordinates]], such as WGS 84 (latitude/longitude) or UTM (projected meters). Harmonizing CRS between [[data|data]] [[layers|layers]] is essential for accurate [[spatial-analysis|spatial analysis]].
- **ES.** Un [[framework|marco]] para definir [[coordinates|coordenadas]] geográficas del mundo real, como WGS 84 (latitud/longitud) o UTM (metros proyectados). Armonizar el sistema de referencia de coordenadas entre los conjuntos de [[data|datos]] es esencial para un [[spatial-analysis|análisis espacial]] preciso.

## Understand these first / Entiende esto primero
- [[geographic-information-systems|Geographic information systems]] — *software*
    - **EN.** In GIS, the CRS is essential for aligning and analyzing spatial data from different sources.
    - **ES.** En SIG, el sistema de referencia de coordenadas es esencial para alinear y analizar datos espaciales de diferentes fuentes.
- [[coordinates|Coordinates]] — *primitive*
    - **EN.** A coordinate reference system (CRS) defines how geographic coordinates map to positions on the Earth.
    - **ES.** Un sistema de referencia de coordenadas (CRS) define cómo las coordenadas geográficas se asignan a posiciones en la Tierra.
