# JupyterLab / JupyterLab

> **level 3** · *software* · sources: howto

**What it is / Qué es.**

- **EN.** A web-based interactive [[development|development]] [[environment|environment]] for Jupyter notebooks, code, and [[data|data]]. It enables [[reproducible-analysis-system|reproducible analysis]] by providing a pre-configured environment with all dependencies and data ready to run.
- **ES.** [[environment|Entorno]] de [[development|desarrollo]] interactivo basado en web para Jupyter Notebooks, código y [[data|datos]]. Permite [[analysis|análisis]] reproducible al proporcionar un entorno preconfigurado con todas las dependencias y datos listos para ejecutar.

## Understand these first / Entiende esto primero
- [[reproducible-analysis-sessions|Reproducible analysis sessions]] — *method*
    - **EN.** Reproducible analysis sessions are one-click JupyterLab environments pre-configured with code, data, and dependencies.
    - **ES.** Las sesiones de análisis reproducibles son entornos JupyterLab de un solo clic preconfigurados con código, datos y dependencias.
- [[reproducible-analysis-system|Reproducible analysis system]] — *method*
    - **EN.** JupyterLab provides the interactive web-based interface for launching the reproducible analysis system.
    - **ES.** JupyterLab proporciona la interfaz web interactiva para lanzar el sistema de análisis reproducible.
- [[software|Software]] — *primitive*
    - **EN.** JupyterLab provides a platform for interactive code development, data analysis, and visualization.
    - **ES.** JupyterLab proporciona una plataforma para el desarrollo interactivo de código, análisis de datos y visualización.
