# SOM analysis / análisis SOM

> **level 0** · *method* · sources: th_quality_control

**What it is / Qué es.**

- **EN.** A [[quality-control|quality control]] [[methods|method]] for [[training-dataset|training data]] that uses [[self-organizing-maps|Self-Organizing Maps]] to [[cluster|cluster]] [[time-series|time series]], detect confusion between [[label|labels]], and identify noisy or mislabeled [[samples|samples]] for removal.
- **ES.** Un método de [[quality-control|control de calidad]] para [[training-dataset|datos de entrenamiento]] que utiliza [[self-organizing-maps|mapas autoorganizados]] para agrupar series temporales, detectar confusión entre etiquetas e identificar [[samples|muestras]] ruidosas o mal etiquetadas para su eliminación.

## This unlocks / Esto habilita
- [[self-organizing-maps|Self-Organizing Maps]] — SOM analysis uses Self-Organizing Maps to cluster time series and detect confusions in training data.
- [[quality-control|Quality Control]] — It serves as a quality control step to identify mislabeled samples before training a classifier.
