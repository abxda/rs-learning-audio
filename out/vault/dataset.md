# Dataset / Conjunto de datos

> **level 1** · *data* · sources: th_data_sources

**What it is / Qué es.**

- **EN.** A structured [[collection|collection]] of [[data|data]], such as [[satellite-image-collections|satellite image collections]], [[ground|ground]] [[survey-records|survey records]], or [[tabular-statistics|tabular statistics]], used together in [[analysis|analysis]].
- **ES.** [[collection|Colección]] estructurada de [[data|datos]], como [[satellite-image-collections|colecciones de imágenes satelitales]], [[survey-records|registros de encuestas]] de [[field|campo]] o [[tabular-statistics|estadísticas tabulares]], utilizada conjuntamente en [[analysis|análisis]].

## Understand these first / Entiende esto primero
- [[data|Data]] — *primitive*
    - **EN.** A dataset is a structured collection of data, such as satellite images or survey records.
    - **ES.** Un conjunto de datos es una colección estructurada de datos, como imágenes satelitales o registros de encuestas.

## This unlocks / Esto habilita
- [[collection|Collection]] — A collection is a set of datasets or satellite images sharing a common source, format, and theme.
- [[spatial-datasets|Spatial datasets]] — Spatial datasets are datasets that include geographic location information, such as satellite images or vector boundaries.
- [[source-datasets|Source datasets]] — Source datasets are the original datasets used as inputs, such as satellite imagery or survey data.
- [[public-datasets|Public datasets]] — Public datasets are openly available collections of satellite imagery or ground observations.
- [[entire-dataset|Entire dataset]] — An entire dataset refers to the complete collection of all available data points.
- [[database|Database]] — A database is a structured collection of datasets designed for efficient storage and retrieval.
- [[reference-datasets|Reference datasets]] — They are curated datasets containing labeled samples for the study area.
- [[uhr-cfp-dataset|UHR CFP dataset]] — The UHR CFP dataset is a labeled collection of UAV images for parcel detection and segmentation.
- [[data-division|Data Division]] — Data division splits a dataset into training, validation, and test subsets for model development.
- [[random-training-testing-datasets|Random training testing datasets]] — Random training-testing datasets split a dataset into subsets by random assignment for evaluation.
- [[national-datasets|National datasets]] — National datasets are datasets covering an entire country, so they build from the general dataset concept.
- [[training-dataset|Training dataset]] — A training dataset is a labeled dataset used to train models, extending the general dataset concept.
