# Machine learning / Aprendizaje automático

> **level 3** · *method* · sources: th_machine_learning · introduction

**What it is / Qué es.**

- **EN.** A branch of artificial intelligence that uses [[algorithm|algorithms]] to learn [[patterns|patterns]] from [[data|data]] and make [[predictions|predictions]] or decisions without explicit programming for each task.
- **ES.** Una rama de la inteligencia artificial que utiliza algoritmos para aprender [[patterns|patrones]] a partir de [[data|datos]] y hacer [[predictions|predicciones]] o decisiones sin programación explícita para cada tarea.

## Understand these first / Entiende esto primero
- [[data|Data]] — *primitive*
    - **EN.** Machine learning uses algorithms to learn patterns from data and make predictions or decisions.
    - **ES.** El aprendizaje automático utiliza algoritmos para aprender patrones de datos y hacer predicciones o decisiones.
- [[training|Training]] — *method*
    - **EN.** Training uses machine learning algorithms to learn patterns from data.
    - **ES.** El entrenamiento utiliza algoritmos de aprendizaje automático para aprender patrones a partir de datos.

## This unlocks / Esto habilita
- [[algorithm|Algorithm]] — Machine learning involves algorithms that improve through experience with data.
- [[early-season-crop-yield-mapping|Early season crop yield mapping]] — Machine learning models predict early season crop yield from satellite features.
- [[classification-method|Classification method]] — A classification method is a machine learning algorithm that learns from samples to assign classes.
- [[machine-learning-models|Machine learning models]] — Machine learning models are algorithms trained on data to recognize patterns and make predictions.
- [[neural-networks|Neural networks]] — Neural networks are a class of machine learning algorithms inspired by biological neurons.
- [[random-forest|Random Forest]] — This ensemble method reduces overfitting compared to a single decision tree.
- [[machine-learning-algorithms|Machine learning algorithms]] — Machine learning algorithms are specific techniques like random forest or SVM used within machine learning.
- [[deep-learning|Deep learning]] — Deep learning is a subset of machine learning using multi-layered neural networks to learn hierarchical features.
- [[prediction-probabilities|Prediction Probabilities]] — Machine learning models often produce prediction probabilities for each class, indicating confidence.
- [[self-organizing-maps|Self-Organizing Maps]] — Self-organizing maps use a neural network to learn a topological representation of input data without labels.
- [[multi-task-learning|Multi-task learning]] — Multi-task learning trains a single model simultaneously on multiple related tasks, improving generalization.
- [[data-science|Data Science]] — Using machine learning algorithms, data science builds models from data for predictions.
- [[crop-classification-models|Crop classification models]] — Machine learning provides the algorithms that learn patterns from labeled samples.
