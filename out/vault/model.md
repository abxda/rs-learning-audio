# Model / Modelo

> **level 0** · *method* · sources: th_machine_learning

**What it is / Qué es.**

- **EN.** A mathematical or algorithmic representation trained on [[data|data]] to perform [[tasks|tasks]] such as classifying [[land-cover|land cover]] or predicting [[crop-yield|crop yield]].
- **ES.** Representación matemática o algorítmica entrenada con [[data|datos]] para realizar [[tasks|tareas]] como clasificar [[land-cover|cobertura del suelo]] o predecir [[crop-yield|rendimiento de cultivos]].

## This unlocks / Esto habilita
- [[training|Training]] — A model is created through training on labeled data to perform tasks like classification.
- [[classification|Classification]] — A trained classification model maps input data to class labels.
- [[performance|Performance]] — Performance evaluates how well a model achieves its task using metrics like accuracy.
- [[parameters|Parameters]] — Parameters are configurable values that control the behavior of a model, such as learning rate or tree count.
- [[predictions|Predictions]] — Predictions are outputs of models that estimate agricultural quantities like yield or area from satellite data.
- [[machine|Machine]] — A machine is a trained model that maps input features to output predictions.
- [[predictors|Predictors]] — In a model, predictors are the input variables that help forecast the outcome.
- [[machine-learning-models|Machine learning models]] — A model is a representation that learns from features and labels to perform classification or regression.
- [[statistical-indicators|Statistical indicators]] — Model performance is evaluated using statistical indicators like RMSE and R-squared.
- [[model-generalization-capabilities|Model generalization capabilities]] — A model's generalization capability refers to its ability to perform well on new, unseen data beyond its training set.
- [[plsr-model|PLSR model]] — PLSR model is a statistical model that relates predictor variables to a response variable.
- [[model-development|Model Development]] — Model development is the process of creating, training, and tuning a computational model.
- [[model-evaluation|Model evaluation]] — Model evaluation assesses how well a trained model performs on unseen data.
- [[modeling|Modeling]] — Modeling is the process of constructing a mathematical or computational representation from data.
- [[model-ability|Model ability]] — Model ability refers to the capability of a model to generalize or discriminate across tasks.
- [[satellite-derived-models|Satellite-derived models]] — Satellite-derived models use satellite imagery as input to predict biophysical variables like crop yield or biomass.
- [[transferability|Transferability]] — Transferability is a model's ability to maintain predictive performance when applied to new regions or times.
