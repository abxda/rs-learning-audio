# Remote image classification / Clasificación de imágenes remotas

> **level 3** · *method* · sources: th_quality_control · th_machine_learning

**What it is / Qué es.**

- **EN.** The [[process|process]] of assigning semantic [[label|labels]], such as [[crop-types|crop type]], to [[pixels|pixels]] in [[satellite-imagery|satellite imagery]] using [[algorithm|algorithms]] trained on [[reference-datasets|reference data]].
- **ES.** [[process|Proceso]] de asignar etiquetas semánticas, como el [[type|tipo]] de [[crop|cultivo]], a [[pixels|píxeles]] en [[satellite-imagery|imágenes satelitales]] utilizando algoritmos entrenados con [[ground-truth-data|datos de referencia]].

## Understand these first / Entiende esto primero
- [[classification|Classification]] — *method*
    - **EN.** Remote image classification applies the general classification process to satellite imagery instead of other data.
    - **ES.** La clasificación de imágenes remotas aplica el proceso general de clasificación a imágenes satelitales en lugar de otros datos.
- [[images|Images]] — *data*
    - **EN.** Remote image classification uses satellite images as the input data to assign class labels.
    - **ES.** La clasificación de imágenes remotas utiliza imágenes satelitales como datos de entrada para asignar etiquetas de clase.
