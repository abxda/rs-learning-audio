# Component / Componente

> **level 1** · *primitive* · sources: howto#the-five-core-components

**What it is / Qué es.**

- **EN.** A custom [[software|software]] element in the [[reproducible-analysis-system|reproducible analysis system]] that performs a specific task, such as building compute [[images|images]], packaging [[data|data]] artifacts, generating a button, or deploying a session.
- **ES.** Un elemento de [[package|software]] personalizado en el [[reproducible-analysis-system|sistema de análisis reproducible]] que realiza una tarea específica, como construir [[images|imágenes]] de cómputo, empaquetar artefactos de [[data|datos]], generar un botón o implementar una sesión.

## Understand these first / Entiende esto primero
- [[system|System]] — *other*
    - **EN.** The system defines the framework into which components are integrated as functional modules.
    - **ES.** El sistema define el marco en el cual los componentes se integran como módulos funcionales.
- [[function|Function]] — *primitive*
    - **EN.** Functions are atomic executables that components rely on to perform specific tasks.
    - **ES.** Las funciones son ejecutables atómicos de los que dependen los componentes para realizar tareas específicas.
- [[software|Software]] — *primitive*
    - **EN.** Components are custom software pieces that execute well-defined analytical functions.
    - **ES.** Los componentes son piezas de software personalizadas que ejecutan funciones analíticas bien definidas.
