# Classification task / Tarea de clasificación

> **level 3** · *method* · sources: th_machine_learning

**What it is / Qué es.**

- **EN.** A specific problem in [[remote-sensing|remote sensing]] where the goal is to assign a [[label|label]], such as [[crop-types|crop type]] or [[land-cover-classes|land cover class]], to each [[pixels|pixel]] or image object.
- **ES.** Un problema específico en [[remote-sensing|teledetección]] cuyo objetivo es asignar una [[label|etiqueta]], como [[type|tipo]] de [[crop|cultivo]] o clase de [[land-cover|cobertura del suelo]], a cada píxel u objeto de la imagen.

## Understand these first / Entiende esto primero
- [[classification|Classification]] — *method*
    - **EN.** A classification task is a specific problem where a classifier assigns labels to data.
    - **ES.** Una tarea de clasificación es un problema específico donde un clasificador asigna etiquetas a datos.
