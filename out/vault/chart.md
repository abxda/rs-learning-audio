# Chart / Gráfico

> **level 2** · *primitive* · sources: th_quality_control

**What it is / Qué es.**

- **EN.** A graphical representation of [[data|data]], such as a bar chart, [[line|line]] [[plot|plot]], or [[confusion-matrix|confusion matrix]], used to visualize [[results|results]] like [[accuracy-metrics|accuracy metrics]] or [[area-statistics|area estimates]].
- **ES.** Una representación gráfica de [[data|datos]], como un gráfico de barras, un gráfico de líneas o una [[confusion-matrix|matriz de confusión]], utilizada para visualizar [[results|resultados]] como [[accuracy-metrics|métricas de precisión]] o [[estimates|estimaciones]] de [[area|área]].

## Understand these first / Entiende esto primero
- [[data|Data]] — *primitive*
    - **EN.** It displays data in a graphical format, such as bar charts or line plots.
    - **ES.** Muestra datos en un formato gráfico, como gráficos de barras o gráficos de líneas.
- [[figure|Figure]] — *primitive*
    - **EN.** A chart is a visual element that graphically represents data trends or comparisons.
    - **ES.** Un gráfico es un elemento visual que representa gráficamente tendencias o comparaciones de datos.
