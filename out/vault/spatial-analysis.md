# Spatial Analysis / Análisis espacial

> **level 5** · *method* · sources: uav_agriculture_cook_islands

**What it is / Qué es.**

- **EN.** The [[process|process]] of examining locations, attributes, and relationships of features in [[spatial-datasets|spatial data]]. In this [[handbook|handbook]], it includes operations like [[zonal-statistics|zonal statistics]], [[buffering|buffering]], and [[coordinate-reference-system|coordinate reference system]] harmonization.
- **ES.** El [[process|proceso]] de examinar ubicaciones, atributos y relaciones de características en [[spatial-datasets|datos espaciales]]. En este [[handbook|manual]], incluye operaciones como [[zonal-statistics|estadísticas zonales]], [[buffering|búfer]] y armonización del [[coordinate-reference-system|sistema de referencia de coordenadas]].

## Understand these first / Entiende esto primero
- [[geographic-information-systems|Geographic information systems]] — *software*
    - **EN.** Spatial analysis uses GIS tools to examine locations, patterns, and relationships in geographic data.
    - **ES.** El análisis espacial utiliza herramientas SIG para examinar ubicaciones, patrones y relaciones en datos geográficos.
- [[geospatial-data|Geospatial Data]] — *data*
    - **EN.** It operates on geospatial data, including satellite imagery and vector boundaries, to derive spatial statistics.
    - **ES.** Opera sobre datos geoespaciales, incluyendo imágenes satelitales y límites vectoriales, para derivar estadísticas espaciales.
- [[geospatial-analysis|Geospatial analysis]] — *method*
    - **EN.** Geospatial analysis applies spatial analysis techniques to geographic data like satellite imagery.
    - **ES.** El análisis geoespacial aplica técnicas de análisis espacial a datos geográficos como imágenes satelitales.
