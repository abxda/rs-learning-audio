# Regression / Regresión

> **level 5** · *method* · sources: cy_poland

**What it is / Qué es.**

- **EN.** A [[statistical|statistical]] [[methods|method]] that models the relationship between a dependent variable (e.g., [[crop-yield|crop yield]]) and one or more independent variables (e.g., [[vegetation-indices|vegetation indices]]), used for [[predictions|prediction]].
- **ES.** Método [[statistical|estadístico]] que modela la relación entre una variable dependiente (por [[example|ejemplo]], [[yield|rendimiento]] del [[crop|cultivo]]) y una o más variables independientes (por ejemplo, [[vegetation-indices|índices de vegetación]]), utilizado para la predicción.

## Understand these first / Entiende esto primero
- [[statistics|Statistics]] — *statistic*
    - **EN.** Regression is a statistical method that models relationships between a dependent and independent variables.
    - **ES.** La regresión es un método estadístico que modela relaciones entre una variable dependiente y una independiente.
- [[values|Values]] — *primitive*
    - **EN.** It uses numeric values of predictors to predict a continuous outcome, such as crop yield.
    - **ES.** Utiliza valores numéricos de predictores para predecir un resultado continuo, como el rendimiento de cultivos.
- [[plsr-model|PLSR model]] — *method*
    - **EN.** PLSR model is a multivariate regression method that handles many correlated predictors.
    - **ES.** El modelo PLSR es un método de regresión multivariante que maneja muchos predictores correlacionados.
- [[crop-yield-estimation|Crop yield estimation]] — *method*
    - **EN.** Crop yield estimation often employs regression models relating vegetation indices to yield.
    - **ES.** La estimación del rendimiento de cultivos a menudo emplea modelos de regresión que relacionan índices de vegetación con el rendimiento.
