# Spatiotemporal array / Arreglo espacio-temporal

> **level 7** · *data* · sources: th_data_cubes

**What it is / Qué es.**

- **EN.** A three-dimensional array of [[earth-observation|Earth observation data]] with two spatial dimensions and one [[temporal-dimension|temporal dimension]], where each cell has a fixed spatial [[resolution|resolution]], temporal duration, and set of spectral attributes.
- **ES.** Arreglo tridimensional de [[data|datos]] de [[earth-observation|observación de la Tierra]] con dos dimensiones espaciales y una [[temporal-dimension|dimensión temporal]], donde cada celda tiene una [[resolution|resolución]] espacial fija, una duración temporal y un [[set|conjunto]] de atributos espectrales.

## Understand these first / Entiende esto primero
- [[time|Time]] — *primitive*
    - **EN.** A spatiotemporal array organizes satellite imagery along both spatial coordinates and a time dimension, enabling analysis of change.
    - **ES.** Un arreglo espacio-temporal organiza las imágenes satelitales a lo largo de coordenadas espaciales y una dimensión temporal, lo que permite el análisis de cambios.
- [[satellite-data-cube|Satellite data cube]] — *data*
    - **EN.** A spatiotemporal array is a data cube where layers represent time slices of registered satellite images.
    - **ES.** Un arreglo espacio-temporal es un cubo de datos donde las capas representan cortes temporales de imágenes satelitales registradas.
