# Analysis Ready Data / Datos Listos para Análisis

> **level 2** · *data* · sources: th_data_sources

**What it is / Qué es.**

- **EN.** [[satellite-data|Satellite data]] processed to meet CEOS standards, including geometric, radiometric, and atmospheric corrections, with [[cloud|cloud]] and shadow masks, and [[surface-reflectance|surface reflectance]] [[values|values]] for [[optical|optical]] [[sensors|sensors]] or GammaNought backscatter for SAR.
- **ES.** [[satellite-data|Datos satelitales]] procesados para cumplir con los estándares de CEOS, incluyendo correcciones geométricas, radiométricas y atmosféricas, con máscaras de nubes y sombras, y [[values|valores]] de [[surface-reflectance|reflectancia superficial]] para [[sensors|sensores]] ópticos o retrodispersión GammaNought para SAR.

## Understand these first / Entiende esto primero
- [[images|Images]] — *data*
    - **EN.** Analysis Ready Data (ARD) are satellite images that have been processed to a consistent, analysis-ready state.
    - **ES.** Los Datos Listos para Análisis son imágenes satelitales procesadas a un estado consistente y listo para análisis.
- [[processing|Processing]] — *method*
    - **EN.** ARD is produced by applying geometric, radiometric, and atmospheric corrections to raw satellite imagery.
    - **ES.** Los Datos Listos para Análisis se producen aplicando correcciones geométricas, radiométricas y atmosféricas a las imágenes satelitales en bruto.
- [[ard-image-collections|ARD image collections]] — *data*
    - **EN.** ARD image collections are sets of satellite images processed to Analysis Ready Data standards.
    - **ES.** Las colecciones de imágenes ARD son conjuntos de imágenes satelitales procesadas según los estándares de Datos Listos para Análisis.
