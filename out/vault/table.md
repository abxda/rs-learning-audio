# Table / Tabla

> **level 1** · *primitive* · sources: th_data_cubes

**What it is / Qué es.**

- **EN.** A table is a structured arrangement of [[data|data]] in rows and columns. In the [[satellite-image-time-series-2|sits]] [[package|package]], a [[tibble|tibble]] (a [[type|type]] of table) stores [[time-series|time series]] [[samples|samples]] with [[metadata|metadata]] such as [[label|labels]], [[coordinates|coordinates]], and temporal observations.
- **ES.** Una tabla es una disposición estructurada de [[data|datos]] en filas y columnas. En el [[package|paquete]] sits, un [[tibble|tibble]] (un [[type|tipo]] de tabla) almacena [[samples|muestras]] de series temporales con [[metadata|metadatos]] como etiquetas, [[coordinates|coordenadas]] y observaciones temporales.

## Understand these first / Entiende esto primero
- [[data|Data]] — *primitive*
    - **EN.** A table organizes data into rows and columns for easy analysis.
    - **ES.** Una tabla organiza los datos en filas y columnas para facilitar el análisis.
- [[tibble|Tibble]] — *primitive*
    - **EN.** In R, a tibble is a modern table structure that simplifies data manipulation.
    - **ES.** En R, un tibble es una estructura de tabla moderna que simplifica la manipulación de datos.
