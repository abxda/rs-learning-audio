# List frame / Marco de lista

> **level 8** · *method* · sources: crop_statistics_calibration

**What it is / Qué es.**

- **EN.** A [[sampling|sampling]] [[methods|method]] where the [[population|population]] of [[agricultural|agricultural]] holdings or [[parcels|parcels]] is enumerated in a list (often from a census), and a [[samples|sample]] is selected from that list. It typically provides one [[georeferenced-information|georeferenced]] centroid per parcel without direct [[area|area]] measurement.
- **ES.** Un método de [[sampling|muestreo]] donde la [[population|población]] de explotaciones agrícolas o [[parcels|parcelas]] se enumera en una lista (a menudo de un censo) y se selecciona una muestra de esa lista. Por lo general, proporciona un centroide georreferenciado por [[plot|parcela]] sin medición directa del [[area|área]].

## Understand these first / Entiende esto primero
- [[sampling-frame|Sampling frame]] — *method*
    - **EN.** A list frame is a sampling frame that enumerates all agricultural holdings or parcels in a list.
    - **ES.** Un marco de lista es un marco de muestreo que enumera todas las explotaciones agrícolas o parcelas en una lista.
