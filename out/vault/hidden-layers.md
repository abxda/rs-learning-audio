# Hidden layers / Capas ocultas

> **level 5** · *method* · sources: th_machine_learning

**What it is / Qué es.**

- **EN.** Intermediate [[layers|layers]] in a [[neural-networks|neural network]] between the [[input|input]] and output layers that transform inputs through weighted connections; deeper hidden layers capture more complex features.
- **ES.** [[layers|Capas]] intermedias en una red neuronal entre las capas de [[input|entrada]] y salida que transforman las entradas mediante conexiones ponderadas; las capas ocultas más profundas capturan atributos más complejos.

## Understand these first / Entiende esto primero
- [[neural-networks|Neural networks]] — *method*
    - **EN.** Hidden layers are intermediate layers in a neural network between input and output that transform features.
    - **ES.** Las capas ocultas son capas intermedias en una red neuronal entre la entrada y la salida que transforman los atributos.
