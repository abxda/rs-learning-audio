# Training / Entrenamiento

> **level 2** · *method* · sources: th_quality_control#outline

**What it is / Qué es.**

- **EN.** The [[process|process]] of fitting a [[machine-learning|machine learning]] [[model|model]] to [[labeled-samples|labeled samples]] so it can learn to classify new [[data|data]].
- **ES.** El [[process|proceso]] de ajustar un [[model|modelo]] de [[machine-learning|aprendizaje automático]] a [[labeled-samples|muestras etiquetadas]] para que pueda aprender a clasificar nuevos [[data|datos]].

## Understand these first / Entiende esto primero
- [[model|Model]] — *method*
    - **EN.** A model is created through training on labeled data to perform tasks like classification.
    - **ES.** Un modelo se crea mediante entrenamiento con datos etiquetados para realizar tareas como la clasificación.
- [[samples|Samples]] — *data*
    - **EN.** Training fits a model to labeled samples so it learns to classify new data.
    - **ES.** El entrenamiento ajusta un modelo a las muestras etiquetadas para que aprenda a clasificar nuevos datos.

## Lineage to fundamentals / Linaje hasta los fundamentos
[[data|Data]] → [[samples|Samples]] → [[training|Training]]

## This unlocks / Esto habilita
- [[machine-learning|Machine learning]] — Training uses machine learning algorithms to learn patterns from data.
- [[model-development|Model Development]] — Model development involves training a model on labeled data to learn patterns.
- [[modeling|Modeling]] — Modeling involves training a model on labeled samples to learn patterns.
- [[model-training|Model training]] — Model training is the process of fitting a machine learning algorithm to labeled data by adjusting its parameters.
- [[foundation-model|Foundation model]] — Foundation models are trained on massive unlabeled datasets and can be fine-tuned for specific tasks.
- [[training-dataset|Training dataset]] — A training dataset is the data used during the training process of a machine learning model.
