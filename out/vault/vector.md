# Vector / Vector

> **level 5** · *data* · sources: uav_field_parcels

**What it is / Qué es.**

- **EN.** [[geographic-data|Geographic data]] represented by [[points|points]], lines, and [[polygons|polygons]], used to delineate features like [[field|field]] [[boundaries|boundaries]].
- **ES.** [[geospatial-data|Datos geográficos]] representados por [[points|puntos]], líneas y [[polygons|polígonos]], utilizados para delinear atributos como los [[boundaries|límites]] de los campos.

## Understand these first / Entiende esto primero
- [[polygons|Polygons]] — *data*
    - **EN.** A vector object includes points, lines, and polygons; polygons represent field parcels in mapping.
    - **ES.** Un objeto vectorial incluye puntos, líneas y polígonos; los polígonos representan parcelas de campo en el mapeo.
- [[points|Points]] — *data*
    - **EN.** A vector can represent a point, line, or polygon, enabling spatial analysis of discrete features.
    - **ES.** Un vector puede representar un punto, línea o polígono, permitiendo el análisis espacial de atributos discretos.

## Lineage to fundamentals / Linaje hasta los fundamentos
[[area|Area]] → [[land|Land]] → [[field|Field]] → [[parcels|Parcels]] → [[polygons|Polygons]] → [[vector|Vector]]
