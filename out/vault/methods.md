# Methods / Métodos

> **level 1** · *method* · sources: introduction

**What it is / Qué es.**

- **EN.** The techniques and procedures used in the [[handbook|handbook]], including [[data-collection|data acquisition]], preprocessing, [[classification|classification]], [[validation|validation]], and [[estimation|estimation]].
- **ES.** Las técnicas y procedimientos utilizados en el [[handbook|manual]], incluidos la adquisición de [[data|datos]], el preprocesamiento, la [[classification|clasificación]], la [[validation|validación]] y la [[estimation|estimación]].

## This unlocks / Esto habilita
- [[analysis|Analysis]] — Analysis applies defined methods to transform data into results.
- [[algorithm|Algorithm]] — Methods are systematic techniques, often implemented through algorithms, for processing data.
- [[approach|Approach]] — An approach is a methodology or strategy used to achieve a goal within a set of methods.
- [[development|Development]] — Development is the process of creating or improving analytical methods or tools.
- [[step|Step]] — A step is a specific procedure within a method that is executed to achieve a sub-goal.
- [[process|Process]] — A process implements a method by executing its steps in a defined order.
- [[applications|Applications]] — Applications implement methods to solve real-world problems in agriculture and land management.
- [[methodology|Methodology]] — Methodology is a systematic set of procedures, extending individual methods to ensure consistent and reproducible analysis.
- [[handbook|Handbook]] — A handbook is a practical guide that compiles methods, case studies, and reproducible code for users.
- [[tools|Tools]] — They operationalize methods into usable workflows for crop mapping and monitoring.
- [[implementation|Implementation]] — Implementation puts a method into practice within a specific operational context.
- [[methodological-improvement|Methodological improvement]] — A methodological improvement refines an existing method to enhance accuracy, efficiency, or applicability.
