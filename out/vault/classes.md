# Classes / Clases

> **level 0** · *data* · sources: th_lucc

**What it is / Qué es.**

- **EN.** Discrete categories or [[label|labels]] assigned to [[pixels|pixels]] or [[parcels|parcels]] in a [[classification-scheme|classification scheme]], such as [[crop-types|crop types]] or [[land-cover|land cover]] types.
- **ES.** Categorías discretas o etiquetas asignadas a [[pixels|píxeles]] o [[parcels|parcelas]] en un [[classification-scheme|esquema de clasificación]], como [[crop-types|tipos de cultivos]] o tipos de [[land-cover|cobertura del suelo]].

## This unlocks / Esto habilita
- [[land|Land]] — Classes are discrete categories assigned to land cover types in a classification scheme.
- [[samples|Samples]] — Samples are assigned to classes, providing ground truth for supervised learning.
- [[classification|Classification]] — Classification assigns one of the predefined classes to each pixel or object in imagery.
- [[fao-land-cover-classification-system|FAO Land Cover Classification System]] — The FAO Land Cover Classification System defines hierarchical classes and diagnostic criteria for land cover.
- [[set|Set]] — A set is a collection of distinct elements, such as land cover classes or pixels.
- [[classification-framework|Classification framework]] — It defines how classes are structured and related in a hierarchical system.
- [[land-cover-classes|Land cover classes]] — Land cover classes are a specific set of classes that describe the observed biophysical material on Earth's surface.
- [[classification-scheme|Classification scheme]] — A classification scheme defines the set of classes used to label land cover features.
- [[patterns|Patterns]] — For each class, we extract a typical pattern from its training samples to represent the class signature.
- [[label|Label]] — A label is the specific class assignment for a pixel or sample, using the set of classes.
- [[land-classes|Land classes]] — Land classes are the set of discrete categories, such as forest or cropland, assigned to geographic areas.
- [[confusion-matrix|Confusion matrix]] — Each cell of the confusion matrix counts the number of samples for specific predicted versus actual class combinations.
- [[land-cover|Land cover]] — In classification, land cover is one of the classes assigned to pixels or objects.
- [[strata|Strata]] — Strata are often defined by land cover classes to improve sampling efficiency.
- [[land-classification|Land classification]] — It uses predefined classes, such as forest or cropland, to categorize land cover.
