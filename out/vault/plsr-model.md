# PLSR model / modelo PLSR

> **level 1** · *method* · sources: uav_wheat_growth

**What it is / Qué es.**

- **EN.** A multivariate [[statistical|statistical]] [[methods|method]] that models the relationship between predictor variables and response variables, especially useful when [[predictors|predictors]] are numerous and correlated.
- **ES.** Un método [[statistical|estadístico]] multivariante que modela la relación entre variables predictoras y variables de respuesta, especialmente útil cuando los [[predictors|predictores]] son numerosos y están correlacionados.

## Understand these first / Entiende esto primero
- [[model|Model]] — *method*
    - **EN.** PLSR model is a statistical model that relates predictor variables to a response variable.
    - **ES.** El modelo PLSR es un modelo estadístico que relaciona variables predictoras con una variable de respuesta.
