# Roadmap / Hoja de ruta

> **level 4** · *primitive* · sources: howto · howto

**What it is / Qué es.**

- **EN.** A strategic plan outlining the steps and timeline for developing or deploying a [[system|system]] or [[methodology|methodology]]. The [[handbook|handbook]] provides an [[implementation|implementation]] roadmap for the [[reproducible-analysis-system|reproducible analysis system]].
- **ES.** Un plan estratégico que describe los pasos y el cronograma para desarrollar o implementar un [[system|sistema]] o [[methodology|metodología]]. El [[handbook|manual]] proporciona una hoja de ruta de [[implementation|implementación]] para el [[reproducible-analysis-system|sistema de análisis reproducible]].

## Understand these first / Entiende esto primero
- [[development|Development]] — *primitive*
    - **EN.** It guides the development and deployment of a system or methodology over time.
    - **ES.** Guía el desarrollo y despliegue de un sistema o metodología a lo largo del tiempo.
- [[strategy|Strategy]] — *primitive*
    - **EN.** A roadmap is a strategic plan that outlines steps and milestones for achieving long-term objectives.
    - **ES.** Una hoja de ruta es un plan estratégico que describe pasos e hitos para lograr objetivos a largo plazo.
