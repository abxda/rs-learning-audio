# Crop type mapping / Mapeo de tipos de cultivo

> **level 3** · *method* · sources: ct_mexico

**What it is / Qué es.**

- **EN.** The [[process|process]] of identifying and delineating the spatial [[distribution|distribution]] of different [[crop-types|crop types]] from [[satellite-imagery|satellite imagery]].
- **ES.** El [[process|proceso]] de identificar y delinear la [[distribution|distribución]] espacial de diferentes [[crop-types|tipos de cultivos]] a partir de [[satellite-imagery|imágenes satelitales]].

## Understand these first / Entiende esto primero
- [[crop|Crop]] — *phenomenon*
    - **EN.** Crop type mapping aims to delineate the spatial distribution of different crop types.
    - **ES.** El mapeo de tipos de cultivo busca delinear la distribución espacial de diferentes tipos de cultivos.
- [[classification|Classification]] — *method*
    - **EN.** Crop type mapping is a specific classification task that identifies crop types from satellite imagery.
    - **ES.** El mapeo de tipos de cultivo es una tarea de clasificación específica que identifica tipos de cultivos a partir de imágenes satelitales.
