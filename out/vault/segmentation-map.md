# Segmentation map / Mapa de segmentación

> **level 4** · *product* · sources: th_parcel_extraction#results

**What it is / Qué es.**

- **EN.** A [[raster|raster]] or [[vector|vector]] [[product|product]] where each [[pixels|pixel]] or [[polygons|polygon]] is assigned a [[classes|class]] [[label|label]] representing a [[land-cover|land cover]] or [[land-use|land use]] category.
- **ES.** Un [[product|producto]] [[raster|ráster]] o vectorial donde cada píxel o polígono se asigna una [[label|etiqueta]] de clase que representa una categoría de [[land-cover|cobertura del suelo]] o [[land-use|uso del suelo]].

## Understand these first / Entiende esto primero
- [[map|Map]] — *product*
    - **EN.** A segmentation map is a thematic map showing homogeneous regions like fields or forest patches.
    - **ES.** Un mapa de segmentación es un mapa temático que muestra regiones homogéneas como campos o parches de bosque.
- [[segmentation|Segmentation]] — *method*
    - **EN.** A segmentation map stores the output of an image segmentation algorithm, labeling each pixel with a segment ID.
    - **ES.** Un mapa de segmentación almacena la salida de un algoritmo de segmentación de imágenes, etiquetando cada píxel con un ID de segmento.
