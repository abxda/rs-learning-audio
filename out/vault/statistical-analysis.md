# Statistical analysis / Análisis estadístico

> **level 6** · *method* · sources: index

**What it is / Qué es.**

- **EN.** The [[process|process]] of collecting, exploring, and presenting large quantities of [[data|data]] to discover underlying [[patterns|patterns]] and trends. In this [[handbook|handbook]], it refers to generating [[official-agricultural-statistics|official agricultural statistics]] from EO data.
- **ES.** El [[process|proceso]] de recolectar, explorar y presentar grandes cantidades de [[data|datos]] para descubrir [[patterns|patrones]] y tendencias subyacentes. En este [[handbook|manual]], se refiere a la generación de [[official-agricultural-statistics|estadísticas agrícolas oficiales]] a partir de datos EO.

## Understand these first / Entiende esto primero
- [[data-analysis|Data Analysis]] — *method*
    - **EN.** Statistical analysis applies data analysis techniques to estimate quantities and test hypotheses using models.
    - **ES.** El análisis estadístico aplica técnicas de análisis de datos para estimar cantidades y probar hipótesis usando modelos.
- [[statistical-methods|Statistical methods]] — *method*
    - **EN.** It employs statistical methods to ensure estimates are valid, unbiased, and accompanied by uncertainty measures.
    - **ES.** Emplea métodos estadísticos para asegurar que las estimaciones sean válidas, insesgadas y acompañadas de medidas de incertidumbre.
