# Feature / Atributo

> **level 1** · *data* · sources: th_machine_learning · ct_mexico

**What it is / Qué es.**

- **EN.** A measurable property or characteristic extracted from [[data|data]], such as [[spectral-reflectance|spectral reflectance]], [[vegetation-indices|vegetation index]], or texture, used as [[input|input]] for [[analysis|analysis]].
- **ES.** Una propiedad o característica medible extraída de los [[data|datos]], como la [[spectral-reflectance|reflectancia espectral]], el índice de [[vegetation|vegetación]] o la textura, utilizada como [[input|entrada]] para el [[analysis|análisis]].

## Understand these first / Entiende esto primero
- [[data|Data]] — *primitive*
    - **EN.** A feature is a measurable property or characteristic extracted from data, like spectral reflectance.
    - **ES.** Un atributo es una propiedad o característica medible extraída de datos, como la reflectancia espectral.

## This unlocks / Esto habilita
- [[predictors|Predictors]] — Predictors are features such as spectral bands, vegetation indices, or time series used in a model.
- [[identification|Identification]] — Identification recognizes features such as crop types by analyzing their spectral or temporal patterns.
- [[decision-tree|Decision tree]] — A decision tree splits the data based on feature values to create a tree-like model of decisions.
