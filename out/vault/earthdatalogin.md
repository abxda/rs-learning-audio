# Earthdatalogin / Earthdatalogin

> **level 1** · *software* · sources: th_data_sources

**What it is / Qué es.**

- **EN.** An [[r-package|R package]] that helps [[users|users]] set up credentials for accessing [[nasa|NASA]] [[earthdata|Earthdata]] services, such as creating a [[netrc-file|.netrc file]] for authentication.
- **ES.** Un [[r-package|paquete de R]] que ayuda a los [[users|usuarios]] a configurar credenciales para acceder a los servicios de [[earthdata|EarthData]] de la [[nasa|NASA]], como la creación de un [[netrc-file|archivo .netrc]] para autenticación.

## Understand these first / Entiende esto primero
- [[netrc-file|.netrc file]] — *primitive*
    - **EN.** earthdatalogin creates a .netrc file with user credentials for automatic Earthdata login.
    - **ES.** earthdatalogin crea un archivo .netrc con credenciales de usuario para el inicio de sesión automático en Earthdata.
