# Tools / Herramientas

> **level 2** · *primitive* · sources: introduction

**What it is / Qué es.**

- **EN.** [[software|Software]] systems, [[platforms|platforms]], and libraries used for [[processing|processing]], analyzing, and classifying [[earth-observation|Earth observation data]]. Examples include [[satellite-image-time-series-2|SITS]], Sen4Stat, Google [[earth|Earth]] Engine, and Python packages.
- **ES.** Sistemas de [[package|software]], [[platforms|plataformas]] y bibliotecas utilizadas para procesar, analizar y clasificar [[data|datos]] de [[earth-observation|observación de la Tierra]]. Los ejemplos incluyen SITS, Sen4Stat, Google Earth Engine y paquetes de Python.

## Understand these first / Entiende esto primero
- [[methods|Methods]] — *method*
    - **EN.** They operationalize methods into usable workflows for crop mapping and monitoring.
    - **ES.** Operacionalizan los métodos en flujos de trabajo utilizables para el mapeo y monitoreo de cultivos.
- [[package|Package]] — *software*
    - **EN.** A package is a set of tools organized into a distributable unit for ease of use.
    - **ES.** Un paquete es un conjunto de herramientas organizadas en una unidad distribuible para facilitar su uso.
- [[software|Software]] — *primitive*
    - **EN.** Tools are software systems that implement methods for processing and analyzing Earth observation data.
    - **ES.** Las herramientas son sistemas de software que implementan métodos para procesar y analizar datos de observación de la Tierra.
