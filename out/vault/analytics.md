# Analytics / Analítica

> **level 5** · *method* · sources: ad_world_cereal

**What it is / Qué es.**

- **EN.** The systematic computational [[analysis|analysis]] of [[data|data]], often using [[statistical|statistical]] or [[machine-learning|machine learning]] [[methods|methods]], to extract insights and support decision-making.
- **ES.** El [[analysis|análisis]] computacional sistemático de [[data|datos]], a menudo utilizando [[statistical-methods|métodos estadísticos]] o de [[machine-learning|aprendizaje automático]], para extraer [[information|información]] y apoyar la toma de decisiones.

## Understand these first / Entiende esto primero
- [[analysis|Analysis]] — *method*
    - **EN.** Analytics is the systematic application of computational methods to extract insights from data.
    - **ES.** La analítica es la aplicación sistemática de métodos computacionales para extraer información de los datos.
- [[data-science|Data Science]] — *primitive*
    - **EN.** It uses statistical and machine learning techniques for understanding Earth observation data.
    - **ES.** Utiliza técnicas estadísticas y de aprendizaje automático para comprender los datos de observación de la Tierra.
