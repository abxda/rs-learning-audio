# Decision tree / Árbol de decisión

> **level 5** · *method* · sources: th_machine_learning

**What it is / Qué es.**

- **EN.** A supervised [[machine-learning|machine learning]] [[model|model]] that splits [[data|data]] into branches based on [[feature|feature]] [[values|values]], forming a tree structure. Each leaf represents a [[classes|class]] [[label|label]]. Ensembles like [[random-forest|Random Forest]] combine multiple trees.
- **ES.** Un [[model|modelo]] de [[machine-learning|aprendizaje automático]] supervisado que divide los [[data|datos]] en ramas basadas en [[values|valores]] de atributos, formando una estructura de árbol. Cada hoja representa una [[label|etiqueta]] de clase. Los conjuntos como [[random-forest|Random Forest]] combinan múltiples árboles.

## Understand these first / Entiende esto primero
- [[classification|Classification]] — *method*
    - **EN.** In classification, a decision tree partitions the feature space into regions assigned to classes.
    - **ES.** En clasificación, un árbol de decisión divide el espacio de atributos en regiones asignadas a clases.
- [[feature|Feature]] — *data*
    - **EN.** A decision tree splits the data based on feature values to create a tree-like model of decisions.
    - **ES.** Un árbol de decisión divide los datos según los valores de los atributos para crear un modelo de decisiones en forma de árbol.
- [[random-forest|Random Forest]] — *method*
    - **EN.** Random forest extends decision trees by building many trees on random data subsets and averaging.
    - **ES.** Random forest extiende los árboles de decisión construyendo muchos árboles sobre subconjuntos aleatorios de datos y promediando.
