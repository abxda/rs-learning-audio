# Pixel classification / Clasificación de píxeles

> **level 3** · *method* · sources: th_data_cubes

**What it is / Qué es.**

- **EN.** Assigning a [[classes|class]] [[label|label]] to each individual [[pixels|pixel]] in an image based on its spectral or temporal characteristics.
- **ES.** Asignación de una [[label|etiqueta]] de clase a cada píxel individual de una imagen basada en sus características espectrales o temporales.

## Understand these first / Entiende esto primero
- [[classification|Classification]] — *method*
    - **EN.** Pixel classification applies classification to each individual pixel rather than objects.
    - **ES.** La clasificación de píxeles aplica clasificación a cada píxel individual en lugar de a objetos.
- [[pixels|Pixels]] — *data*
    - **EN.** Pixel classification assigns a class label to every pixel based on its spectral values.
    - **ES.** La clasificación de píxeles asigna una etiqueta de clase a cada píxel basándose en sus valores espectrales.
