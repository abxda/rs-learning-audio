# Instances / Instancias

> **level 2** · *primitive* · sources: th_machine_learning

**What it is / Qué es.**

- **EN.** Individual [[data|data]] [[points|points]] or objects in a [[dataset|dataset]], such as a single [[training|training]] [[samples|sample]] in [[machine-learning|machine learning]] or a single [[field|field]] [[parcels|parcel]] in object-based image [[analysis|analysis]].
- **ES.** [[points|Puntos]] de [[data|datos]] u objetos individuales en un [[dataset|conjunto de datos]], como una muestra de [[training|entrenamiento]] única en [[machine-learning|aprendizaje automático]] o una [[plot|parcela]] de [[field|campo]] individual en [[object-based-classification|clasificación basada en objetos]].

## Understand these first / Entiende esto primero
- [[samples|Samples]] — *data*
    - **EN.** Instances are individual training or validation data points, such as a single pixel's time series.
    - **ES.** Las instancias son puntos de datos individuales de entrenamiento o validación, como la serie temporal de un solo píxel.
- [[observation|Observation]] — *data*
    - **EN.** An instance can be an observation; a collection of instances forms a dataset for machine learning.
    - **ES.** Una instancia puede ser una observación; una colección de instancias forma un conjunto de datos para el aprendizaje automático.
